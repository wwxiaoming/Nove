# 《星辰之下》小说改编 - 验证检查清单

## 目录与文件结构
- [ ] adapted_chapters/目录已创建
- [ ] 改编大纲.md已创建
- [ ] 所有改编章节文件命名规范（01_章节名.txt格式）
- [ ] 原作chapters/目录下的原始文件未被修改

## 大纲检查
- [ ] 总章数在45-50章之间
- [ ] 每章都有明确的剧情概括
- [ ] 每章都标注了H场景类型或"无H"
- [ ] 没有连续3章以上无H的情况
- [ ] 新设定（紧身衣/败北/星辰血脉/默默/第二季）都有明确引入位置
- [ ] 原作所有重要剧情节点都被覆盖
- [ ] 标记了至少2-3处重点败北场景
- [ ] 结局有明确的第二季钩子

## 设定一致性检查
- [ ] 紧身衣战斗服在第1章就自然登场
- [ ] 紧身衣有质感描写（材质、贴身感、颜色、身体线条）
- [ ] 战斗中紧身衣有破损、出汗、沾污等色气描写
- [ ] 辰辰星辰血脉设定从中前期开始铺垫，后期逐步揭露
- [ ] 星光精能设定明确：在精液中，是改造人/魔物渴望的能量源
- [ ] 默默出场始终保持神秘感
- [ ] 默默有展现强大力量但不解释来源
- [ ] 默默的话语中暗示外星/异界存在
- [ ] 第二季伏笔从开篇就开始（异常信号/异常样本/异常星象）
- [ ] 伏笔逐步升级，结局时明确点出外星魔物威胁

## H场景检查
- [ ] 每1-2章必有H
- [ ] 基诺×辰辰甜蜜H占比约60%，温柔宠溺
- [ ] 战斗后安抚H占比约20%，情感浓烈
- [ ] 败北凌辱H占比约15%，调教程度高但不重口
- [ ] 其他角色H占比约5%
- [ ] H场景与剧情自然融合，不生硬
- [ ] H描写露骨但唯美，不粗俗
- [ ] H中保持人物性格（基诺宠溺、辰辰软萌）

## 调教分级检查
- [ ] 基诺×辰辰H中无刻意羞辱
- [ ] 基诺×辰辰H中只有自然的占有欲（称呼、姿势主导、轻咬标记）
- [ ] 基诺×辰辰H后有宠溺清洁/安抚情节
- [ ] 敌人H中有明确的捆绑/束缚描写
- [ ] 敌人H中有紧身衣破损/撕裂的羞耻感描写
- [ ] 敌人H中有强迫取精/榨取星光精能情节
- [ ] 敌人H中有语言羞辱
- [ ] 敌人调教场景无重度重口细节，用意象/感受带过侵入行为

## 重口元素处理检查
- [ ] 原作食人场景淡化处理，不详细描写吃人的过程和细节
- [ ] 血腥场景点到为止，侧重氛围和心理感受
- [ ] 虐杀/重度伤害不正面描写
- [ ] 败北调教侧重羞耻感和身体反应，不详细描写重度疼痛/损伤
- [ ] 所有重口内容都保持在美感范围内，不引起生理不适

## 文风检查
- [ ] 保持萌系轻小说感
- [ ] 人物对话符合年龄（基诺外冷内热、辰辰软萌可爱、冷血活泼残暴）
- [ ] 有细腻的心理描写
- [ ] 星辰/光/翅膀的意象贯穿全文
- [ ] 战斗场景有画面感，同时融入色气元素
- [ ] 日常互动甜蜜自然

## 人物OOC检查
- [ ] 基诺始终宠溺辰辰，即使H中也不真正伤害他
- [ ] 辰辰软萌但不懦弱，有小男子汉的一面
- [ ] 冷血虽然残暴但对基诺有兄弟情，后期洗白合理
- [ ] 默默神秘强大，话少但每句都有分量
- [ ] 反派的行为逻辑合理（觊觎星光精能）

## 第二季伏笔检查
- [ ] 第1章游轮事件时就有异常生物信号描写
- [ ] "希尔的组织样本"来源暗示非地球
- [ ] 默默出场时的话语有外星/异界暗示
- [ ] 中期有星象异常/能量异常的小细节
- [ ] 抢救默默时发现她身体构造异常
- [ ] 敌人组织背后有更强大的存在暗示
- [ ] 最终高潮辰辰星光精能爆发引来未知存在
- [ ] 默默出手击退敌人后留下"它们来了"的警告
- [ ] 结局天空出现不属于地球的星象/异象
- [ ] 明确留下第二季外星魔物文明的钩子
