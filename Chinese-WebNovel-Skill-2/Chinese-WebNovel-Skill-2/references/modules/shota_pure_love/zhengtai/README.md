# 正太综合题材模块

这是正太纯爱文风模块组的**综合补充工具箱**，当星辰风（萌系奇幻）、房东风（第一人称后宫）、英雄风（黑暗调教）三种主风格都不匹配时，按题材从这里选择对应子类参考。

## 模块定位

星辰、房东、英雄三种主风格覆盖了80%以上的常见正太纯爱写作场景，但仍有一些特定题材无法被很好覆盖：
- 校园欢喜冤家、伪兄弟日常纯爱
- 综漫同人、快穿系统爽文
- 会所、拍卖、交易类黑暗设定
- 需要强节奏张力、极致感官描写的场景

本子模块就是为这些场景准备的**技法工具箱**——不是单一风格，而是按题材分类的多种写法集合。

## 子类选择方法

### 第一步：先排除三个主风格
1. 如果是**萌系奇幻/单CP纯爱/杀手设定** → 回去选 [xingchen/](../xingchen/)
2. 如果是**第一人称/现代日常/后宫甜宠** → 回去选 [nianqing/](../nianqing/)
3. 如果是**英雄/正义角色陷落/黑暗调教** → 回去选 [yingxiong/](../yingxiong/)

### 第二步：按题材选择本子类
如果以上三种都不匹配，再根据你的题材选择：

1. **现代背景、平等关系、日常甜宠？**
   - 伪兄弟、继兄弟、表兄弟 → 日常纯爱风（《宝贝今年十四岁》）
   - 校园、同学、欢喜冤家、宿舍同居 → 校园纯爱风（《运动裤下的秘密》）

2. **综漫同人、快穿、系统、爽文？**
   - 穿越多个动漫世界 → 综漫爽文风
   - 有系统面板、攻略任务、金手指 → 综漫爽文风
   - 需要主角能力碾压、爽点密集 → 综漫爽文风

3. **黑暗背景、设定系、产业规则？**
   - 会所、拍卖、俱乐部、交易 → 黑暗产业风
   - 需要详细构建等级制度、行业规则 → 黑暗产业风
   - 主角是职业从业者视角 → 黑暗产业风

4. **需要学习节奏把控、感官描写？**
   - 亲密场景写得太平铺直叙、没有张力 → 参考鬼畜风的节奏技法
   - 感官描写单薄、没有画面感 → 参考鬼畜风的五感调动方法
   - ⚠️ **注意**：只学技法，不直接搬重口内容

### 不确定时的默认选择
如果实在不确定选哪个，**优先选日常/校园纯爱风**——这是最基础、最通用的正太写法，适用场景最广。

## 推荐使用顺序

1. 先读你当前目录的 [README.md](README.md)（也就是这份）了解模块定位
2. 再读 [source_index.md](source_index.md) 知道5篇小说分别对应什么子类
3. 需要系统学习时读 [tutorial.md](tutorial.md)——这是主教程，≥500行，覆盖所有子类的写法指导
4. 写作/改稿时按 [runtime.md](runtime.md) 的流程做诊断和计划
5. 去 [good_examples.md](good_examples.md) 找2-4个近似正例，学习技法
6. 去 [bad_examples.md](bad_examples.md) 找1-2个近似反例，避开坑
7. 动笔前先按runtime.md里的模板写写作计划

## 文件说明

- **tutorial.md**：主教程，包含：
  - 模块定位说明
  - 四个子类的文风详解（每个都配写法指导）
  - 通用正太写作技巧（年龄感把控、身份关系写法、视角内心）
  - 调用决策树

- **good_examples.md**：正例库，≥25个例子，按子类分组
  - 每个例子包含：原文片段+使用的技法+详细拆解
  - 覆盖各子类的核心写法，尤其是亲密场景

- **bad_examples.md**：反例库，≥20个反例
  - 覆盖正太题材通用错误：年龄感混乱、题材杂糅、OOC、亲密场景无风格、人物关系不清等
  - 每个错误说明问题所在+改进方向

- **runtime.md**：运行规则，包含：
  - 子类选择流程（5步判断法）
  - 写作诊断清单
  - 动笔前的计划模板

- **source_index.md**：资料索引，对5篇小说做了详细分类和参考价值评级

## 重要说明

1. **主风格优先**：本子模块是补充工具箱，优先使用xingchen/nianqing/yingxiong三个主模块
2. **风格不要混用**：一篇文里选一个子类作为主基调，可以借鉴其他子类的技法，但不要同时用多个子类的核心写法
3. **技法可迁移，内容需谨慎**：
   - 鬼畜风的节奏、张力、感官技法可以用在纯爱场景
   - 但重口、非自愿、极端内容不要直接搬到你的文里
4. **年龄感是核心**：不管用哪个子类，精准把控正太的年龄感都是第一位的
5. **关系决定尺度**：伪兄弟、同学、主仆、陌生人的亲密推进速度和互动模式完全不同

## 与其他模块的组合使用

- **日常纯爱 + 房东风**：想写第一人称伪兄弟日常，可以房东风为基调，加入《宝贝今年十四岁》的细节
- **校园纯爱 + 星辰风**：想写萌系校园，可以星辰风为基调，加入《运动裤下的秘密》的欢喜冤家冲突
- **综漫爽文 + 英雄风**：想写黑暗向综漫，可以英雄风为基调，加入综漫的穿越和系统元素
- **所有子类都可以参考鬼畜风的节奏技法**：用来提升亲密场景的张力，但不要过度
