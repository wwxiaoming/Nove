## 第一节：反AI腔（去AI味）

"去AI味"不是最后润色，而是先判断空在哪、假在哪、同质化在哪，再替换。

### 1. 什么时候优先处理AI味
- 用户说"太像AI写的""太模板""太假"
- 反复出现抽象判断、空泛情绪、氛围套话
- 角色说话都太像作者本人
- 章节能看，但句面发空、没有落地物
⚠️ 如果先有因果断裂/动机不成立/转场断片/人物崩，**先修底层，再回来去AI味**。

### 2. AI味四层诊断（先定位在哪一层）

#### 第1层：空泛总结层
表现：很多"她很难过""气氛很微妙""他很危险"；句子都在下判断，没有现场。
优先修：动作化、物件化、反应链。

#### 第2层：人物声音同质层
表现：所有人都很会解释；所有人说话都完整、工整、礼貌；身份感和关系差不见了。
优先修：身份差、立场差、压人方式差。

#### 第3层：套话氛围层
表现："空气仿佛凝固""气氛一时有些微妙""时间仿佛静止"。
优先修：场内变化、可听见的小声音、距离和动作变化。

#### 第4层：平均节奏层
表现：句子都差不多长；情绪越大越完整；没有断句、停顿、被打断。
优先修：句长拉开、动作截断、短句插刀。

### 3. 动笔前先写替换计划（改正文前脑中写清6句）
- 这一段最空的一句是什么：
- 我打算删掉它，还是拆开它：
- 这份情绪最适合落在哪个动作或物件上：
- 这一段谁最该说一句带身份的话：
- 哪一句需要缩短，哪一句需要被打断：
- 改完之后，局面会不会比原来更有压：

说不清这6句，先不要急着润色。

### 4. 硬规则
- 能用动作，不用总结
- 能用对白，不用解释
- 能写具体，不写抽象
- 能带身份感，不写统一腔
- 能改局面，不只改词面
- 能短就别拖，能停就别讲满
- 不要把去AI味理解成故意写糙

### 5. 禁用词/句替换表
| ❌ 禁用 | ✅ 替换方向 |
|--------|-----------|
| 眼中闪过精光/精光一闪 | 眼神变化+具体动作（眯眼/瞳孔收缩/目光锐利） |
| 嘴角勾起弧度/邪笑 | 具体唇部动作+表情（扯了扯嘴角/露出一点笑意） |
| 倒吸一口凉气 | 具体生理反应（呼吸顿了一下/喉咙发紧/抽气声） |
| 指节泛白 | 手指攥紧/指甲掐进掌心 |
| 弓起身子 | 腰腹绷紧/后背绷紧/腰胯不自觉前挺 |
| 不是...而是... | 直接写结果，不要先否定 |
| 破折号—— | 用逗号/句号 |
| ...的时候/...的瞬间 | 直接改成逗号 |
| XX的声音/语气/语调 | 删掉，用对话内容和动作表现 |
| 农牛/精牛 | 推送/套弄/榨取（全书≤5次） |

### 6. 写完后反AI自查6条
1. 有没有一句删掉后整段几乎不受影响的空泛总结句
2. 有没有三个以上连续判断句，没有一个落动作
3. 有没有角色在高压场景里说出过分完整、过分讲理的话
4. 有没有同一段里堆了多个同类形容词
5. 有没有"气氛很微妙"这种作者代读者下结论的句子
6. 改完后读者能不能不靠解释也知道谁怕谁、谁让谁、谁在硬撑

## 第二节：三层链路专项解法

写作问题分三层：**前置规划层 → 正文执行层 → 完稿收口层**。先判断问题在哪层，再找对应解法。

### 1. 前置规划：概念规划（concept_planning）

**什么时候用**：
- 点子值不值得写、题材和消费点清不清、卖点和异常局面亮不亮
- hook/premise立不立得住、故事引擎能不能跑过前10-20章
- 更适合短/中/长篇/连载、第一卷最小目标和卷末变化清不清

**硬要求**：
- 先判断值不值得写，再写正文
- 先给卖点、冲突、故事引擎，再给大设定
- 简介压不成最小骨架时，不要急着开写

**最小骨架五要素**：
- 题材与消费点（读者为什么要看这本书）
- 一句话hook（最吸引人的异常局面）
- premise（如果...那么...）
- 故事引擎（什么机制持续推动剧情）
- 第一卷方向（主角从什么状态变到什么状态）

### 2. 开篇执行：开头成交（opening）

**什么时候用**：开头没抓手、卖点露头太慢、异常局面不够立、黄金三章起得慢

**硬要求**：
- 前300-500字给抓手（异常/冲突/悬念）
- 前1500-3000字给卖点
- 前6000字让读者知道：主角是谁、遇到什么、卖点是什么、接下来想看什么

**开头常见抓手类型**：
- 危机压身（一上来就在战斗/被抓/危险中）
- 异常局面（不符合常识的设定直接扔出来）
- 关系张力（两个人一见面就有压）
- 反差冲击（日常里突然出现不日常的东西）

### 3. 中长线结构：卷章规划（volume_outline）

**什么时候用**：黄金三章推进无力、第一卷跑法不清、前10-20章章纲散、中段容易发散、卷末没有兑现点

**硬要求**：
- 长篇网文必须写章纲；没有章纲容易支线乱长、设定膨胀、章末疲软、追更点断掉
- 每章必须回答四件事（见SOUL.md）
- 每场戏必须有：目标→阻碍→变化；无变化的场景删、并、压缩
- 卷末必须有兑现点（主角状态/关系/格局发生不可逆变化）

**节奏档位分配**（长篇适用）：
- 慢档（铺垫/羁绊/日常/余韵）：约30%，每3-4章至少1章，主线零推进
- 中档（次要矛盾升温/未爆发/小冲突）：约60%
- 快档（主线突破/大战/H高潮/战败/重大关系升级）：每卷≤2-3次，快档后必须有慢/中档缓冲
- **配额铁律**：每章A/B/C三类大事件（主线实质推进/关系决定性升级/核心秘密揭露）最多触发1项

### 4. 底层结构：剧情逻辑（plot_logic）

**什么时候用**：因果断裂、动机不足、触发不成立、后果不落地

**因果链标准模板**：
动机 → 触发事件 → 角色决策 → 行动 → 直接后果 → 后续涟漪
任何一环断裂，读者就会出戏。

**常见逻辑问题**：
- 角色为什么要做这件事？（动机是否充分）
- 这件事为什么现在发生？（触发是否自然）
- 角色的决策符合他的性格和信息吗？（决策是否合理）
- 行动产生了什么后果？后果有没有落地？（后果不能消失）
- 后果有没有影响后续剧情？（涟漪必须传递）

### 5. 人物状态：人物一致性（character_consistency）

**什么时候用**：人设崩、目标漂移、情绪断片、关系断片、身体和声音连续性丢失

**五类连续性必查**：
1. **目标连续性**：主角这章到底想做什么？过程中有没有无故漂移？
2. **情绪连续性**：情绪有没有来路？是不是凭空产生？
3. **关系连续性**：关系温度是不是和上一场对得上？
4. **身体连续性**：伤势、疲劳、敏感部位状态、衣服/战衣状态有没有丢？
5. **声音连续性**：角色说话方式有没有突然变成另一个人？

**反机器人规则**：
- 角色对话禁止使用精确数字描述身体状态
- 观察基于五感模糊印象（"好像""似乎""有点""显得"）
- 即使理性人设也不做科研式分析，用"言简意赅""反应略平淡"
- 写到新心声、新举动、新台词时，刻意制造和上文的不同点
- 不同角色手部技法有区别（宽厚掌→推送/抛光；细长指→寸止/钻探；短粗指→揉捏/掐根）

**反抗心理必须保留**：即使身体爽到失控，内心也要有 `不能在这里射...` `可恶...身体怎么会...` 这类抵抗声音。

### 6. 场景桥梁：转场衔接（transition）

**什么时候用**：时间跳切生硬、空间切换让读者迷路、情绪承接断裂、视角切换突兀、章末接不上章

**转场硬规则**：
- 场景切换时重新交代时间地点
- 上一场的余力（情绪/伤势/未解决的张力）必须带到下一场
- 时间推进符合常理，禁止大跨度跳时（除非明确交代）
- 跳时后要快速让读者重新定位：现在在哪、什么时候、谁在场、刚才发生了什么

**情绪承接**：
- 一场高强度戏（战斗/H/争吵）后，不能立刻切到完全无关的轻松场景，要有情绪过渡
- 可以用身体反应过渡（喘息、颤抖、余韵）、环境细节过渡、角色内心过渡

### 7. 关系施压：对白刀口（dialogue）

**什么时候用**：高张力对白、人物声音不突出、信息嵌入生硬、对白像说明书

**对白硬规则**：
- 对话必须有目的（抒情/社交/求知/表达/工作/娱乐欲求），无欲求则沉默
- 口语化、情绪化，插入停顿、口吃、填充词
- 不同角色语言纹理差异巨大：词汇量、句式复杂度贴合年龄、知识面、素养
- 禁止翻译腔，要有血肉感、呼吸感
- 合理使用？！...~等标点和拟声词
- 每句对白用同一类型引号
- 禁止"他说""她问"这种简短提示语，用人称+动作/神态
- 最推荐：连续独立成段纯对话 + 流水句旁白独立成段，不加提示语
- 禁止"声音/语气/语调"类描写

**反派"科研话术"参考**："同步率测试""肌肉数据采集""能量采样""精能转化率分析"——用正当名义包装猥亵行为。

**心理描写格式**：用反引号 ` 包裹心理活动（`真实想法`），第一人称短句，高频出现，可以有欺骗性（嘴硬身体诚实）。

### 8. 章节收束：章末钩子（chapter_ending）

**什么时候用**：用户要求补章末/补追更点、文本收在总结/讲理/抒情上、章末没把变化收住、章末虽然炸但下一章接不上、收尾太平/太散

**先诊断卡在哪一层**：

| 层级 | 表现 | 优先补 |
|-----|------|-------|
| 第1层：没有落点 | 写完就停，最后一段像散场 | 最后一刀落点、本章最值钱的变化 |
| 第2层：有落点没后劲 | 收尾完整但无追更拉力 | 未尽之力、下一章自然入口 |
| 第3层：硬钩感太重 | 每章都像电视剧广告点，钩子不是从本章生长出来 | 从关系/情绪/决定里长出钩子 |
| 第4层：上下章脱节 | 章末蓄力了，下一章第一拍没接住 | 和转场联动，让下一章第一拍兑现或反打余力 |

**动笔前先明确**：
- 本章最后一拍收什么：
- 收在谁的动作/谁的话/谁的决定上：
- 读者下一秒最想知道什么：
- 下一章第一拍怎么接：

**章末硬规则**：
- 不要收在总结句
- 不要收在讲道理
- 不要把悬念解释没
- 不要为了钩而钩
- 不要忘了下一章还得接

## 第三节：NSFW详细写作技法

### 1. 五感描写细则

#### 视觉
- 汗湿肌肤灯光下泛珍珠般水色（强调油腻感）
- 乳首/性器官因快感充血：粉→艳红→深紫颜色变化
- 后穴被撑开时内壁嫩肉翻涌的湿亮粉色
- 精液与爱液混合后拉出的晶莹淫丝
- 体液在皮肤/床单/紧身衣上留下的痕迹、反光
- 肌肉绷紧/放松的线条变化、颤抖频率
- **紧身衣专属**：胯下浸透的深色水渍晕开、布料勒出轮廓、战衣破裂布片挂身、指示灯/目镜变粉、射精时布料绷紧精液从缝隙渗出

### 2. 动作-感受绑定（核心）
- 攻方动作作为陪衬简单描写，减少字数
- 每个动作必须呈现**主观感受**和**心理反应**
- 快感具象化：电流窜过、骨头酥了、小腹发热、意识空白、魂飞天外、整个人融化、酸胀、酥麻、疼爽、龟头被蹭时的麻痒、冠状沟被揉时的酸胀、马眼被刮时的锐麻
- 高潮必须有情感释放/心理变化+身体反应（拒绝转配合/痉挛/脚趾扣紧/眼前发白/意识断片/肌肉僵直脱力）
- 禁止：仅写器官+动作无主观感受；旁观者冷静生理记录；上帝视角纯动作记录
- 禁止用"弓"作为快感动作

### 3. 手部榨精技法描写框架
（20+种具体手法序列→ NOTES_榨精手法与道具.md）

#### 基础套弄6步框架
1. **握持姿态**：手怎么放上去（虎口卡根？反手握？侧握？）
2. **第一下触感**：手掌包裹温度、肉棒在掌心跳动反馈
3. **推送过程**：从根到冠，掌心蹭过哪里（柱身→冠状沟→龟头）
4. **被榨者反应**：每一下推送时的颤抖、呻吟、肌肉绷紧
5. **节奏变化**：什么时候轻/重/停（寸止）
6. **辅助手动作**：另一只手在做什么（揉睾丸？按会阴？捏乳头？）

#### 关键手法描写提示
- **虎口环扣**："虎口卡在根部形成密闭的环，五指收拢顺着柱身向上推送，掌心肉垫恰好蹭过冠状沟下缘..."
- **轻重交替**："第一下轻得像羽毛拂过，第二下突然加重力道大力推到龟头，在马眼上碾了一下才松开..."
- **龟头抛光**："掌心包住通红的龟头快速画圈，前液被蹭得发出咕叽咕叽的水声，每一圈都带着掌纹粗糙的摩擦..."
