## 一、核心母题与子类型

### 1.1 核心母题（TOP 5）
1. **紧身衣高弹贴合**：8/8作品命中，是所有战队特摄的母型
2. **榨精魔物**：至少10种动物（蛇/蜈蚣/蝎子/蛤蟆/蜘蛛/狗/猫/鸟/鱼/虫）
3. **5色战队多P同时**：群像机制标准范式
4. **演武场+战败**：特摄文化专有场景
5. **取精舱/精能基础设施**：把"榨精"基础设定化

### 1.2 5大子类型
| 编号 | 子类型 | 核心范式 |
|:---:|---|---|
| ① | 紧身衣战士 | 紧身衣男战士+胯下攻击+强制高潮+战败羞辱 |
| ④ | 陷阱捕获 | 看似求助/结盟实为陷阱，捕获后榨精/当众羞辱 |
| ⑤ | 精液=战斗力 | 精液=超能力来源，越榨越弱、越弱越被榨的死亡螺旋 |
| ⑥ | 竹马/队友互调 | 竹马从小一起→觉醒后互撸+互调+口交 |
| ⑧ | 玩具化/羞辱 | 被洗脑/被改造/被公开→当众使用 |

## 三、关键场景库

### 3.1 训练/任务/后勤三场所
- **训练室**：撞奶/被压制/教官喷乳/胯下按摩器/撞见当众勃起
- **巡逻**：小巷强吻/飞行中完全勃起/公共场合隐秘手交
- **宿舍**：沙发舔脚/龟责/凌晨口交/竹马互撸
- **浴室**：公共淋浴室三人偷听/互相涂抹/足交
- **任务地点**：洞穴菌魔/植物陷阱/演武场对决

### 3.2 标志性场景
1. **演武场对决**：踩裆+玩弄肿胀下体+逼出认主宣言
2. **决战前夕示众仪式**：全部堕落→发起"决战"→实为公开羞辱大会
3. **救援队反转被捕**："英雄救美"反转成"英雄被救"再被捕
4. **射精篮球赛**：震动棒+尿道棒+乳夹+痒纹+口球累加+败者惩罚
5. **7天7夜博士实验**：廉价旅馆+能量探测器+电击+强制高潮+直播威胁

## 五、紧身衣描写词库（6种角度）
1. **日常训练**：缝线摩擦龟头/囊袋，每走一步都蹭过敏感点
2. **战衣合体**：纳米材料绷紧，勾出勃起轮廓，冠状沟/龟头/乳头/屁股缝都勒出形状
3. **射精痕迹**：前列腺液沿阴茎轮廓滴到睾丸，精液浸透裆部布料，晕开深色水渍
4. **肿胀勒紧**：下体肿胀被布料勒得发疼，龟头被布料压得变形，想蹭又不敢蹭
5. **战损踩裆**：战靴踩下，胯下布料变形包裹住龟头，金属贞操锁发出清脆碰撞声
6. **唯一遮蔽**：战衣是唯一遮蔽但已被体液污染，布料半透明贴在身上，肉棒激凸清晰可见

## 六、关系配置模板
1. **队长↔队员**：队长最先堕落，队员跟随
2. **竹马↔竹马**：从兄弟到爱人，无血缘自然过渡
3. **主角↔非血缘长辈**：教官=非血缘长辈，训练擦胸/取精舱共处/任务中暗示
4. **主角↔驯化者**：反派=养成者，温柔驯化代替暴力压制
5. **主角↔镜像CP**：两人同时堕落，双线叙事对照
6. **主角↔榨精魔物**：兽型魔物代替触手，每种动物对应一种裆部攻击方式
7. **兄弟↔宿敌↔同学↔主仆**：多关系网叠加
