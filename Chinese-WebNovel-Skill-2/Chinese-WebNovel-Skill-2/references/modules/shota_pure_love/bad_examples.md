# 正太纯爱文风反例库

这份反例库按「错误簇」分组，不是平铺罗列。

使用方法：

1. 先判断自己当前最像哪一类错误。
2. 再看这类错误通常断在什么地方。
3. 回到 `tutorial.md` 和 `runtime.md`，决定该补反差、萌点细节、情感递进还是尺度把控。

## A 组：新手型基础错误

这组最常见，也最容易修。

它们通常不是「写得不高级」，而是压根没摸到正太纯爱的门道。

### SP-A01 为了萌而萌，全是叠词没有内容

```text
小可爱乖乖吃饭饭软软萌萌，小手手拿着勺勺，眼睛亮晶晶水汪汪，真是太可爱啦。
```

问题：叠词堆砌成灾，读者只看到一堆重复的字，看不到一个具体的、有性格的小孩。萌是从细节里自然流出来的，不是喊出来的。正确方向：写一个具体动作（比如饭粒沾在嘴角，或者用勺子戳胡萝卜），让读者自己觉得可爱，而不是作者强行按头说「他很萌」。

### SP-A02 直接告诉读者「他很萌」而不是展示

```text
他是一个非常可爱的小男孩，今年十二岁，长得特别萌，性格也很软，是个人见人爱的小正太。
```

问题：这是人物设定表，不是小说。读者看完只知道作者觉得他萌，完全感受不到他萌在哪里。正确方向：用具体的事件、动作、神态来展示——比如他紧张时会揪衣角，说话声音小小的但很认真，吃到好吃的会眼睛弯起来。

### SP-A03 正太说话像大人

```text
「兄长，我认为此事不妥。父亲临行前嘱托我们要谨慎行事，切不可鲁莽。」
十二岁的小男孩板着脸，一副老气横秋的样子。
```

问题：正太的「小大人感」不是真的让他说大人话。十二岁的孩子就算再懂事，说话也会有属于他的稚气——会用错词，会结巴，会在正经话里不小心漏出小孩的语气。正确方向：保留他想装成熟的动机，但让他偶尔破功（比如话说到一半卡壳，或者尾音不自觉带点撒娇）。

### SP-A04 大人说话像小孩

```text
「宝宝乖哦～叔叔带你去买糖糖吃好不好呀～」
二十五岁的男人蹲下来，捏了捏小孩的脸，用叠词哄着。
```

问题：正太纯爱里的攻方（房东/哥哥/监护人）应该是温柔成熟的，不是弱智。用叠词哄一两岁婴儿没问题，但面对已经有自我意识的正太还这样说话，只会显得油腻且不尊重人。正确方向：温柔是体现在语气和细节上（比如放低声音、注意他的情绪、帮他整理领子），不是自己也变成小孩说话。

### SP-A05 开篇没有反差直接平铺直叙

```text
林晓今年二十五岁，是一名普通的上班族。他租了一间房子，房东是一个十二岁的小男孩。他们就这样住在了一起。
```

问题：正太纯爱的核心萌点之一就是「身份/年龄反差」——房东是小孩，租客是大人；或者哥哥是小孩却要照顾大人。开篇直接平铺直叙把这个反差当背景板交代，浪费了最抓人的点。正确方向：把反差放到第一个场景里展示——比如小孩踮着脚递钥匙，一本正经地说「房租要按时交」，而大人站在那里一脸错愕。

### SP-A06 第一章见面就告白/上床，完全没有情感递进

```text
林晓第一次见到小星就觉得他特别可爱。当天晚上，小星红着脸敲开他的房门，说「哥哥我喜欢你」，然后他们就在一起了。
```

问题：纯爱的核心是「慢慢来」。正太纯爱尤其讲究小心翼翼的试探、靠近、动心的过程。第一章直接把所有进度条拉满，读者根本没有代入的机会，只会觉得突兀。正确方向：从第一次见面的印象开始，写日常相处里的小细节——他递水时碰到手会躲开，会偷偷看你，会因为你和别人说话而鼓脸。情感是一点点累积的。

## B 组：人物与情感断裂问题

这组是最伤文的一类，因为它不只是写得不好，还会把人物写崩。

### SP-B01 上一章还很抗拒下一章就主动投怀送抱

```text
（上一章：小星红着脸推开林晓，说「我不要你碰我」，然后跑回了房间。）
（下一章：第二天早上，小星主动钻进林晓怀里，蹭着他的胸口说「哥哥抱」。）
```

问题：人物态度一百八十度大转弯，中间没有任何过渡和心理铺垫。读者只会觉得「这小孩是不是有什么毛病」或者「作者是不是忘了自己上一章写了什么」。正确方向：中间必须有台阶——比如他晚上想了很久，或者他做了噩梦害怕，或者他看到林晓受伤了心软。抗拒到接受之间，需要一个具体的事件来撬开他的心防。

### SP-B02 吃醋写得像恶毒女配

```text
小星看到林晓和女邻居说话，气得指甲掐进了掌心。
「那个女人有什么好的！」他在心里恶狠狠地想，「我一定要让她知道，哥哥是我的！」
第二天，他故意把水洒在女邻居的裙子上，还朝她做了个鬼脸。
```

问题：正太吃醋应该是软的、可爱的、让人心疼的——鼓脸、赌气、撒娇、故意不理人但又忍不住偷瞄，这些才对。写成陷害、辱骂、恶意报复，直接把可爱小正太写成了恶毒女配，读者只会觉得反感。正确方向：把吃醋写成笨拙的占有欲——比如他会挤到你们中间，会故意大声说话刷存在感，会回家后委屈地问「哥哥是不是不喜欢我了」。

### SP-B03 第一人称内心戏太猥琐变人渣

```text
看着小星细皮嫩肉的样子，我心里痒痒的。
这么嫩的小孩，不玩白不玩。
反正他也喜欢我，到时候哄两句就行了。
```

问题：正太纯爱里的攻方可以有欲望，但不能有这种「玩弄」的心态。第一人称写这种猥琐心理，直接把温柔房东写成了恋童癖人渣，读者会直接弃文。正确方向：欲望应该是伴随着珍惜和克制的——比如你会觉得他可爱，会忍不住想抱他，但同时你会紧张，会怕吓到他，会责怪自己怎么能对这么小的孩子有这种想法。

### SP-B04 保护欲写成控制欲

```text
「不准你和别人说话！」林晓把小星锁在房间里，「以后除了我身边，你哪里都不准去！谁也不准见！」
```

问题：保护欲是「我怕你受伤，所以我会在你身边陪着你」；控制欲是「你是我的所有物，不准你有自己的社交」。过度占有只会让攻方看起来像个变态，而不是温柔的恋人。正确方向：保护欲应该体现在细节上——比如你会提醒他过马路小心，会在他被欺负时站出来，会在他晚归时去接他，但你不会干涉他正常的人际交往。

### SP-B05 人物性格前后不一致

```text
（前三章：小星特别害羞，说话声音小得像蚊子，和林晓对视都会脸红到耳根。）
（第四章：家里来了客人，小星主动上去打招呼，还当众坐在林晓腿上喂他吃水果，大方得像换了个人。）
```

问题：人物性格突变如果没有合理的原因，读者会觉得非常出戏。害羞的小孩不会突然就变得豪放，除非有什么特殊的触发点（比如他在宣誓主权，或者他其实是在强装镇定）。正确方向：如果要写他在某个时刻变得大胆，必须给出理由——比如他喝了点果酒有点醉，或者他看到别人和林晓太亲近急了，或者他其实是在偷偷练习怎么和喜欢的人亲近。

### SP-B06 悲伤/委屈时没有具体动作只有「他很难过」

```text
听到林晓要搬走的消息，小星很难过。
他站在原地，心里特别伤心。
```

问题：又是「告诉」而不是「展示」。「难过」是一个抽象的词，读者看不到也摸不着。正确方向：用具体的动作和细节来表现——比如他的嘴唇抖了抖，眼眶一下子红了，但他咬着嘴唇不让眼泪掉下来；或者他攥着衣角，手指都捏白了，半天说不出一句话；或者他点点头说「我知道了」，然后转身跑回房间，关门的声音比平时小。

## C 组：风格混乱与节奏问题

这组问题的特点是：看起来写得顺，但读起来就是怪。

### SP-C01 星辰风和房东风混用

```text
小星鼓着脸坐在沙发上，像一只生气的小仓鼠。
靠，这小孩怎么这么可爱啊，我他妈心都化了。
林晓走过去揉了揉他的头发。
```

问题：正太纯爱有两种常用视角——第三人称（星辰风，细腻唯美）和第一人称（房东风，带点吐槽但温柔）。同一段里第三人称突然跳第一人称吐槽，镜头直接碎了。正确方向：选定一个视角就坚持到底，如果要切换，请用章节或者明确的段落分隔，并且要有清晰的边界提示。

### SP-C02 日常全是流水账

```text
早上七点，小星准时起床。
他刷牙洗脸，然后去厨房做早餐。
吃完早餐，他背着书包去上学。
下午放学回家，他先写作业，然后吃晚饭。
晚上十点，他准时睡觉。
```

问题：日常不是事无巨细地报流水账。读者不想看他一天的日程表，想看的是日常里的「萌点」和「事件」。正确方向：在日常里找冲突、找细节、找互动——比如做早餐时煎糊了鸡蛋，写作业时咬笔帽，晚上害怕打雷偷偷跑到你房间门口犹豫要不要敲门。

### SP-C03 萌点堆砌太密集导致审美疲劳

```text
小星歪头杀。
小星鼓脸。
小星蹭手心。
小星咬嘴唇。
小星眼泪汪汪。
小星软乎乎地喊哥哥。
每一段都在卖萌，每一页都有萌点。
```

问题：萌点就像糖，偶尔吃一颗觉得甜，一口气吃十颗只会觉得腻。每一段都在卖萌，没有喘息的空间，读者很快就会审美疲劳，再可爱的动作也变得无感。正确方向：萌点要疏密有致——紧张的剧情之后放一个萌点调剂，平淡的日常里穿插一个小事件，让萌点成为亮点而不是背景噪音。

### SP-C04 情感递进跳台阶

```text
他们认识才三天。
第一天牵手。
第二天拥抱。
第三天接吻。
然后直接在一起了。
```

问题：纯爱的心动就在于中间那些「差一点」的瞬间——想碰又缩回去的手，差点说出口又咽回去的话，不小心碰到时两个人同时脸红。把这些试探、害羞、心动全跳过，直接从牵手跳到本垒，纯爱感直接没了。正确方向：把每一步都放慢——第一次不小心碰到手两个人都愣了三秒，第一次拥抱时他身体僵硬得像块木头，第一次接吻时他紧张得牙齿都在抖。

### SP-C05 多人物戏份完全失衡

```text
配角小王只在主角需要亲密互动时出现，平时连影子都没有。
他存在的唯一意义就是当背景板，或者在亲密戏前说一句「那我先走了」。
```

问题：工具人角色太明显，读者一眼就能看出来「这个角色就是为了方便主角谈恋爱才存在的」。正确方向：给配角自己的性格和故事线——哪怕他只有几场戏，也要让他像个活人，有自己的事情要做，有自己的情绪，而不是主角的背景板。

### SP-C06 意象用了一次就再也不提

```text
林晓第一次见面送了小星一条星辰挂坠，说「看到它就像看到我一样」。
小星特别感动，把挂坠攥在手里。
然后……然后就没有然后了。
后面几十章再也没提过这条挂坠。
```

问题：重要的意象/物件是情感的锚点。送的时候写得特别郑重，后面完全不出现，就像没存在过一样，读者会觉得自己被耍了。正确方向：让意象在关键节点反复出现——比如他紧张时会摸挂坠，吵架时会把挂坠摘下来，和好时又会戴上，挂坠丢了会疯找。让物件成为他们感情的见证。

## D 组：尺度把握失衡问题

这组问题在于尺度要么太过要么不及，破坏纯爱氛围。

**重要提醒：本组反例只展示错误的写法结构，不会出现任何露骨内容。重点分析为什么这是错的，以及应该怎么写。**

### SP-D01 没有情感铺垫直接写亲密

```text
他们认识才一周，连手都没牵过。
[这里直接写了大段亲密场景]
```

问题：纯爱的亲密是情感累积到一定程度后的自然爆发，不是为了写而写。前面连暧昧都没有，直接上亲密，读者只会觉得突兀、尴尬，甚至觉得被冒犯，完全不会觉得甜。正确方向：先把情感铺垫到位——心动、试探、确认心意，所有步骤都走完了，两个人都确定对方喜欢自己，这时候的亲密才是水到渠成的甜。

### SP-D02 前戏写得像机械动作说明书

```text
他先做了A动作。
然后做了B动作。
接着是C动作。
整个过程只有动作描写，没有心理活动，没有眼神交流，没有对话。
```

问题：亲密戏不是动作分解教学。纯爱亲密的核心是「情感交流」，不是身体动作。只写动作不写心理感受和互动，就像在看一份说明书，完全没有甜感。正确方向：重点写感受——他的手在抖，他不敢看你的眼睛，他小声地喊你哥哥，他紧张得屏住呼吸。心理感受永远重于身体动作。

### SP-D03 露骨描写太多太细破坏纯爱氛围

```text
[这里写了非常细致、露骨的身体描写，篇幅很长，细节很多]
完全没有留白，没有克制，把所有东西都摊开给读者看。
```

问题：正太纯爱的尺度讲究「点到即止」「留白有余韵」。露骨描写太多太细，直接把纯爱变成了色情文，甜感全无，只剩下尴尬和不适。正确方向：用暗示、用感受、用反应来代替直白描写——比如写他泛红的耳根，写他抓紧床单的手指，写他细碎的声音，剩下的留给读者自己想象。

### SP-D04 情感还没确认就越界

```text
小星还在犹豫，还没说过「我喜欢你」，甚至还在躲着林晓。
林晓就直接对他做了越界的事情。
```

问题：正太纯爱特别强调「尊重」和「克制」。在对方还没确认心意、还没准备好的时候越界，直接把温柔攻写成了强迫人渣。正确方向：克制是苏点——你想碰他，但你怕吓到他，所以你忍住；你等他主动，等他愿意，等他说「我喜欢你」，等他主动靠近你。越克制，后面的爆发才越甜。

### SP-D05 关键时刻停止写得像被打断很扫兴

```text
就在气氛刚好、两个人都动情的时候。
突然有人敲门/电话响了/着火了/地震了。
然后就没有然后了，场景直接切到第二天。
```

问题：用外力强行打断不是「克制」，是「扫兴」。读者的情绪已经被调动起来了，突然被外力硬切，就像吃到好吃的东西突然被人抢走，只会觉得憋屈。正确方向：停止应该是内生的——因为珍惜他，所以停下来；因为他害羞了，所以不忍心继续；因为尊重他，所以想等他再准备准备。停下来的原因应该是出于爱，而不是作者强行卡进度。

### SP-D06 用第一人称写猥琐心理

```text
（第一人称视角）
看着他的样子，我心里想：「这身体真嫩/以后可以玩很久/反正他也不会反抗」
```

问题：和B03类似，但在亲密场景里写这种心理尤其致命。直接把温柔房东写成了彻头彻尾的人渣，读者会直接吐出来。正确方向：亲密时的第一人称心理应该是珍惜、是紧张、是小心翼翼——「他会不会疼」「我是不是太急了」「他怎么这么可爱」「我想对他好一辈子」。欲望要包裹在温柔和珍惜里。

## E 组：视角与叙事问题

这组是叙事层面的高阶错误，不是不会写，而是写得乱。

### SP-E01 第三人称全知视角乱跳

```text
林晓看着小星，心里想这小孩怎么这么可爱。
小星被他看得脸红，心里想哥哥是不是发现我喜欢他了。
旁边路过的邻居阿姨想，这两兄弟感情真好。
送外卖的小哥想，这家人怎么还不下来拿餐。
```

问题：一段文字里跳了四个人的内心，镜头乱得像打摆子。读者没有稳定的站位，不知道该代入谁，最后谁的情绪都感受不到。正确方向：一个场景里固定一个视角人物——要么全程跟着林晓，写他看到的小星是什么样的；要么全程跟着小星，写他眼里的林晓是什么样的。上帝视角不是不能用，但不要随便跳别人的内心。

### SP-E02 第一人称突然知道自己不可能知道的事

```text
（林晓第一人称）
我看着小星跑回房间的背影。
他肯定是害羞了，心里肯定在想我怎么这么坏，说不定还在偷偷摸自己发烫的耳朵。
```

问题：第一人称的限制就是「你只能知道你看到的、你听到的、你猜到的，但你不能直接知道别人心里在想什么」。直接写「他肯定在想XX」，等于你开了上帝视角，破坏了第一人称的真实感。正确方向：写你观察到的细节——「他跑回房间的时候，耳朵尖红得快要滴血。关门的声音轻轻的，没像平时那样摔门。」让读者从细节里自己推断「他害羞了」，而不是你直接说出来。

### SP-E03 心理描写全是直白说「他很心动」没有具体表现

```text
小星看着林晓做饭的背影，觉得很心动。
他心脏跳得很快，特别喜欢林晓。
```

问题：又是「告诉」而不是「展示」。「心动」是抽象的，读者感受不到。正确方向：用身体反应来表现心动——心跳漏了一拍，耳根悄悄红了，手不自觉抓紧了衣角，眼神黏在他身上移不开，他突然回头看你时你吓得赶紧低下头，假装在看别的地方。

### SP-E04 对话全是「说」没有动作和神态

```text
「我喜欢你。」林晓说。
「我也是。」小星说。
「那我们在一起吧。」林晓说。
「好。」小星说。
```

问题：两个人像机器人在念台词，完全没有画面感。读者看不到他们说话时的样子，听不到他们的语气，感受不到那种紧张又甜蜜的氛围。正确方向：给对话加上动作、神态、语气——「林晓的声音有点抖，他攥着裤腿，不敢看小星的眼睛。」「小星的脸一下子红透了，他揪着衣角，声音小得像蚊子叫：『……我也是。』」

### SP-E05 环境描写和情绪脱节

```text
小星得知林晓要走了，特别难过，一个人躲在房间里哭。
窗外阳光明媚，小鸟在唱歌，花儿开得特别鲜艳，一切都生机勃勃的。
```

问题：难过的时候写阳光明媚，要么是刻意用乐景衬哀情但没衬好，要么就是作者写嗨了忘了人物现在是什么情绪。环境要么烘托情绪，要么反衬情绪，但不能完全没关系地各写各的。正确方向：让环境和情绪呼应——难过时天是阴的，下雨，连风都带着凉意；或者真的要用乐景衬哀情，就要写出「外面越热闹，他越觉得孤单」的对比感。
