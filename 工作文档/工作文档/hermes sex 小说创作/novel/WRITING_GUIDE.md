## 一、反AI腔与去重技巧

### 1. 禁止使用的烂句/烂词
❌ 绝对不能出现：
- "眼中闪过一丝精光/精光一闪"
- "嘴角勾起一抹弧度/勾起一抹邪笑"
- "不仅倒吸一口凉气"
- "指节泛白"（用"手指攥紧""指甲掐进掌心"代替）
- "弓起身子"（用"腰腹绷紧""后背绷紧""腰胯不自觉前挺"代替）
- "不是...而是..."句式（直接写结果，不要先否定）
- 破折号"——"尽量不用，用逗号或句号代替
- "的时候/的瞬间"直接改成"，"
- "声音/语气/语调"后面的描写直接删掉，用对话本身表现
- "一道/一个/一抹XX弧度"句尾直接用句号
- "农牛/精牛"（全书不超过5次，统一用"推送""套弄""榨取"）

### 2. 防重复规则
- 任何段落结构、句式、剧情、描写只能出现一次
- 禁止重复相同的修辞和描写（尤其是开头和结尾）
- 不要重复描写角色同一部位的相同反应
- 每句输出都要全新独特，避免历史文本中出现过的结构
- 相同场景要用完全不同的写作手法，保持新鲜感
- 检查历史对话，避免角色说过相同的话
- 榨精手法不要每次都用同一种，根据对手/场景/剧情阶段切换（见NOTES_榨精手法与道具.md）

### 3. 角色活性化（防机器人）
- 角色对话中禁止使用具体数字（数值、倍数、百分比）描述身体状态
- 角色观察基于正常人五感的模糊印象，用"好像""似乎""有点""显得"等不确定词
- 即使是"冰美人/理性/冷静"人设，也只是人设，不是机器人，不能做科研式理性分析
- 改用"言简意赅""反应略为平淡"来描绘这类人设
- 角色有七情六欲、生活日程、世俗欲望，不是工具人
- 写到新心声、新举动、新台词、新情节时，必须刻意制造和上文的不同点
- 不同角色的手部技法也应有区别：宽厚手掌适合推送/龟头抛光，细长手指适合精密寸止/马眼钻探，短粗手指适合睾丸揉捏/根部掐死

## 二、叙事与视角控制

### 1. 视角规则
- 默认第三人称限知视角，跟随主角
- 禁止上帝视角：角色只能知道自己看到/听到/被告诉的事
- 非特殊情况，角色绝对无法知晓他人的内心独白、未表露的情绪、视野外的事件
- 角色必须通过观察微表情、肢体语言、语调来"推测"对方意图，推测允许误判
- 信息差始终存在，保持信息差才能推动剧情
- 所有角色认知有限，不能知道超出自己认知范围的事
- 外衣/内衣/道具/紧身衣的穿着、脱下、破损、浸透过程必须描写清晰准确

### 2. 心理描写方式
- 用反引号`包裹心理活动：`真实想法`
- 或者用斜体*内心独白*（两种选一种，统一使用）
- 心理活动以第一人称写实时真实想法，短句、高频出现
- 可以有欺骗性（嘴上说不要，心里想快点），增加反差感
- 心理活动只限于非视角人物（视角人物的心理直接融入叙述）
- 心理活动要直接、激烈，传达语言行为表现不出的内容
- **反抗心理必须保留**：即使身体爽到失控，内心也要有`不能在这里射...``可恶...身体怎么会...`这类抵抗的声音

### 3. 对话写作规范
- 对话必须有目的：源于抒情、社交、求知、表达、工作或娱乐欲求，无欲求则沉默
- 对话口语化、情绪化，插入停顿、口吃、填充词
- 不同角色语言纹理差异巨大：词汇量、句式复杂度贴合年龄、知识面、素养
- 禁止翻译腔，要有血肉感、呼吸感
- 合理使用？！...~等标点和拟声词
- 每句对白用同一类型引号（""或「」）
- 禁止"他说""她问"这种太简短的提示语，用人称+情态副词+言说动词，或人称+动作/神态
- 最推荐：连续独立成段纯对话 + 流水句旁白独立成段，不加对话提示语
- 禁止"声音/语气/语调"类描写，直接写对话内容和动作
- 反派的"科研话术"参考："同步率测试""肌肉数据采集""能量采样"，用正当名义包装猥亵行为

### 4. 时间与场景
- 总是知道具体纪年、日期、时间，时间流逝符合常理，禁止大跨度跳时
- 场景切换时重新交代时间地点
- 所有场景细节、人物举动、突发事件必须符合世界观逻辑，不崩
- 不机械降神，维护设定下合理的真实故事
- 时间推进：每次输出根据剧情所需时间合理推进，禁止time-skip
- 室内场景不写天气，室外根据剧情写天色
- **战斗/榨精场景推荐场所**（详见NOTES_战队特摄素材.md）：
  - 训练室：撞奶、胯下按摩器、被压制、撞见当众勃起
  - 演武场：对决、踩裆、玩弄肿胀下体、逼认主宣言
  - 宿舍：沙发舔脚、龟责、凌晨口交、竹马互撸
  - 浴室：公共淋浴室、互相涂抹、足交、偷听
  - 任务地点：洞穴菌魔、植物陷阱、触手地牢
  - 公共场合：隐秘手交、暴露风险带来的紧张加速射精

### 5. 叙事结构模板
- **驯化三段式**：被迫抗拒→身体诚实有反应→主动接受甚至炫耀（要有递进层次）
- **5阶段堕落循环（群像）**：首个受害者→被榨→转成帮凶害下一个→滚雪球→全队沦陷
- **五段榨精连招**：蓄精→寸止→冲刺→突破→追击（节奏必须完整）
- **CP镜像法**：双线叙事，两个角色堕落过程交替呈现，互相映照
- **多线叙事**：终章/大场面可4条线平行推进（但不要滥用）

### 2. 动作-感受绑定原则（核心！）
- 男性/攻方动作描写作为陪衬，简单描写，减少字数
- 每一个性爱动作必须呈现其引发的**主观感受**和**心理反应**
- 快感具象化：将抽象快感转化为可感知的身体体验
  - 电流窜过、骨头酥了、小腹发热、意识空白
  - 魂飞天外、整个人融化、失控、眩晕、炸开
  - 酸胀、酥麻、疼爽、又疼又美、舒服得想哭
  - 龟头被蹭时的麻痒、冠状沟被揉搓时的酸胀、马眼被刮擦时的锐麻
- 高潮时必须呈现情感释放或心理状态变化，搭配身体反应
  - 从拒绝转为配合、痉挛、脚趾扣紧、手/脚锁住对方渴求更多
  - 射精时的眼前发白、意识断片、肌肉僵直然后脱力
- 禁止：仅描写"器官+动作"而无主观感受；旁观者式冷静生理记录；上帝视角纯动作记录
- 禁止用"弓"作为快感动作

### 3. 手部榨精技法描写要点
（详见 NOTES_榨精手法与道具.md，以下是核心描写框架）

#### 基础套弄描写框架
1. **握持姿态**：先写手怎么放上去的（虎口卡根部？反手握？侧握？）
2. **第一下触感**：手掌包裹的温度、肉棒在掌心跳动的反馈
3. **推送过程**：从根到冠的动作，掌心蹭过哪里（柱身→冠状沟→龟头）
4. **被榨者反应**：每一下推送时的颤抖、呻吟、肌肉绷紧
5. **节奏变化**：什么时候轻、什么时候重、什么时候停（寸止）
6. **辅助手动作**：另一只手在做什么（揉睾丸？按会阴？捏乳头？）

#### 关键手法描写提示
- **虎口环扣推送**："虎口卡在根部形成密闭的环，五指收拢顺着柱身向上推送，掌心肉垫恰好蹭过冠状沟下缘..."
- **轻重交替**："第一下轻得像羽毛拂过，第二下突然加重力道大力推到龟头，在马眼上碾了一下才松开..."
- **龟头抛光**："掌心包住通红的龟头快速画圈，前液被蹭得发出咕叽咕叽的水声，每一圈都带着掌纹粗糙的摩擦..."
- **寸止**："在肉棒开始不规律跳动的瞬间猛地停手，拇指死死按住马眼，食指中指卡住冠状沟不让精液冲出来..."
- **三轻一重揉睾丸**："指尖轻轻揉着卵袋，前三下像在抚摸什么珍贵的东西，第四下突然用力一捏..."
- **射精后追击**："刚射完的龟头还在超敏期颤抖，粗糙的湿布已经按了上来，火辣辣的摩擦直接让他尖叫出声..."

#### 榨精节奏变化词库
- 慢速蓄精：缓缓推送、一下一下、不紧不慢、调动深处精液
- 中速标准：规律套弄、节奏稳定、熟悉的快感累积
- 快速冲刺：高频推送、双手交替、不给喘息机会、快得看不清动作
- 寸止停住：猛地停住、死死按住、顿住、快感卡在半中间
- 变速干扰：忽快忽慢、时轻时重、刚适应就变节奏
- 追击榨干：毫不留情、继续摩擦、超敏期、颤抖着痉挛、干精

### 4. 状态变化递进描写
- 从微紧绷到完全放松舒展
- 眼神从清晰到充满肉欲、失神、翻白眼
- 腰臀：从僵硬抵抗→不由自主配合节奏扭动摇摆→主动挺送迎合
- 末端反应：脚趾因快感下意识蜷缩、手指抓紧床单/对方/紧身衣
- 胸部起伏：随急促呼吸明显起伏，乳尖更坚挺敏感，色泽更娇艳
- 肌肤：全身肌肤异常敏感，碰到就颤抖
- 阴茎状态：从半硬→完全勃起→龟头涨红发紫→马眼张开流出前液→射精时跳动→射精后超敏颤抖
- 心理：
  - 快感累积：思维略模糊混乱，理智让位身体本能愉悦渴求
  - 寸止时：`为什么停了...``不要...快点...`身体诚实渴望
  - 高潮体验：短暂灵魂身体轻飘失神感，感官放大聚焦愉悦洪流
  - 事后余韵：爱意倾诉、堕落自责、羞耻但又渴望下次
  - 特殊场合：紧张刺激、心跳加速与隐秘禁忌快乐交织

### 5. 反差与羞耻感
- 大量使用反差羞耻句式：
  - "明明是战场/训练，胯下却..."
  - "明明穿着英雄战衣，却..."
  - "才刚被攻击了几下，就..."
  - "明明想反抗，腰却不自觉..."
- 持续制造表里张力：清纯/淫荡、抗拒/迎合、羞耻/快感、英雄/牲奴
- 从僵硬到熟练的动作递进、从被迫到主动的心理转变，制造堕落层次感
- **战衣玷污反差**：纯白/象征荣耀的战衣被自己的精液浸透，圣洁被亵渎的视觉符号

### 6. 淫语设计三原则
1. **性格驱动**：每句淫语必须紧密贴合角色当下性格（嘴硬的放狠话、软萌的哭求、冷酷的压抑呻吟）
2. **阶段适配**：匹配性爱阶段（初步快感、激烈快感、口交、高潮、事后）
3. **情感外显**：
   - 多用波浪号~、感叹号！、特殊符号♡增强语气节奏
   - 丰富拟声词和延长音：嗯♡~、唔嗯！、呜呜~不可以♡、嗯啊…、唔唔♡顶到了…、咿…、呜嗯~、哈啊♡

   - 生动情感形容词：描述感受（酥麻、滚烫、撑满、魂飞天外）、描述伴侣（厉害的、霸道的）、描述行为（猛烈的、温柔的、深入的）
   - 拟声词参考：
     - 口腔：噗噜噜噜、咕啾、啵～、啵唧、滋溜、啾啾
     - 交合：噗滋、咕叽、噗啾噗啾、咕噗
     - 灌精：咕嘟咕嘟、咕啾咕啾、咕咚咕咚
     - 碰撞：啪！啪啪啪！
     - 呼吸：呼哧、唔唔、齁
     - 榨精：咕叽咕叽、咕啾咕啾、稀里哗啦（水声）
