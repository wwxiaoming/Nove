# 房东风写作流程与运行指南

本文件是房东风的执行手册，不是理论材料。写作或改稿时，严格按以下顺序走。

---

## 第一步：动笔前诊断

动笔之前，先问自己10个问题，确认当前内容匹配房东风风格：

### 1.1 基础设定诊断
- [ ] 视角是不是第一人称"我"？（房东风必须是第一人称）
- [ ] "我"是不是20-30岁的年上男性？（房东/学长/老师/邻居等身份）
- [ ] 故事背景是不是现代都市？
- [ ] 恋爱对象是不是年下正太/少年？
- [ ] 基调是不是甜宠日常为主，不是虐文/强剧情文？

如果以上有任何一个"否"，你可能不适合用房东风，考虑换模块。

### 1.2 主角人设诊断
- [ ] "我"是不是表面正经、内心吐槽的"嘴硬心软"型？
- [ ] "我"有欲望但有没有底线？（不强迫、负责任、会怜惜）
- [ ] "我"的行动是不是以"对他好"为前提，不是只想着满足自己？
- [ ] "我"会不会自我反思？会不会觉得"我是不是太过分了"？
- [ ] "我"有没有幽默感？会不会吐槽自己？

如果主角写出来像个猥琐男/渣男，回去调整人设再写。

### 1.3 当前章节类型诊断
先确定你要写的是什么类型的场景，再匹配对应技法：
- [ ] 日常场景（做饭/吃饭/睡觉/逛街等）→ 去tutorial第五章
- [ ] 情感过渡场景（从日常到亲密）→ 去tutorial第六章
- [ ] 亲密场景 → 去tutorial第七章
- [ ] 新人物加入 → 去tutorial 4.1
- [ ] 多人物互动/争宠 → 去tutorial 4.2
- [ ] 单元剧结构 → 去tutorial 8.1

## 第二步：问题匹配与解法

写作过程中遇到问题，对照下表找解法：

| 问题类型 | 具体表现 | 对应教程位置 | 解决要点 |
|---------|---------|-------------|---------|
| **内心戏问题** | 写出来像猥琐男 | tutorial 2.3 | 加自我反思、加底线、加怜惜、加幽默感，不要把对方当物品 |
| | 内心戏太多太杂 | tutorial 2.1 | 按5个级别来，不要全程都在想色情内容，日常场景只用级别1-2 |
| | 没有反差感 | tutorial 2.2 | 表面正经vs内心吐槽的对比，说的话vs想的话的对比 |
| **对话问题** | 所有人说话一个味 | tutorial 3.2 | 给每个人加语言指纹：口癖、说话方式、常用词 |
| | 对话太书面不自然 | tutorial 3.1 | 怎么说话怎么写，加语气词、加省略、加口癖 |
| | 对话太多没有动作 | 参考good_examples对话部分 | 对话中间加动作、加表情、加内心活动 |
| **新人物问题** | 新人物出场太突兀 | tutorial 4.1 | 选5种加入方式之一，至少铺垫2-3次再正式出场 |
| | 新人物像工具人 | tutorial 4.4 | 给他独特的性格、独特的语言指纹、和其他人的互动 |
| **多人物问题** | 有人全程背景板 | tutorial 4.3 | 用轮C位法、同框分戏法、专属互动法，每个人都要有台词动作 |
| | 争宠写得像宫斗 | tutorial 4.2 | 用6种可爱争宠模式，争宠是闹别扭不是陷害，最后要和好 |
| | 后宫和谐不起来 | tutorial 4.4 | 按5个底层逻辑来：每个人得到想要的、他们之间也有感情、主角公平 |
| **日常问题** | 日常像流水账无聊 | tutorial第五章 | 加细节、加糖点、加小插曲、加人物互动，不要只报菜名 |
| | 日常没有糖 | tutorial第五章每个场景的"糖点设置" | 每个日常场景都要有至少1-2个糖点，不要只是平铺直叙 |
| **过渡问题** | 从日常到亲密太突兀 | tutorial第六章 | 选8种触发点之一，有铺垫、有路径，不要按头 |
| | 过渡太长拖沓 | tutorial第六章 | 触发点→升温→亲密，路径清晰，不要绕太久 |
| **亲密问题** | 写得像色情小说 | tutorial 7.1 | 按5个原则：情感在前、重点在感受、有节奏、符合性格、有事后温存 |
| | 亲密场景没有感情 | tutorial 7.2-7.3 | 加前戏的情感交流、加过程中的对话和心理互动 |
| | 第一次写得太粗暴 | tutorial 7.5 | 慢、温柔、耐心、顾及对方感受、有安抚 |
| | 做完就完了没有温存 | tutorial 7.6 | 必须有事后勤勉：抱、吻、清理、关心、陪伴 |
| | 多人物亲密戏份失衡 | tutorial 7.7 | 每个人都照顾到、有先后但不偏向、符合人物性格 |
| **结构问题** | 单元剧松散 | tutorial 8.1 | 用5种单元剧变体，每个单元有小高潮、有感情升温 |
| | 伏笔埋了不回收 | tutorial 8.2 | 3-5个单元内必须回收，定期检查伏笔 |
| | 多季连载前后不一致 | tutorial 8.3 | 人物成长线、细节一致性、定期回顾、旧人不忘新人融入 |

## 第三步：写作计划模板

写一章/一个单元之前，先填这个模板，再动笔：

```
【当前单元基本信息】
单元类型：日常小事/事件驱动/节日生日/视角切换/过渡承上启下
主要人物：（谁是C位，谁出场）
时间地点：（什么时候，在哪里）
核心事件：（这一章主要写什么事，一句话概括）
感情进展：（这一章结束后，感情升温到什么程度）

【人物设定检查】
□ "我"的状态：（今天心情怎么样，有没有什么事）
□ C位人物的状态：（他今天心情怎么样，有什么特别的事）
□ 其他出场人物的状态：（他们今天怎么样）

【糖点/亮点预设】
1. （至少1个糖点/名场面）
2. （可选，多个更好）

【触发点/过渡预设】
如果有亲密场景，用什么触发点：______
过渡路径：______（怎么从日常走到亲密的）

【伏笔设置/回收】
这一章要埋什么伏笔：______
这一章要回收什么伏笔：______

【结构安排】
开头：______（怎么切入）
中间：______（事件怎么发展，有什么小插曲）
结尾：______（怎么收尾，留什么钩子）
```

## 第四步：写作中实时检查

写的时候，每写完1000字左右，停下来快速检查：

### 快速检查清单（30秒）
1. 有没有出现繁体"房東"？→ 全部改成简体"房东"
2. 主角有没有说/做不符合人设的话/事？
3. 对话是不是符合每个人的语言指纹？
4. 有没有一直在写动作/对话，忘了加内心戏？
5. 有没有一直在写内心戏，忘了写动作/对话？
6. 多人物出场时，是不是每个人都有戏份？
7. 如果在写亲密场景，有没有忘记写情感和感受？

---

## 第五步：改稿检查清单

写完之后，按以下顺序逐项检查，不通过就改：

### 5.1 基础项（必过）
- [ ] 全文都是简体中文，没有繁体"房東"
- [ ] 第一人称贯穿始终，没有乱换视角（除非是故意的视角切换单元）
- [ ] 人物名字统一，没有写错名字
- [ ] 时间线、人物设定前后一致

### 5.2 主角人项（必过）
- [ ] 主角是"嘴硬心软"，不是猥琐男/渣男
- [ ] 主角有底线：不强迫、负责任、会怜惜
- [ ] 主角有幽默感，会吐槽，不是圣人也不是变态
- [ ] 主角的行动符合25岁左右白领的身份，不OOC

### 5.3 对话项
- [ ] 每个人说话方式不一样，有自己的语言指纹
- [ ] 对话口语化，像真人说的话，不书面
- [ ] 对话中间有动作、表情、内心活动，不是只有对话
- [ ] 口癖、口头禅符合人物设定

### 5.4 日常/情节项
- [ ] 日常场景有细节、有糖点，不是流水账
- [ ] 情感过渡自然，有触发点、有铺垫，不突兀
- [ ] 如果有新人物出场，出场自然，不空降
- [ ] 多人物互动时，每个人都有台词和动作，没有人当背景板
- [ ] 争宠写得可爱，不是宫斗，最后会和好
- [ ] 伏笔埋得自然，回收得合理，不挖坑不填

### 5.5 亲密场景项（如果有）
- [ ] 情感在前，动作在后，不是为了写肉而写肉
- [ ] 重点在感受、声音、表情、心理互动，不是器官描写
- [ ] 节奏有快有慢，前戏充足，进入温柔，过程有交流
- [ ] 反应符合人物性格，不是千人一面
- [ ] 第一次尤其温柔、耐心、有安抚
- [ ] 有事后温存：抱、吻、清理、关心、陪伴，不是拔屌无情
- [ ] 多人物亲密时，每个人都照顾到，戏份平衡
- [ ] 整体"甜而不淫"，不低俗不下流

### 5.6 最终通读
- [ ] 通读一遍，有没有不通顺的句子
- [ ] 有没有哪里读起来尴尬、出戏？→ 改
- [ ] 有没有哪里读起来太干、没有感情？→ 加细节加心理
- [ ] 有没有哪里写得太过火、像色情小说？→ 删/改，多加点情感描写
- [ ] 整体读起来有没有"甜"、"暖"、"治愈"的感觉？如果没有，回去加糖

## 附录：常见问题急救方案

### 问题1：写着写着主角变猥琐了怎么办？
**急救：**
1. 马上加一段主角的自我反思："我在想什么呢，他还这么小/他还在生病，我怎么能想这种事。"
2. 让主角做点负责任的事：给他盖被子、去给他倒水、转身去冷静一下。
3. 后面找机会让主角表现出温柔/负责任的一面，抵消猥琐感。

### 问题2：日常写得无聊像流水账怎么办？
**急救：**
1. 加一个小插曲：比如小迪抢菜打翻了碗、小杰和小雨拌嘴、小飞打碎了杯子。
2. 加一个糖点：比如"我"自然地帮小迪擦掉嘴角的饭粒、小雨偷偷给"我"夹菜、小杰从后面蒙住"我"的眼睛。
3. 加一点小暧昧：比如递东西的时候不小心碰到手、对视一眼然后脸红。
4. 加一点内心吐槽："这几个小家伙，吃饭都不安生。"

### 问题3：过渡太突兀，刚还在吃饭突然就上床了怎么办？
**急救：**
1. 找一个合理的触发点（参考tutorial第六章8种触发点）。
2. 补过渡段：比如吃完饭大家一起看电视→看恐怖片小迪往怀里钻→气氛暧昧→慢慢亲密。
3. 不要急，气氛慢慢升温，给够铺垫。

### 问题4：亲密场景写得像动作小说/色情小说怎么办？
**急救：**
1. 删掉1/3的动作描写，换成情感/感受描写。
2. 加对话：问他疼不疼、舒不舒服，说情话。
3. 加心理活动：写"我"有多珍惜他、有多怕弄疼他、有多爱他。
4. 加表情/声音描写：他脸红的样子、他咬嘴唇的样子、他的呻吟。
5. 确保有前戏、有事后温存。

### 问题5：多人物场景有人全程一句话没说怎么办？
**急救：**
1. 给他加一句符合他性格的台词：小迪抢着说"我也要！"，小雨别扭地说"哼"，小杰痞痞地调侃一句，小飞小声说"谢谢哥哥"。
2. 给他加一个动作：小迪在抢吃的，小雨在瞪小迪，小杰在笑，小飞在帮忙递东西。
3. 让"我"和他互动一下：摸一下他的头、问他一句"你觉得呢？"、给他夹菜。
