# 英雄风模块运行规则

这份文件不是教程，而是给模型的执行流程——什么时候调用英雄风模块、怎么诊断写作问题、怎么制定动笔计划。

---

## 一、什么时候必须调用英雄风模块

当任务出现这些信号时，优先调用本模块，而不是直接写：

### 1.1 用户明确提到的关键词

当用户输入包含以下词汇时，必须调用本模块：
- 英雄调教、英雄陷落、正义堕落、英雄被俘虏
- 超级英雄H、战队沦陷、圣骑士堕落、侠客被擒
- 黑暗英雄、调教英雄、宠物英雄、奴化英雄
- 英雄SM、英雄羞耻play、英雄前列腺开发
- 类似"原本正义的角色被反派抓住然后……"的描述

### 1.2 文本特征判断

即使没有明确关键词，如果正在写/改的文本出现以下特征，也应该调用本模块：
- 正义方角色被俘，接下来要写被折磨/被侵犯的情节
- 角色立场要从正义转向堕落，但转变过程写得太突兀
- H场景里有明显的"强者被弱者支配""公众人物被私密羞辱"的反差
- 调教场景写得像纯色情，没有心理描写和张力
- 角色被俘后很快就屈服了，读者觉得"不真实"
- 反派写得太脸谱化，只会说"哈哈哈你叫破喉咙也没用"
- 英雄陷落之后像变了个人，之前的性格特质全没了

### 1.3 不要调用的场景

如果是以下情况，不要调用本模块：
- 纯甜文/纯恋爱文，没有陷落和调教情节
- 纯虐文，主角不是正义英雄，也没有身份反差
- 反派为主角的文，主角一开始就是黑的
- 普通BL文，没有"英雄堕落"这个核心戏剧冲突

## 二、先判断这是哪一层次的问题

不要一上来就说"这段张力不够"。先判断问题属于哪一层，再对症下药。

### 第1层：基础设定问题（动笔前没规划好）

**典型表现**：
- 英雄像个圣人，完全没有缺点和脆弱面
- 被俘理由太牵强："他不小心被打晕了"
- 反派就是个纯变态，没有魅力也没有逻辑
- 还没想好五阶段就开始写调教，写着写着不知道该怎么推进

**优先解决**：
- 回去补三层人设（公众形象/战斗形象/私下脆弱面）
- 重新设计陷阱：不能是硬打不过，要让他"主动"落网
- 给反派设计类型、行事风格、核心歪理
- 先填完"动笔计划模板"再写正文

### 第2层：节奏与阶段问题（写到一半卡壳）

**典型表现**：
- 刚被俘两章就开始叫主人了，后面没东西写
- 写了一万字还在抵抗期，读者已经腻了
- 五阶段跳着走：刚抵抗完直接就屈服了，中间跳过了犹豫和动摇
- 调教场景写了很多，但感觉都是重复的，没有升级

**优先解决**：
- 对照"五阶段速查表"，判断现在写到哪个阶段了
- 检查这个阶段的标志性事件有没有写
- 如果跳阶段了，回去补缺失阶段的关键转折场景
- 如果进度太慢，安排一个"转折点事件"加速推进

### 第3层：张力与羞耻感问题（场景写得平）

**典型表现**：
- H场景就是抽插流水账，没有心理活动
- 脱光了就直接上，羞耻感一层都没铺垫
- 角色没有"嘴上说不要身体很诚实"的矛盾
- 羞辱只会骂"贱货"，不会戳痛点
- 调教完英雄和之前一样，没有任何心理变化

**优先解决**：
- 对照"羞耻感七层递进法"，看看现在写到哪一层了
- 检查有没有"双线并行"：身体快感和精神动摇是不是绑定了
- 给反派写几句戳痛点的台词，不要只会说脏话
- 写完每个调教场景后，加一段英雄的贤者时间自我厌恶

### 第4层：人物弧光问题（陷落前后像两个人）

**典型表现**：
- 屈服之后英雄就变成了纯舔狗，之前的骄傲一点没剩
- 叫主人叫得特别顺，一点挣扎都没有
- 后面的剧情里完全看不出他曾经是英雄
- 为了讨好主人什么都愿意做，连不伤害无辜的底线都没了

**优先解决**：
- 给他设定一个"最后的底线"，无论什么时候都不突破
- 在驯化日常里加几个"闪回"片段，提醒读者他的过去
- 加几个"不甘心"的小细节：攥拳头、声音颤抖、眼神偶尔锐利一下
- 结局不要让他彻底变成肉便器，留一点余味

## 三、子类匹配流程

判断完问题层级后，再判断你写的是哪个子类，匹配对应的参考范本。

### 3.1 子类判断决策树

```
开始
  │
  ├─ 主角是单个英雄？
  │   ├─ 是 → 反派是犯罪集团/黑医/不良少年？→ moecloud少年英雄系（参考《少年英雄》四部曲）
  │   │
  │   └─ 否 → 继续判断
  │
  ├─ 主角是一个团队/战队（3人以上）？
  │   ├─ 是 → 正义战队系（参考《正义战队》）
  │   │
  │   └─ 否 → 继续判断
  │
  ├─ 主角是兄弟/搭档两人？
  │   ├─ 是 → 一方先陷落，另一方来救结果也陷了？→ 神奇少年系（参考《神奇少年》）
  │   │
  │   └─ 否 → 默认按moecloud少年英雄系写
  │
  └─ 结束
```

### 3.2 不同子类的写作侧重点提醒

| 子类 | 重点关注 | 容易踩的坑 | 优先参考范本 |
|------|---------|-----------|-------------|
| **moecloud少年英雄系** | 五阶段完整递进、身份反差、宠物化、直播/拍卖展示 | 陷落太快、道具堆砌、身份公开太早 | 《少年英雄》主篇看完整节奏，《零》看语言屈服线 |
| **正义战队系** | 团队羁绊先立住、差异化陷落、内部背叛、连锁打击 | 所有人陷落方式雷同、背叛太突兀、团队互动不够 | 《正义战队》看绿→黄→黑的差异化写法 |
| **神奇少年系** | 兄弟羁绊、弟先陷哥后陷、机械调教设计、用陷落者控制未陷落者 | 弟弟陷落太快、机器调教没设计感、兄弟互动不够 | 《神奇少年》看托弗救多米段落 |

## 四、匹配素材的最低配置

每次写/改英雄风场景，至少匹配：

- 3-5 个正例（从 `good_examples.md` 选）
- 2-3 个反例（从 `bad_examples.md` 选，避免踩坑）

### 4.1 按场景选正例的映射表

不要每次都从整份例库里乱翻。优先按下面这张表选：

#### 如果你在写"被俘/陷阱"场景

先看：
- 正例：`good_examples.md` A组（被俘与陷阱设计）
- 反例：`bad_examples.md` A组（设定硬伤类错误）

重点学：
- 怎么设计"让英雄主动落网"的陷阱
- 怎么建立被俘瞬间的力量反转
- 怎么用英雄的原则反过来对付他

#### 如果你在写"初期拘束/第一次羞耻"场景

先看：
- 正例：`good_examples.md` B组（初期拘束与羞耻）
- 反例：`bad_examples.md` B组（节奏问题类错误）

重点学：
- 拘束姿势的可视化描写
- 装备/制服被剥离的象征意义
- 第一层羞耻感（失去行动自由）怎么写

#### 如果你在写"身体开发/调教中"场景

先看：
- 正例：`good_examples.md` C组（身体开发与调教）
- 反例：`bad_examples.md` C组（H场景写法问题）

重点学：
- 身体开发的递进节奏
- 怎么写"痛但有快感"的矛盾感
- 前列腺高潮、尿道开发这些具体部位的写法

#### 如果你在写"动摇/语言屈服"场景

先看：
- 正例：`good_examples.md` D组（动摇与屈服节点）
- 反例：`bad_examples.md` B组、D组

重点学：
- 从"不"到"请"到"主人"的语言变化线
- 边缘控制怎么写出张力
- 压垮骆驼的最后一根稻草怎么设计

#### 如果你在写"公众展示/身份公开"场景

先看：
- 正例：`good_examples.md` E组（公众羞辱与身份公开）
- 反例：`bad_examples.md` D组、E组

重点学：
- 见证人越多羞耻感越强的写法
- 镜面/摄像头/直播屏幕的运用
- 身份精确信息打击的冲击力

#### 如果你在写"驯化/长期相处"场景

先看：
- 正例：`good_examples.md` F组（驯化与人物弧光保持）
- 反例：`bad_examples.md` E组（人物弧光断裂类错误）

重点学：
- 日常驯化的小细节
- 怎么保留最后的底线
- 闪回和不甘心小细节怎么加

### 4.2 选材补充规则

1. 先从对应场景组挑3个正例。
2. 再从相邻组补1-2个。
3. 如果一个场景同时跨两组（比如公众展示+身份公开），先解决更靠前的硬伤（先写好公众羞辱的层次感，再处理身份公开的暴击）。
4. 反例优先找和你现在写法最像的错误——比如你发现自己写得太快，就去看"陷落太快太突兀"那类反例。

## 六、写作诊断流程（改稿时用）

当你要修改一段已有的英雄风文字时，按这个顺序诊断：

### 第一步：通读全文，回答5个问题
1. 现在故事进行到五阶段的哪一个？
2. 这个阶段的标志性事件写了吗？
3. 羞耻感写到哪一层了？有没有跳级？
4. 身体快感和精神动摇是不是绑定了？
5. 人物弧光还在吗？能看出他曾经是英雄吗？

### 第二步：定位问题类型
根据第一步的答案，对应到"四层问题"（基础设定/节奏/张力/弧光），找出最严重的1-2个问题。

### 第三步：匹配正反例
从good_examples.md找3个对应场景的正例，从bad_examples.md找2个你现在正在犯的错误的反例。

### 第四步：修改优先级排序
不要一次性全改，按这个优先级来：
1. 先改基础设定硬伤（如果英雄三层人设都没立住，先回去补）
2. 再改阶段跳级问题（如果刚被俘就叫主人，回去补犹豫和动摇期）
3. 再改张力问题（增加双线心理描写、反派戳痛点台词、羞耻感层次）
4. 最后润色H场景细节（动作、感官、节奏）

## 七、专项硬规则

这些规则无论写什么子类、什么阶段都必须遵守，写完逐条检查：

1. **英雄被俘不能是"不小心被打晕了"**，必须是因为他的原则/软肋被利用，主动走进陷阱。
2. **五阶段不能跳**，不能刚抵抗完就直接屈服，犹豫和动摇期是灵魂。
3. **羞耻感不能跳级**，不能刚被绑就直接全城直播身份公开。
4. **每一次身体高潮，必须伴随一次精神动摇**，不能射完跟没事人一样。
5. **反派不能只会骂脏话**，必须有自己的风格和戳痛点的台词。
6. **即使屈服了，英雄也不能变成没有灵魂的舔狗**，要保留最后的底线和偶尔的闪回。
7. **调教场景不能是重复的活塞运动**，每一次都要有新内容、新升级、新打击。
8. **不要为了黑暗而黑暗**，每一次折磨都要推进剧情或人物，没有目的的折磨不要写。
9. **道具不能一上来全用上**，要按阶段递进出现。
10. **结局不能"英雄被救然后幸福生活"**，必须有转变或创伤，人物不能回到从前。

## 八、写完后的自查问题

写完一个场景/一个章节后，问自己这10个问题。只要有3个以上答"否"或"不确定"，就回去重写：

1. 读者能看出现在是五阶段的哪一个吗？
2. 这个场景有没有推进人物关系或陷落进程？（如果去掉不影响剧情，就是多余的）
3. 英雄的心理活动写了吗？有没有"嘴上不要身体诚实"的矛盾？
4. 反派的台词符合他的人设吗？有没有戳中英雄的痛点？
5. 羞耻感比上一个场景升级了吗？
6. 这个场景的调教手段比之前更进了一步吗？（不能一直重复）
7. 身体快感和精神动摇绑定了吗？（高潮后有没有自我厌恶？）
8. 还能看出这个角色曾经是英雄吗？（有没有残存的骄傲/底线？）
9. 有没有给读者留"可惜了"的感觉？（而不是只有"好惨"或"好黄"）
10. 如果删掉这段，故事会不会少了关键的一步？（如果不会，这段就是注水）
