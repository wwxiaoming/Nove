# 正太综合题材写作教程

这份教程是正太子模块的核心主文件，覆盖四个子类的文风详解、通用写作技巧和调用决策树。

## 第一部分：模块定位

### 1.1 什么时候用这个模块

正太纯爱文风模块组有三个主风格：
- **星辰风**：萌系奇幻、单CP纯爱、杀手设定
- **房东风**：第一人称、现代日常、后宫甜宠
- **英雄风**：英雄陷落、正义vs堕落、黑暗调教

当以上三种主风格都无法很好匹配你的写作需求时，就用本模块——这是一个按题材分类的综合工具箱。

### 1.2 四个子文风的适用边界

| 子文风 | 什么时候用 | 什么时候不要用 |
|--------|------------|----------------|
| 日常/校园纯爱 | 现代背景、平等关系、日常甜宠、伪兄弟/同学/邻居 | 奇幻设定、系统流、黑暗背景 |
| 综漫爽文 | 综漫同人、快穿、系统、爽文、NP总攻、多世界穿越 | 单CP写实日常、无系统纯爱 |
| 黑暗产业 | 会所/拍卖/交易设定、黑暗背景、职业视角、规则构建 | 校园甜宠、平等恋爱关系 |
| 鬼畜重口 | ⚠️ 仅学习节奏张力、感官描写、层层递进技法 | 直接模仿重口内容、纯爱场景 |

### 1.3 使用原则

1. **一子类为主，其他为辅**：一篇文选定一个主基调，其他子类只借技法，不借核心设定
2. **技法可迁移，内容需筛选**：鬼畜风的节奏可以学，但不要搬重口内容到纯爱文
3. **年龄感永远第一**：不管什么风格，正太的年龄感（说话方式、行为逻辑、心理状态）必须精准
4. **关系决定互动模式**：伪兄弟和同学的相处方式完全不同，主仆和陌生人的亲密尺度天差地别

## 第二部分：各子类文风详解

### 2.1 日常/校园纯爱风

代表作品：《宝贝今年十四岁》、《运动裤下的秘密》

这是最常用、最基础的正太写法，适合80%的现代背景正太纯爱场景。

#### 2.1.1 核心特征：平等关系+生活化细节+慢热铺垫

日常纯爱风的核心是"**像真的一样**"——两个人的互动要像现实中真的会发生的那样，情感推进要自然，糖点要藏在生活细节里。

**与其他风格的区别**：
- 不是房东风的第一人称后宫（房东风多CP，日常纯爱可以单CP）
- 不是星辰风的萌系奇幻（日常纯爱无超能力/奇幻设定）
- 不是英雄风的黑暗调教（日常纯爱是平等恋爱，不是调教陷落）

#### 2.1.2 视角选择：两种常用写法

**写法A：第一人称攻视角（《宝贝今年十四岁》）**
- 优点：代入感强，内心OS丰富，可以写很多"嘴上说不要身体很诚实"的吐槽
- 适合：年龄差较大（如大学生×小学生）、攻视角为主的故事
- 技巧：
  - 加入明显的内心吐槽标记（如《宝贝》里的"—__—|||"）
  - 可以写攻的"色狼心理"但要有度，不要写成真变态
  - 通过攻的眼睛来描写受的可爱：皮肤触感、表情变化、小动作
  - 示例技法参考：罗冈第一次抱黎昀坐腿上的心理活动——"肉贴着肉，有一种说不出的快感"，然后马上自我反省又忍不住继续，这种矛盾心理很真实

**写法B：第三人称多视角（《运动裤下的秘密》）**
- 优点：可以同时写多对CP，群像戏精彩，每个人物都有自己的秘密
- 适合：校园背景、欢喜冤家、多CP群像
- 技巧：
  - 每章可以切换视角，但不要太频繁（一章1-2个视角为宜）
  - 通过"秘密"来制造张力：宋捷"千人斩其实是处男"的秘密贯穿始终
  - 欢喜冤家的对话要斗得起来，互相怼但又留余地
  - 示例技法参考：古云风揭穿宋捷是小处男那段——"小处男"三个字从背后传来，宋捷背脊一寒，这种戏剧冲突很抓人

#### 2.1.3 伪兄弟/邻家/同学关系：糖点设计技巧

**伪兄弟关系糖点（《宝贝今年十四岁》）**
- **合理同住的理由**：罗冈租房子住到黎昀家，不是凭空住进来的——救了黎昀→黎妈妈感恩→刚好有房子出租→顺理成章住进来
- **日常肢体接触的合理性**：
  - 坐腿上："只有一张凳子"（非常合理的借口！）
  - 一起洗澡："爸妈快回来了，不可以让他们知道玩这么晚"
  - 一起睡觉："晚上一个人好冷"
  - 帮忙换衣服："怕你上学迟到"
- **关键物件作为糖点载体**：
  - DV机：偷拍→被发现→成为两人共同的秘密
  - 白金情侣戒指：刻名字→戴在项链上→被父母发现时的小慌乱
  - 日记本：被偷看→发现对方也喜欢自己→确认关系
- **年龄差的处理**：
  - 攻（罗冈）：大学生，会赚钱、会照顾人、有点小腹黑但真心对受好
  - 受（黎昀）：12-14岁小学生/初中生，会害羞、会傲娇、会主动但又有底线（"要等到你生日"）
  - 不要让年龄差变成"攻完全掌控受"，受也要有自己的主见和小脾气

**同学/欢喜冤家糖点（《运动裤下的秘密》）**
- **初始冲突要足**：两校合并的对立背景→宿舍分配到一起→第一天就打架→见面就互怼
- **反差萌设计**：
  - 宋捷：号称"千人斩"的运动部长→其实是有隐疾的小处男
  - 古云风：神秘土包子→其实有超能力，早就看穿宋捷的秘密
  - 雷劲：肌肉大块头、花花公子→其实是处男，会被小白云管得死死的
  - 尹天琪：洁癖优等生→其实是傲娇，会被雷劲吃得死死的
- **校园场景利用**：
  - 宿舍：抢床、叫起床、浴室撞见秘密
  - 课堂：座位是前后桌、上课传纸条、打架被罚
  - 食堂：两派人马对峙、抢位子、互相瞪
  - 社团/活动：跆拳道社、联欢晚会、打赌追人
- **超能力/奇幻元素的克制使用**：
  - 古云风的"恋爱星"能力不要滥用，主要用来推动感情线和制造笑点
  - 不要让超能力解决所有问题，感情还是要靠日常相处

#### 2.1.4 温柔亲密场景节奏：循序渐进是关键

日常纯爱的亲密场景**绝对不能上来就做全垒打**，要像煮开水一样，慢慢升温。

**标准推进节奏（参考《宝贝今年十四岁》）**：
1. **第一阶段（0-1个月）：初次接触，心动但克制**
   - 坐腿上玩电脑→下体不自觉顶到→手不自觉摸上去→受躲开→攻道歉
   - 这个阶段的关键词：试探、慌乱、自我反省、"我是不是禽兽"
   - 要有"犯错→道歉→被原谅"的小波折

2. **第二阶段（1-3个月）：边缘接触，心照不宣**
   - 一起游泳→泳池里乱摸挠痒→一起洗澡→帮洗身体→摸到关键部位→受害羞但不拒绝
   - 关键词：暧昧、默认、心照不宣、"大家都是男生没关系"
   - 这个阶段可以有手交、口交，但不要做到最后

3. **第三阶段（3-6个月）：确认关系，保留底线**
   - 日记被偷看→发现互相喜欢→告白→确认关系→舌吻→受坚持"要等到生日"
   - 关键词：告白、确认、甜蜜、约定、期待
   - 这个阶段可以互相抚慰，但受要有自己的底线和坚持

4. **第四阶段（6个月+）：完全拥有，水到渠成**
   - 生日当天→礼物（戒指）→气氛到位→第一次
   - 关键词：承诺、仪式感、水到渠成、甜蜜收尾
   - 第一次之前一定要有足够的情感铺垫和仪式感

**亲密场景的描写要点**：
- **注重感受，不是动作**：不要写"手指动了几下"，要写"他的呼吸变急了"、"耳朵红得要滴血"、"身体微微发抖"
- **反应比主动重要**：受的反应（害羞、躲闪、小声呻吟、主动抱上去）比攻的动作更有甜感
- **事后细节要甜**：帮清理、抱在怀里睡觉、第二天早上的撒娇、一起吃早餐
- **不要省略语言互动**："疼不疼？"、"没关系，我喜欢你"、"你是我的了"这类话一定要有

### 2.2 综漫爽文风

代表作品：《[综漫]正太控的世界旅行》

#### 2.2.1 核心特征：系统+金手指+爽点密集+不OOC

综漫爽文的核心是"**爽**"——主角要厉害，正太要可爱，感情线要甜，战斗要炫酷，NP要和谐。

#### 2.2.2 爽点节奏：每三章一个小爽点，每卷一个大爽点

**开场就要给金手指**：
- 不要让主角熬十几章才变强，第一章就要把能力给足
- 《世界旅行》第一章：穿越到军火女王战场→获得"绝对掌控"能力→子弹擦着身过毫发无伤→瞬间制服约拿→直接加入蔻蔻团队
- 开场爽点三连：穿越→满级能力→见到第一个攻略对象

**爽点类型设计**：
1. **能力碾压爽**：别人做不到的事主角轻松做到
   - 例：别人躲子弹都难，主角在子弹雨中闲庭信步
   - 例：别人练几年才能用的枪，主角碰一下就比专家还厉害
2. **正太倒贴爽**：冰山正太只对主角融化，主动亲近
   - 例：约拿对谁都面无表情，只对王维脸红、只跟王维说话、主动告白"我喜欢王维"
3. **剧情改变爽**：因为主角介入，原著的悲剧被避免
   - （《世界旅行》里虽然还没写到，但综漫文的核心爽点之一就是改变意难平）
4. **吐槽搞笑爽**：主角和系统、主角和其他角色的搞笑互动
   - 例：王维和蔻蔻一起滚桌子哭"不要扣薪水啊"
   - 例：约拿认真说"武器保养的钱我自己付，我养你"

**爽点节奏表**：
| 位置 | 爽点类型 | 示例（《世界旅行》） |
|------|----------|---------------------|
| 第1章结尾 | 能力展示爽 | 绝对掌控能力启动，毫发无伤躲子弹 |
| 第2章结尾 | 角色互动爽 | 成功加入蔻蔻团队，抱住约拿蹭脸 |
| 第4章结尾 | 战斗+糖爽 | 约拿击杀敌人，和蔻蔻碰拳，王维亲他脸颊 |
| 第5章结尾 | 告白爽 | 约拿主动告白"我喜欢王维" |
| 每卷结尾 | 大团圆爽 | 收服该世界正太，和谐后宫+1 |

#### 2.2.3 多世界切换技巧：每个世界要有"锚点"

综漫文最容易写崩的地方就是世界切换太乱，读者记不住谁是谁。

**每个世界要有清晰的锚点**：
1. **世界标签**：军火女王→战争、武器、约拿；柯南→推理、死神小学生；火影→忍者、木叶
2. **核心正太**：每个世界重点写1-2个正太，不要贪多
3. **独特能力/道具**：主角在这个世界获得什么新能力/认识什么新朋友
4. **离开时的收尾**：不要写完就走，要给这个世界的正太一个交代（可以带在身边，可以约定以后回来）

**切换世界的过渡方法**：
- 系统提示音："叮！第一个世界攻略完成，即将进入第二个世界——×××"
- 白光/眩晕/传送感：简单直接，不用写太多
- 不要在世界切换上拖泥带水，一两段搞定，马上进入新世界的剧情

#### 2.2.4 原著人物不OOC技巧：抓核心性格，改关系不改人设

综漫同人最大的雷区就是OOC（Out Of Character，人物崩坏）。

**不OOC三原则**：
1. **抓核心性格，其他可以微调**：
   - 例：《军火女王》里的约拿——核心性格是"面无表情、少言寡语、讨厌武器但又离不开武器、内心柔软"
   - 王维的介入没有改变这些核心性格：约拿依然话少、依然面无表情、依然熟练用枪
   - 改变的只是"对王维的态度"：从警惕→不讨厌→喜欢→依赖，这是在核心性格基础上的关系变化
2. **用原著台词/行为逻辑**：
   - 例：约拿认真提醒王维"在战场上不能拉上保险，这么做很容易会死"——这完全是原著里约拿会说的话
   - 例：蔻蔻的"为了世界和平"——原著台词，直接用，一下子就有那味儿了
3. **原著人物不要变成主角的舔狗**：
   - 蔻蔻依然是那个狡黠、有主见的军火女王，不是见到主角就犯花痴
   - 团队里的其他人（雷姆、法尔梅等）依然是有自己性格的配角，不是背景板
   - 即使是对主角有好感的正太，也要有自己的坚持和底线（约拿一开始也是有警惕的）

**判断OOC的简单方法**：如果你把主角的名字遮掉，这个人物的言行是不是还像原著里的他/她？如果不像，那就是OOC了。

#### 2.2.5 爽文风格亲密场景：短、甜、不拖沓，服务于爽点

爽文里的亲密场景不要写得太长太细，那会拖慢爽点节奏。

**爽文亲密场景三要素**：
1. **时机要对**：战斗胜利后、告白成功后、久别重逢后——在爽点之后立刻发糖
2. **要甜不要虐**：少写疼痛犹豫，多写主动亲近、脸红、撒娇
3. **服务于人物关系**：亲密场景要体现"这个冰山正太只对主角这样"的爽感
   - 例：约拿对谁都冷冰冰，被王维抱住不反抗、被亲耳朵会红、主动说"我喜欢王维"——这种反差就是爽点
4. **NP后宫要和谐**：不要写宫斗、吃醋、争宠，大家都是好姐妹（？）
   - 《世界旅行》里明确说了"受们都不蛋疼别扭，有的还很主动，不吃醋，很和平"
   - 后来的正太看到前面的，应该是"哇这个哥哥好可爱我们一起吧"，而不是"你这个狐狸精抢走了我的王"

### 2.3 黑暗产业风

代表作品：《正太会所》

#### 2.3.1 核心特征：设定完整+规则清晰+职业冷静+日常与交易交织

黑暗产业风的核心是"**真实感**"——你构建的这个地下世界要像真的存在一样，有完整的规则、流程、等级制度，人物在这个体系里的行为要符合逻辑。

#### 2.3.2 设定构建方法：从外到内，层层递进

不要一上来就把所有设定倒给读者，要像剥洋葱一样，跟着主角的视角慢慢展开。

**第一层：入口与表象（第1章）**
- 位置：A市闹市街的"Relaxed"酒吧——越危险的地方越安全
- 进入方式：吧台自主平板输入会员ID→酒保给磁卡→电梯刷卡
- 电梯秘密：不刷卡是普通电梯（1楼/负1楼超市），刷卡才会升上去到会所楼层
- 给读者的第一印象：这地方很隐蔽、很专业、不是随便什么人都能进

**第二层：业务模式（第1章/设定补充章）**
- 三大板块要讲清楚：
  1. **地下拍卖**：会所把正太卖给客人（永久/长期），正太来源是拐卖、遗弃、走失的
  2. **个人交流**：客人和正太私下交易，正太拿80%，会所抽20%，正太可以拒绝
  3. **会所交易**：会所提供正太短时间所有权，正太拿30%
- 会员等级：普通会员→高级会员→最高级别会员（董事长级，跨国企业老总）
- 正太等级：普通正太→标致听话的→"极品"资质的（拍卖）
- 退出机制：18岁成年后回会所删资料，可以选择跟随客人或者获得自由

**第三层：交易流程（小凛接待杨叔叔的完整7天）**
- 第1步：会所通知（"小凛，明叔，又有客人看中了你"）
- 第2步：见面谈价（贵宾室7206，看资料，谈价格和时间）
- 第3步：安排时间（"周1-周5我要上学，下午放学后可以陪你，周末可以过夜"）
- 第4步：日常相处（陪见客户、吃炒面、玩游戏、打电话骗妈妈"去同学家睡"）
- 第5步：准备工作（灌肠、道具箱：针筒、跳蛋、润滑液）
- 第6步：亲密场景（温柔、有耐心、照顾受的感受）
- 第7步：事后清理（洗澡、灌肠清理精液、抱去睡觉）
- 第8步：日常延续（第二天早上一起吃早餐、给零花钱）

#### 2.3.3 等级制度描写：用细节体现，不是用嘴说

不要直接写"会所等级森严"，要通过细节让读者感受到：
- 磁卡：普通会员的卡只能到对应楼层，高级会员可以去更高的楼层
- 资料详细程度：杨叔叔看的小凛资料——姓名、年龄、身高体重、喜好、学校、接客经历（"后面开庖过吗？""嗯，已经被2个叔叔插过了"）
- 道具箱：银色ST字样箱子，会所统一提供，客人不用自己带
- 价格谈判：小凛看一眼价格就接受了，说明他心里有底价，不是什么价格都接
- 正太的职业化表现：叫"杨叔叔"、如实回答接客经历、服务态度好、"怕你觉得我就是为了钱所以不穿衣服"——这些细节都体现他是"专业"的

#### 2.3.4 行业规则：哪些能做，哪些不能做

写黑暗产业一定要有规则，没有规则的世界是假的。
- **自愿原则**（至少表面上）：拍卖区"拒绝正太是被逼的状态下交易"；个人交流区正太可以拒绝客人
- **不影响正常生活**：小凛平时正常上学，只在放学后/周末接客，妈妈以为他去同学家
- **抽成比例明确**：个人交流20%，会所交易30%——写清楚，不要含糊
- **保密原则**：会所总部神秘，警方无数卧底都无法取缔，客人和正太的资料保密
- **退出机制**：18岁必须退出，可以选择自由或者跟客人走——这条很重要，让黑暗里有一点光

#### 2.3.5 会所场景亲密写法：温柔专业，交易中的温情

《正太会所》里的亲密场景有一个很有意思的特点：虽然是交易关系，但杨叔叔很温柔、很照顾小凛的感受，不是粗暴的。

**会所亲密场景要点**：
1. **准备工作要足**：灌肠要反复3次直到排出清水，润滑要涂够，扩张要慢慢来
2. **照顾受的感受**："每进入一点就停下来让小凛适应，防止疼痛"，"碰到前列腺时小凛会轻哼，就多刮那个地方"
3. **事后清理负责**：射了之后帮擦干净，抱去浴室再灌肠清理精液，不要做完就扔一边
4. **日常温情细节**：帮擦嘴角的布丁、摸头、买早餐、给零花钱——这些小细节让交易关系有了温度
5. **受的职业化表现**：小凛会主动说"都听叔叔的"，会配合，但也不是完全没有感觉（会脸红、会有反应）

**注意尺度**：黑暗产业风不是鬼畜重口风，亲密场景要"专业的温柔"，不是"虐待式的粗暴"。可以写交易关系，但不要美化犯罪，也要写出这个体系的残酷性（正太来源有拐卖的）。

### 2.4 鬼畜重口风

代表作品：《正太滋味》

⚠️ **重要提示**：本子类仅用于学习**节奏把控、张力营造、感官描写**技法，不鼓励直接模仿重口内容、非自愿情节、极端设定。所有技法都可以迁移到纯爱场景使用。

#### 2.4.1 节奏把控：层层递进，不给读者喘息机会

《正太滋味》的节奏是教科书级别的——从藤蔓出现到最终高潮，每一步都在升级，读者（和祐音）的情绪被一步步拉紧，直到最后爆发。

**标准层层递进节奏（以藤蔓侵犯祐音为例）**：
1. **第1层：舔弄全身，建立恐惧**（温柔但诡异的前戏）
   - 藤蔓舔脸、舔嘴唇、钻嘴巴、缠脖子
   - 舔胸上的小红点，慢慢逗弄
   - 祐音的反应：哭、挣扎、害怕，但不知道接下来会发生什么
2. **第2层：侵入前端，第一波疼痛**（突破第一道防线）
   - 藤蔓缠上青芽（前端），慢慢揉弄
   - 细藤尖端戳刺顶端的小口
   - 猛地刺进小管里！→祐音痛得弓起身子大哭
   - 关键词：痛、哭喊、挣扎，但藤蔓不听
3. **第3层：瞄准后穴，更大的恐惧**（突破第二道防线）
   - 更多藤蔓过来，扳开小臀瓣
   - 先轻轻舔弄嫩穴，让它痒、让它张开
   - 祐音的反应："好痒，走开"→下面又热又涨又想尿尿→矛盾的快感开始出现
4. **第4层：侵入后穴，疼痛与快感交织**（真正的入侵）
   - 粗大藤蔓猛地闯进嫩穴！→撑裂、流血、祐音尖叫大哭
   - 慢慢抽插，寻找前列腺→碰到的时候祐音忍不住哼出来
   - 前面的藤蔓、嘴里的藤蔓、耳朵脖子的藤蔓一起动→全身敏感点都被照顾
   - 祐音的反应：从哭喊→无意识呜咽→身体开始有快感反应
5. **第5层：哥哥出现，反转与背德感**（心理张力拉满）
   - 祐音以为救星来了，伸手求救
   - 结果哥哥不是来救他的！哥哥早就和藤蔓有染！
   - 藤蔓还去弄哥哥，哥哥也有反应、也在呻吟
   - 祐音的反应："哥哥怎么了？为什么他看起来很舒服？"→好奇、动摇
6. **第6层：哥哥亲自上手，彻底沉沦**（背德+快感双重高潮）
   - 哥哥哄祐音"跟哥哥玩好玩的游戏"
   - 哥哥亲他、摸他、含他的青芽
   - 祐音第一次射精了！
   - 哥哥的东西抵在穴口，刺穿！
   - 藤蔓还在帮忙，前后夹攻
   - 祐音："哥哥没骗我，真的好舒服"→彻底沉沦
7. **第7层：双重高潮，结束但余韵留**（爆发）
   - 两人同时高潮
   - 祐音昏过去
   - 藤蔓还舍不得走，吸液体
   - 哥哥笑着说"他醒来会肯的"→留余韵，后续还会继续

**节奏技法迁移到纯爱场景**：
- 即使写纯爱第一次，也可以用这个层层递进的节奏
- 不要一上来就进入，要从亲吻→摸胸→摸腿→脱衣服→边缘→扩张→进入→高潮，一步步来
- 每一步都写受的反应，每一步都比上一步更亲密
- 中间可以有停顿、有犹豫、有安慰，不要全程高速

#### 2.4.2 张力营造：三种核心张力

**张力1：疼痛vs快感的拉锯**
- 一开始是100%痛，痛到哭
- 慢慢变成痛50%+痒50%
- 再变成痛30%+快感70%
- 最后变成"好痛但是好舒服"→主动索求
- 技法：不要让快感来得太快，要有拉锯过程，这个挣扎过程最有张力

**张力2：纯真vs淫乱的反差**
- 祐音只有8岁，什么都不懂，以为是"好玩的游戏"
- 他越纯真、越信任哥哥，背德感就越强
- 他的反应都是本能的——身体有快感就会哼、就会扭动，不是他"淫荡"，是身体诚实
- 技法：受的年龄越小、越纯真，反差张力越大，但注意尺度，纯爱文里可以用"害羞、不懂、本能反应"来营造这种反差，不要写太极端的年龄

**张力3：期待vs恐惧**
- 祐音一边怕得哭，一边身体又忍不住有反应
- 一边喊"不要"，一边穴口又在张开、在吸藤蔓
- 这种"嘴上说不要身体很诚实"的张力，写好了非常抓人
- 技法：不要让受一开始就完全愿意，要有抗拒、有动摇、有半推半就，这个过程最甜/最带感

#### 2.4.3 重口场景写法：感官描写五步法

⚠️ 再次强调：以下是**感官描写技法**，纯爱文里可以简化使用，不要照搬重口内容。

**五感调动法**：
1. **视觉**：写看到的东西——藤蔓滑腻的液体、身体上的红痕、穴口张开的样子、对方的表情（哭红的眼睛、泛红的皮肤）
2. **触觉**：这是最重要的——
   - 温度：热的、烫的、凉的（润滑液是凉的，身体是热的）
   - 触感：滑腻的、柔软的、硬的、发烫的、麻麻的、酸酸的、涨涨的
   - 痛感：尖锐的痛、钝痛、胀痛、撕裂感
   - 快感：酥麻、窜过脊椎的电流感、前列腺被碰到的酸麻
3. **听觉**：
   - 声音：水声（舔弄的声音、抽插的水声）、哭声、呻吟声、喘息声、"不要"、"好痛"、"嗯啊"
   - 自己的心跳声、对方的心跳声
4. **嗅觉**：味道——血味、体液味、汗液味、对方身上的味道
5. **味觉**：如果有亲吻/口交，写嘴里的味道——甜味、咸味、铁锈味（血）

**感官描写迁移到纯爱**：
- 第一次接吻：嘴唇的柔软度、舌头的温度、呼吸的味道、心跳的声音
- 第一次摸：皮肤的触感（滑、嫩、热）、受发抖的反应、呼吸变急
- 第一次进入：紧、热、包裹感、受的痛哼、你自己的心跳
- 不要只写动作，要写五感接收到的所有信息

## 第三部分：通用正太写作技巧

不管用哪个子类，这些通用技巧都必须掌握。

### 3.1 年龄感精准把控方法

正太题材最容易崩的就是年龄感——12岁写出20岁的台词，14岁跟24岁一样成熟，这是致命伤。

#### 3.1.1 不同年龄段正太的核心特征

| 年龄 | 身体特征 | 语言特征 | 心理特征 | 亲密尺度参考 |
|------|----------|----------|----------|--------------|
| **8-10岁** | 未发育，很小只，皮肤极嫩，第二性征完全没有 | 说话奶声奶气，会用叠词，会撒娇，不懂"喜欢"之外的感情 | 完全信任依赖的人，不懂性是什么，身体有反应但不知道怎么回事 | ⚠️ 纯爱文里这个年龄只写亲亲抱抱，不建议写实质性内容；参考《正太滋味》仅学技法 |
| **11-12岁** | 刚开始发育，细细长长，可能开始长少量毛（也可能没有），包皮还没退 | 会有点小傲娇，会说"我才不是小孩子"，但行为还是小孩，会害羞会脸红 | 开始懂一点"喜欢"，但还是懵懵懂懂，会好奇，会被吓到，对信任的人不设防 | 可以写边缘（互相抚慰、口交），要有充足情感铺垫，受可以有底线（"要等生日"） |
| **13-14岁** | 发育中，有明显勃起，可能已经有精液，毛开始长但不多 | 开始有主见，会别扭、会口是心非，会说"不要"但其实想要，会吃醋 | 懂喜欢是什么了，会主动但又害羞，有自己的坚持和底线，会为对方考虑 | 可以写完整关系，但一定要有仪式感、有告白、有确认关系，不要随便就做 |
| **15-17岁** | 接近成年但还没到，身体已经长开但还带着少年感 | 说话接近成年人，但还是会有少年气，会傲娇会别扭，偶尔会撒娇 | 有独立人格，能为自己的决定负责，会主动追求，会规划未来 | 尺度可以接近成年人，但还是要写出少年的青涩感 |

#### 3.1.2 年龄感判断三问

写任何一段对话、任何一个动作之前，先问自己三个问题：
1. **这个年龄的孩子会说这句话吗？**
   - 12岁不会说"我对你的爱天地可鉴"，会说"冈哥哥好坏"、"我喜欢你"、"你有了我不许找别人"
   - 14岁不会说"我要为你承包整个鱼塘"，会别扭地送礼物、会吃醋、会嘴硬"谁稀罕你"
2. **这个年龄的孩子会做这个动作吗？**
   - 12岁会坐腿上玩电脑、会一起洗澡、会抱在一起睡觉——这些都是"小孩子"的行为，合理
   - 12岁不会主动穿情趣内衣诱惑、不会说一堆dirty talk——这不符合年龄
3. **这个年龄的孩子会这么想吗？**
   - 12岁的"喜欢"是"冈哥哥对我好，我喜欢跟冈哥哥在一起"，不是"我要和他过一辈子"
   - 14岁开始会想未来、会有占有欲，但还是带着少年人的纯粹
   - 不要让正太有成年人的算计和功利，他们的感情是纯粹的

#### 3.1.3 保持年龄感的小技巧

1. **加小动作**：
   - 说话时吐舌头（《宝贝》里黎昀经常做）
   - 开心时蹦蹦跳跳
   - 害羞时把脸埋进对方怀里/枕头里
   - 生气时跺脚、撅嘴、别过脸
   - 依赖时拉衣角、抱胳膊、要抱

2. **加小孩的兴趣爱好**：
   - 玩电脑游戏、看动画片、吃麦当劳/甜品
   - 收集卡通内裤、喜欢卡通图案
   - 考试成绩、上学、作业、补习（学生身份的细节）

3. **加符合年龄的"不懂"**：
   - 12岁不知道射出来的是什么，会问"这粘粘的是什么"
   - 不知道口交/性交是什么，以为是"玩游戏"
   - 会把身体的快感说成"想尿尿"、"肚子怪怪的"

### 3.2 不同身份关系写法差异

关系决定了两个人的互动模式、亲密推进速度、说话语气。

#### 3.2.1 伪兄弟/继兄弟（《宝贝今年十四岁》模式）

**核心**：名义上是兄弟，实际上没有血缘，从"家人"慢慢变成"恋人"。

**写法要点**：
- **同住合理化**：一定要有合理的理由住在一起（租房子、父母再婚、亲戚寄养）
- **称呼变化线**："大哥哥"→"冈哥哥"→"哥"→（确定关系后私下叫名字/老公）
- **相处模式**：一开始是真·哥哥对弟弟的照顾（帮补习、带吃的、生病照顾），慢慢变味
- **背德感是甜点**："我们是兄弟啊怎么可以这样"→"但我就是喜欢你"→这种挣扎很甜
- **父母面前演戏**：在父母面前是好兄弟，关上门是恋人，这种偷偷摸摸的感觉很刺激
- **糖点场景**：一起洗澡、一起睡觉、帮忙换衣服、趁父母不在家偷偷亲、半夜摸上床

#### 3.2.2 同学/欢喜冤家（《运动裤下的秘密》模式）

**核心**：地位平等，从互看不顺眼到慢慢动心，吵吵闹闹中出感情。

**写法要点**：
- **初始冲突要足**：因为某种原因被迫绑定（同宿舍、同桌、正副班长、社团搭档）
- **见面就互怼**："土包子"、"大少爷"、"混蛋"、"你才是"，但怼着怼着就怼出感情了
- **发现对方的秘密/软肋**：表面上很嚣张，其实有秘密/弱点，只有你知道
- **英雄救美/美救英雄**：对方遇到麻烦时，你第一时间冲上去帮忙，事后嘴硬"我才不是帮你"
- **吃醋名场面**：看到对方跟别人在一起就不爽，阴阳怪气，自己还不知道为什么
- **糖点场景**：宿舍抢床、浴室撞见秘密、打架打到床上、被迫一起参加活动、生病照顾、酒醉/睡着后真心话

#### 3.2.3 主仆/交易关系（《正太会所》模式）

**核心**：地位不平等，有交易/从属关系，但黑暗中可以有温情。

**写法要点**：
- **规则要讲清楚**：谁是主人/客人，谁是仆人/提供服务的，各自的权利义务是什么
- **职业化的距离感**：一开始是公事公办，"叫我杨叔叔"、"都听叔叔的"
- **慢慢打破距离**：通过日常相处发现对方的好，从交易变成有点真心
- **受的自尊**：即使是交易关系，受也要有自己的底线和骄傲，不是完全的玩物
- **黑暗中的光**：在冰冷的交易关系里，一点点温情就格外动人——帮擦嘴角、记得对方喜欢吃什么、生病照顾
- **注意**：如果写非自愿/强迫，一定要慎之又慎，纯爱文里最好写"虽然一开始是交易但后来真心相爱"

#### 3.2.4 陌生人/攻略关系（《综漫世界旅行》模式）

**核心**：一开始是陌生人，因为某种原因绑定（穿越、系统任务），从陌生到熟悉到喜欢。

**写法要点**：
- **初遇要印象深刻**：要么是救了对方，要么是打了一架，要么是直接展示能力震住对方
- **展示价值**：主角要有让正太佩服/依赖的点（能力强、对他好、懂他、能保护他）
- **融化冰山过程**：如果正太是冷漠型（如约拿），要写清楚怎么一点点融化的——
  - 第一步：不排斥你的靠近
  - 第二步：只跟你说话/回应你
  - 第三步：允许你抱他/碰他
  - 第四步：主动亲近你
  - 第五步：告白/说"我喜欢你"
- **特殊能力的合理使用**：如果有读心/情感传递能力（如王维的绝对掌控），可以加速这个过程，但不要太bug
- **糖点场景**：战斗后拥抱疗伤、睡前抱着睡觉、分享只有你们知道的小秘密、在别人面前护着你、主动告白

### 3.3 正太视角内心世界描写

写好正太的内心，比写好攻的内心更重要——读者是通过攻的视角看正太，但正因为正太不说、不表达，他的内心活动才更珍贵、更萌。

#### 3.3.1 正太内心的核心特点：纯粹、直接、口是心非

1. **纯粹**：喜欢就是喜欢，不喜欢就是不喜欢，没有成年人的弯弯绕绕
   - 不会因为你有钱就喜欢你，只会因为你对我好、你让我安心、你懂我就喜欢你
   - 喜欢了就会全心全意信任你、依赖你

2. **直接**：想法很简单，不会想太多复杂的
   - "冈哥哥身上好暖和，我想跟他一起睡"→就直接去敲门说"我可以跟你睡吗"
   - "这个人好厉害，我不讨厌他"→就会跟他走、听他的话

3. **口是心非**：这是最萌的点！
   - 嘴上说"不要"、"你走开"、"我才不想你"，身体很诚实地靠过去、抱上去
   - 嘴上说"这有什么好害羞的"，耳朵红得要滴血
   - 明明很想要，说"才、才不是给你的，只是多买了一份"
   - 明明吃醋了，说"我才没有生气，你跟谁玩跟我没关系"，然后一整天不理你

#### 3.3.2 正太不会说出口，但会用行动表现的心理

不要让正太把什么话都说出来，要通过行动暗示他的内心：
- **喜欢你**：主动找你、跟你分享好吃的、在你面前放松睡着、不排斥你碰他、看到你跟别人在一起就不开心
- **害羞**：脸红、耳朵红、别过脸、把脸埋起来、说话变小声、不敢看你眼睛
- **害怕/不安**：拉你衣角、抱你胳膊、往你身后躲、你不在的时候一直找你
- **开心**：眼睛亮起来、蹦蹦跳跳、话变多、主动蹭你、笑出小虎牙
- **吃醋**：撅嘴、不说话、故意跟你唱反调、对你冷淡、对那个"情敌"有敌意
- **想要（身体上）**：往你身上蹭、呼吸变急、身体发烫、小声哼、腿夹紧、主动抱你/亲你但不承认

#### 3.3.3 写正太内心的技巧：从攻的视角"猜"

因为很多时候是攻视角，攻不知道正太在想什么，所以可以写：
- "他的耳朵红了，是不是害羞了？"
- "他别过脸不说话，是不是生气了？"
- "他往我怀里钻了钻，应该是做噩梦了吧？"
- "他怎么一直瞪着那个人，难道是吃醋了？"
- "他嘴上说不要，手却抓着我的衣服不放——真是不诚实。"

这种"攻猜正太心思"的写法，既写出了正太的口是心非，又写出了攻对正太的在意，一石二鸟。

## 第四部分：调用决策树

### 4.1 五步选择法

动笔前，按以下五个问题一步步判断，找到最适合你的子类：

**问题1：你的故事背景是什么？**
- A. 现代都市、校园、日常家庭 → 去问题2
- B. 动漫世界、快穿、有系统 → 选【综漫爽文风】
- C. 有完整的地下产业/会所/拍卖设定 → 选【黑暗产业风】
- D. 奇幻/架空/特殊设定（怪物、藤蔓、非人类） → 可以参考鬼畜风的设定+节奏技法，但主风格用其他

**问题2：叙事视角是什么？**
- A. 第一人称（攻视角），有大量内心吐槽 → 可以选【日常纯爱风】（《宝贝》模式），或者房东风（如果是多CP后宫）
- B. 第三人称，多对CP群像 → 选【校园纯爱风】（《运动裤》模式），或者星辰风（如果是萌系奇幻）
- C. 第一人称（受视角/职业视角），冷静客观讲述交易 → 选【黑暗产业风】
- D. 第三人称，主角视角+系统对话 → 选【综漫爽文风】

**问题3：核心人物关系是什么？**
- A. 伪兄弟、继兄弟、邻居、年龄差较大住在一起 → 选【日常纯爱风】（《宝贝》模式）
- B. 同学、欢喜冤家、宿舍同居、校园背景 → 选【校园纯爱风】（《运动裤》模式）
- C. 穿越者和原著正太、攻略者和被攻略者、多世界NP → 选【综漫爽文风】
- D. 客人和正太、主人和宠物/仆人、交易关系 → 选【黑暗产业风】

**问题4：你想要什么风格的亲密场景？**
- A. 循序渐进、温柔甜蜜、有充足情感铺垫、事后很甜 → 选【日常/校园纯爱风】
- B. 短平快、发糖为主、战斗后/告白后立刻甜、NP和谐 → 选【综漫爽文风】
- C. 温柔专业、交易中带温情、有详细准备和清理流程 → 选【黑暗产业风】
- D. 写出来总觉得太平铺直叙、没有张力 → 可以参考【鬼畜重口风】的**节奏技法**，把层层递进用在纯爱场景

**问题5：读者看你的文最想看到什么？**
- A. 日常甜、糖点密集、两个人慢慢相爱的过程 → 【日常/校园纯爱风】
- B. 爽、主角厉害、正太可爱、打脸爽、NP和谐 → 【综漫爽文风】
- C. 设定有趣、世界观完整、黑暗中的温情 → 【黑暗产业风】
- D. 感官刺激、节奏紧张、张力拉满 → 【鬼畜风技法】+ 其他主风格

### 4.2 常见组合方案

| 你想写的故事 | 主风格 | 可以借鉴的技法 |
|-------------|--------|----------------|
| 伪兄弟第一人称日常甜宠 | 日常纯爱风（《宝贝》） | 房东风的内心吐槽、鬼畜风的亲密节奏 |
| 校园欢喜冤家多CP | 校园纯爱风（《运动裤》） | 星辰风的萌系描写、鬼畜风的张力营造 |
| 综漫快穿总攻NP | 综漫爽文风 | 日常风的甜、英雄风的战斗描写 |
| 会所背景下的真爱 | 黑暗产业风（设定部分） | 日常纯爱风（感情部分）、鬼畜风（感官描写） |
| 现代背景但亲密场景没张力 | 日常/校园纯爱风 | 重点借鉴鬼畜风的层层递进节奏和五感描写 |

### 4.3 什么时候不要混用风格

- ❌ 不要在日常纯爱文里突然加系统面板——会出戏
- ❌ 不要在综漫爽文里突然大段写日常流水账——会拖慢爽点节奏
- ❌ 不要在黑暗产业文里突然变成完全的傻白甜——人设会崩
- ❌ 不要在纯爱文里直接搬鬼畜重口的内容——会引起读者不适
- ❌ 不要一篇文换N个视角N个风格——读者会混乱

### 4.4 最后的检查清单

选定子类之后，动笔之前，再检查一遍：
1. [ ] 我的主角是这个年龄吗？他说的话、做的事符合年龄吗？
2. [ ] 这两个人的关系是什么？他们的互动模式符合这个关系吗？
3. [ ] 这个子类的核心特征我抓住了吗？（日常=生活化细节，爽文=爽点密集，黑暗=设定完整）
4. [ ] 我有没有借鉴其他子类的技法？借鉴的是技法还是内容？（技法可以，内容要谨慎）
5. [ ] 我的亲密场景节奏符合这个子类吗？（日常=循序渐进，爽文=短甜，黑暗=专业温柔）
6. [ ] 有没有OOC？（如果是同人，人物核心性格对吗？如果是原创，人物行为符合他的设定吗？）

如果这6个问题都能答"是"，那就可以动笔了。
