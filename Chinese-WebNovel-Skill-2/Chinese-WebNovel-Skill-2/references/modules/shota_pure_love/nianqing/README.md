# 房东模块

这是一个专门处理第一人称现代都市后宫甜宠文（房东视角）的独立模块。

本模块基于《年轻房东小房客》全五季提炼，把"房东风"从"几种感觉"拆成六层：

- 核心识别特征与"嘴硬心软"的内心戏体系
- 生活化语言与人物语言指纹
- 多人物后宫写法与争宠模式
- 10+种日常场景的切入与糖点设置
- 从日常到亲密的自然过渡路径
- "甜而不淫"的亲密场景写实写法
- 单元剧结构与多季连载连贯性
- 适配写作全流程的运行顺序

## 模块目标

这个模块主要解决：

- 第一人称写得猥琐，不像好人
- 内心戏要么没有要么太多太杂
- 对话不生活化，所有人说话一个味
- 新人物加入太突兀，像工具人
- 多人物戏份失衡，有人全程背景板
- 争宠写得像宫斗，不可爱
- 日常场景无聊，像流水账
- 从日常到亲密过渡生硬，像按头
- 亲密场景机械无感情，像写说明书
- 多人物亲密场景戏份分配乱
- 单元剧结构松散，没有伏笔
- 多季连载人物前后不一致

## 现在这套模块怎么读

推荐按这个顺序用：

1. 先读 [source_index.md](source_index.md)，知道五季原文各自的特点和重点章节。
2. 再读 [tutorial.md](tutorial.md)，建立完整的房东风写作方法框架。
3. 写作或改稿时，按 [runtime.md](runtime.md) 的顺序诊断问题、匹配解法。
4. 去 [good_examples.md](good_examples.md) 找对应技法的正例。
5. 去 [bad_examples.md](bad_examples.md) 找对应问题的反例。
6. 先搭架子，再写正文。

## 适用题材

本模块的"房东风"适用于以下题材：

1. **第一人称现代都市甜宠文**：房东/室友/邻居/学长/老师等"年上"视角
2. **多人物后宫/团宠文**：2个以上恋爱对象，和谐共处不狗血
3. **正太纯爱/年下甜文**：恋爱对象年纪较小，需要温柔感和保护感
4. **日常流单元剧**：以日常小故事为主，没有强剧情冲突
5. **第一人称暧昧文**：表面正经内心计划通的反差感写法

**不适用：**
- 强剧情冲突的虐文/狗血文
- 第三人称全知视角
- 古风/玄幻/科幻等非现代都市背景
- 纯肉文（房东风的亲密场景是为情感服务，不是为肉而肉）

## 房东风核心精神

房东风的精髓不是"推倒正太"，而是：

1. **人味第一**：主角可以有阴暗想法，但不能是坏人；内心可以吐槽，但行动上要负责任
2. **日常即糖**：最好的糖不是惊天动地的告白，而是一起做饭、吃饭、生病照顾的日常细节
3. **自然过渡**：从日常到亲密永远有触发点，永远不按头
4. **每个人都可爱**：后宫里的每个人都有自己的性格和位置，没有真正的"坏人"或工具人
5. **甜而不淫**：亲密场景重点是情感交流和心理互动，不是单纯的动作描写
6. **成长感**：人物会长大、会变化，关系会深化，不是一成不变的模板

## 说明

- 当前模块的"主参考"是 `sources/raw/` 里的五季原文。
- 亲密场景的写法总结写实但有情感，不回避原文内容，但强调"甜而不淫"的原则。
- 正例强调可迁移的写作结构，不强调句子皮相。
- 反例不是为了嘲笑拙劣写法，而是为了帮助识别内心戏猥琐、对话不生活化、过渡生硬等常见失误。
- **统一使用简体"房东"，绝对不要用繁体"房東"。**
