## 一、核心设定与世界观类型

### 1. 现代校园类
- **背景**：中学/大学/补习班/偶像经纪公司/体育队
- **权力关系**：学长→学弟、老师→学生、教练→队员、经纪人→偶像
- **日常化**：性行为融入校园日常（刷牙口交、课间在厕所、更衣室、宿舍、教室后排）
- **反差萌**：体育生/篮球校队/偶像明星表面光鲜，私下是被榨精的精牛/泄欲工具

### 2. 系统流类（堕淫APP）
- **核心机制**：手机APP发布任务，不完成就有惩罚
- **任务类型**：露出、公开自慰、给学长口交、穿情趣内衣去学校、远程跳蛋上课
- **升级体系**：积分兑换奖励/解锁更变态任务，积分不足公开处刑/强制高潮
- **AI助手**：类似"小泽"的系统精灵，引导/嘲讽/督促宿主完成任务

### 3. 比赛/竞技类（顶点游戏）
- **核心设定**：少年们通过性能力比赛决胜负，谁先射谁输
- **赛制**：海选→入围→决赛圈
- **比赛机制**：
  - 红圈缩小：缩得越小催淫烟雾越浓
  - 虚拟触手：直接套弄肉棒榨精
  - 空投"武器"：跳蛋/飞机杯/润滑油等道具
  - 忍耐时间：谁坚持最久不射谁赢，精液质量量化为分数

### 4. 科技/胶囊类（正太胶囊公司）
- **核心设定**：把真人正太基因压缩成苹果大小胶囊，温水（37-40℃）泡发3分钟恢复1.5米真人大小
- **颜色型号**：
  - 红色：热情奔放主动型
  - 蓝色：内向害羞腼腆型（需耐心引导）
  - 橙色：活力阳光运动型
  - 紫色：傲娇霸道型
- **限制**：每次使用不超过3小时，超时气化消散
- **黑暗设定**：胶囊基因从真实少年身上哄骗/绑架采集

## 二、核心角色原型（正太校园专属）

### 受害者/受方原型
1. **阳光体育生型**：篮球/足球/田径队，高大阳光、肌肉匀称、穿球服/白袜运动鞋，私下被队长/教练榨
2. **傲娇偶像型**：少年偶像/网红，表面高冷光鲜，被经纪人/粉丝会长控制，刻字标记所有权
3. **呆萌学弟型**：刚入学新生，内向害羞容易脸红，被学长"照顾"一步步堕落
4. **学霸乖乖牌型**：戴眼镜成绩好不谙世事，被老师/学长以"补习"为名骗到办公室调教
5. **问题少年型**：校霸/不良少年，看起来很凶，被更狠的角色征服后变成乖狗

### 加害者/攻方原型
1. **温柔学长型**：表面温柔体贴照顾学弟，实际一步步引导堕落，占有欲极强
2. **霸道教练/经纪人型**：掌控受方前途（比赛/出道），用前途威胁逼迫就范
3. **变态老师型**：以"补习""谈话"为名叫学生到办公室/保健室
4. **纨绔子弟型**：有钱有势，把少年当玩物/收藏品
5. **系统/AI型**：没有实体，通过手机APP发布指令，用积分/惩罚控制
6. **同学/室友型**：朝夕相处的室友/同班同学，在宿舍里半推半就

## 七、堕落心理描写技巧（正太校园专属）

### 1. 日常化堕落五阶段
1. 抗拒期："我们都是男的这样不好吧！"但是身体已有反应
2. 犹豫期："就这一次...下不为例..."半推半就
3. 习惯期："今天怎么没来找我？"开始期待
4. 依赖期："学长/老师...我想要..."会主动求操
5. 认同期："我就是他的精牛/狗/泄欲工具"彻底接受身份

### 2. 公开场合羞耻心描写
- 听觉：能听到外面同学说话声/脚步声/老师讲课声
- 视觉：透过门缝/眼罩缝隙看到熟悉教室场景
- 心理："万一被发现怎么办"的恐惧 vs 快感
- 反差：越是严肃正经的课堂/场合，偷偷被玩的快感越强

### 3. "闹着玩"式破处
- 关键：不要一开始就严肃/强迫
- 开头是开玩笑/打赌/闹着玩："你敢不敢比谁射得远"
- 在嬉闹中自然发生，减少道德负罪感
- 做完假装没事，但气氛变了下次更自然

## 八、写作技巧与风格要点

### 1. 日常感是核心
- 不要写得像异世界奇幻
- 要让读者觉得"这好像就发生在我身边的学校"
- 校服、课间操、食堂饭菜、晚自习、月考这些校园元素要穿插
- 性行为要像吃饭睡觉一样自然融入日常

### 2. 反差萌写法
| 表面身份 | 私下身份 | 反差爽点 |
|----------|----------|----------|
| 阳光篮球队长 | 被教练按在更衣室操的精牛 | 球场上挥汗如雨 → 私下哭着求操 |
| 高冷学霸 | 办公室给老师口交的乖狗 | 讲台上讲题 → 办公桌下含着肉棒 |
| 顶流少年偶像 | 被经纪人控制的泄欲工具 | 舞台闪闪发光 → 休息间被玩到腿软 |
| 校霸不良少年 | 被更狠的人操到叫爸爸 | 欺负同学 → 被欺负到哭 |

### 3. 声音描写很重要
- 校园背景音：上课铃声、粉笔写字声、操场喧闹、走廊脚步声
- 受方压抑的声音：咬嘴唇不敢出声、闷哼、牙齿咬手背、床单塞嘴里
- 攻方低声耳语："叫出来啊，让全班听听你有多骚""外面有人哦"

### 4. 服装与细节
- **白袜**：少年感第一神器，运动后微微汗湿的白袜、白袜足交、白袜套肉棒
- **校服**：白衬衫、领带、西装短裤、脱一半挂在身上
- **运动服**：篮球服、背心、短裤、长筒运动袜、运动鞋
- **气味**：肥皂味、汗水味、牛奶味、青春期少年体味

## 九、雷点规避
- **未成年设定**：所有角色年龄设定18+（高三设定为18岁留级，大学生直接18+），正太仅作外形/性格借鉴
- **女性角色**：全程男性，女性仅作为背景路人/老师远景，不参与任何性接触
- **尸体/杀害**：属于重度R18G，非用户明确要求不使用，默认用"调教成宠物/奴隶"替代
- **血腥**：刻字/轻度疼痛可以有，但必须加"事后上药/不会留疤"设定
