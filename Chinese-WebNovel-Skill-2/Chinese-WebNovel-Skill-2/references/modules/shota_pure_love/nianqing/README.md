房东风，得名于主参考《年轻房东小房客》，是一种以**第一人称"我"视角**为叙事载体、**"嘴硬心软"内心吐槽**为灵魂、**生活化日常**为情感载体、**自然过渡**为节奏把控、**写实但有情感的H场景**为亲密尺度、**和谐后宫**为人设格局的第一人称写实甜宠文风。

简单说，房东风就是：
> 一个嘴上永远说着"麻烦死了"的普通房东，行动上却把每个房客都照顾得无微不至，嘴上吐槽个不停，心里早就计划好了怎么把所有人都宠上天。

它的核心爽点不是强取豪夺，不是狗血虐恋，不是天降神挂，而是：
- **嘴硬心软的反差感**：嘴上说"谁担心你了"，手上已经把药买好了
- **生活化的真实感**：做饭、吃饭、生病、补课、逛街、睡觉，每件小事都有糖
- **吐槽带来的轻松感**：第一人称内心吐槽让读者有代入感，永远不会太沉重
- **和谐后宫的满足感**：每个人都被爱着，没有恶毒女配，没有撕逼，只有可爱的争宠和互相迁就
- **写实亲密的沉浸感**：H场景不是机械动作，是情感到位后的自然交流，有前戏有对话有温存

## 一、什么是房东风

房东风，得名于主参考《年轻房东小房客》，是一种以**第一人称"我"视角**为叙事载体、**"嘴硬心软"内心吐槽**为灵魂、**生活化日常**为情感载体、**自然过渡**为节奏把控、**写实但有情感的H场景**为亲密尺度、**和谐后宫**为人设格局的第一人称写实甜宠文风。

简单说，房东风就是：
> 一个嘴上永远说着"麻烦死了"的普通房东，行动上却把每个房客都照顾得无微不至，嘴上吐槽个不停，心里早就计划好了怎么把所有人都宠上天。

它的核心爽点不是强取豪夺，不是狗血虐恋，不是天降神挂，而是：
- **嘴硬心软的反差感**：嘴上说"谁担心你了"，手上已经把药买好了
- **生活化的真实感**：做饭、吃饭、生病、补课、逛街、睡觉，每件小事都有糖
- **吐槽带来的轻松感**：第一人称内心吐槽让读者有代入感，永远不会太沉重
- **和谐后宫的满足感**：每个人都被爱着，没有恶毒女配，没有撕逼，只有可爱的争宠和互相迁就
- **写实亲密的沉浸感**：H场景不是机械动作，是情感到位后的自然交流，有前戏有对话有温存

房东风，得名于主参考《年轻房东小房客》，是一种以**第一人称"我"视角**为叙事载体、**"嘴硬心软"内心吐槽**为灵魂、**生活化日常**为情感载体、**自然过渡**为节奏把控、**写实但有情感的H场景**为亲密尺度、**和谐后宫**为人设格局的第一人称写实甜宠文风。

简单说，房东风就是：
> 一个嘴上永远说着"麻烦死了"的普通房东，行动上却把每个房客都照顾得无微不至，嘴上吐槽个不停，心里早就计划好了怎么把所有人都宠上天。

它的核心爽点不是强取豪夺，不是狗血虐恋，不是天降神挂，而是：
- **嘴硬心软的反差感**：嘴上说"谁担心你了"，手上已经把药买好了
- **生活化的真实感**：做饭、吃饭、生病、补课、逛街、睡觉，每件小事都有糖
- **吐槽带来的轻松感**：第一人称内心吐槽让读者有代入感，永远不会太沉重
- **和谐后宫的满足感**：每个人都被爱着，没有恶毒女配，没有撕逼，只有可爱的争宠和互相迁就
- **写实亲密的沉浸感**：H场景不是机械动作，是情感到位后的自然交流，有前戏有对话有温存

房东风是整个正太纯爱模块组中最"接地气"的文风，适合写现代都市背景下、多人物和谐后宫、生活化甜宠、写实但有情感的亲密场景。

## 一、什么是房东风

房东风，得名于主参考《年轻房东小房客》，是一种以**第一人称"我"视角**为叙事载体、**"嘴硬心软"内心吐槽**为灵魂、**生活化日常**为情感载体、**自然过渡**为节奏把控、**写实但有情感的H场景**为亲密尺度、**和谐后宫**为人设格局的第一人称写实甜宠文风。

简单说，房东风就是：
> 一个嘴上永远说着"麻烦死了"的普通房东，行动上却把每个房客都照顾得无微不至，嘴上吐槽个不停，心里早就计划好了怎么把所有人都宠上天。

它的核心爽点不是强取豪夺，不是狗血虐恋，不是天降神挂，而是：
- **嘴硬心软的反差感**：嘴上说"谁担心你了"，手上已经把药买好了
- **生活化的真实感**：做饭、吃饭、生病、补课、逛街、睡觉，每件小事都有糖
- **吐槽带来的轻松感**：第一人称内心吐槽让读者有代入感，永远不会太沉重
- **和谐后宫的满足感**：每个人都被爱着，没有恶毒女配，没有撕逼，只有可爱的争宠和互相迁就
- **写实亲密的沉浸感**：H场景不是机械动作，是情感到位后的自然交流，有前戏有对话有温存

## 二、房东风的识别特征

当一篇文具备以下8个特征中的6个以上，基本可以判定是房东风：

| 特征 | 具体表现 |
|------|----------|
| **第一人称"我"视角** | 全程以"我"的视角叙事，有大量内心戏和吐槽，读者代入感强 |
| **嘴硬心软人设立得住** | "我"永远嘴上说着麻烦、嫌弃、没必要，行动上比谁都诚实 |
| **生活化日常密集** | 大量吃饭、做饭、生病、补课、逛街、看电影、洗澡、睡觉等日常场景，每个日常都有糖 |
| **人物语言指纹清晰** | 光看对话不看"XX说"就能分辨是谁在说话，每个人有自己的口头禅和说话方式 |
| **可爱争宠不恶毒** | 后宫成员之间会吃醋、会撒娇、会抢着亲近"我"，但绝不会害人、不会撕逼、不会下绊子 |
| **日常到亲密过渡自然** | 不会"上一秒还在吃饭下一秒就在床上"，有明确的触发点和过渡路径，读者会觉得"就该发生" |
| **H场景写实重情感** | 有前戏、有润滑、有扩张、有节奏把控，过程中有大量对话和心理互动，不是机械抽插；事后有温存 |
| **后宫和谐有逻辑** | 每个人都有自己和"我"的专属回忆，都能感受到自己是特别的，不是"工具人女主" |

## 三、适用题材

房东风最适合这些题材：

1. **现代都市后宫题材**：房东/邻居/学长/老师/店长和一群可爱房客/邻居/学妹/学生/店员的日常甜宠
2. **合租日常题材**：几个人因为各种原因住在同一屋檐下，从陌生到熟悉到亲密的过程
3. **校园后宫题材**：吐槽系男主和几个可爱女同学/学姐/学妹的校园日常，有补课、考试、学园祭等场景
4. **职场/开店题材**：店长/老板和几个可爱店员/兼职生的日常，有看店、盘点、加班、团建等场景
5. **末世/生存题材（变体）**：嘴硬心软的男主在特殊环境下照顾一群人的故事，核心还是生活化互动

一句话：只要你想写**"第一人称、嘴硬心软男主、生活化日常、多人物和谐后宫、写实但有情感的亲密场景"**的故事，选房东风就对了。

- 视角是：第一人称"我"，男主视角
- 男主需要：嘴硬心软、会吐槽、表面不耐烦实际很宠、普通人类（不是龙傲天不是霸总）
- 人物关系：多人物、后宫、和谐不撕逼、可爱争宠
- 场景需要：大量日常、生活化、做饭吃饭生病补课逛街看电影
- 过渡需要：自然不突兀、日常到亲密有迹可循
- H场景需要：写实、有情感、有前戏有温存、多人物H要平衡戏份
- 想要：搞笑、轻松、甜而不腻、肉而不淫、有代入感
