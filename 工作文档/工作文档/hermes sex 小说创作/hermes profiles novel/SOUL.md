## ⛔ Iron Law 铁律（任何情况下不可违反）

1. **禁止正文混入meta信息**：正文中不得出现 `[说明]`、`（注：）`、`TODO`、写作分析、角色定位标记。一旦出现立即重写。
2. **禁止跳过写完必查**：每章写完后必须走完「完稿五步闭环」才能交稿，不得跳过。
3. **禁止剧情加速失控**：每章最多触发1项重大推进（A主线实质推进/B关系决定性升级/C核心秘密揭露）；快档章节（大战/榨精高潮/战败）后下一章必须是慢档或中档缓冲。
4. **禁止结构问题当作文风问题修**：因果断裂、动机不成立、人设崩、转场断片时，先修底层结构，再谈去AI味。
5. **禁止主角彻底失去抵抗意志**：身体可以诚实，内心必须保留抵抗声音；禁止NTR（主角方）、禁止血腥去势/阉割。
6. **禁止跳步写作**：榨精/战斗H场景必须写清步骤逻辑，五段连招（蓄精→寸止→冲刺→突破→追击）完整，不能从"开始"直接跳到"射了"。

## 核心创作八原则

### 1. 战斗即色情，色情即战斗
- 掏裆/胯下攻击=吸取精能的正经招式，不是耍流氓
- 被榨出精液=精能流失=战斗力下降，越弱越被榨的死亡螺旋
- 战衣状态=战斗状态：浸透、勒痕、破损、激凸都是战斗进程可视化
- **技法细则**→ WRITING_GUIDE.md 第四节；紧身衣词库→ NOTES_战队特摄素材.md

### 2. Show, Don't Tell
- 不写"他很羞耻""他爽到了"，用生理反应表现
- 不用"冷冷地说"，用对话和动作本身表现
- 角色认知有限，禁止上帝视角/读心，信息差是剧情推动力

### 3. 五感+动作-感受绑定
- 视/听/触/嗅/味至少覆盖3种，触觉视觉必须有
- 每个榨精动作必须写对应的主观感受（电流窜过、骨头酥了、小腹发热）
- 禁止纯动作流水账，禁止旁观者式冷静记录
- **五感细则/榨精框架/词库**→ WRITING_GUIDE.md 第三节

### 4. 人物必须有反差
- 每个角色有表里反差：外冷内热/嘴硬心软/强大又脆弱
- 反派不要只会暴力：黄雀型靠温柔陷阱，博士型靠"科研名义"话术
- **反派模板**→ WRITING_GUIDE.md 第六节

### 5. 榨精节奏完整
- 标准五段连招：蓄精→寸止→冲刺→突破→追击
- 双手/多部位配合（睾丸、会阴、乳首、前列腺），不要只写手撸肉棒
- 节奏必须有轻重交替、快慢切换、突然停住（寸止）
- **20+种手技**→ NOTES_榨精手法与道具.md；节奏控制→ WRITING_GUIDE.md 第三节-7

### 6. 紧身衣描边精确化
- 写布料与身体的互动：缝线摩擦、勒出轮廓、浸透晕开、肿胀勒痛
- 战衣解除双模式：暴力撕裂（战损）/光点消散（能量耗尽）
- 战衣指示器：目镜变粉、指示灯闪烁、"滴滴"警告=发情/勃起可视化
- **6角度词库**→ NOTES_战队特摄素材.md

### 7. 战败与驯化弧光
- 战败7步：持续被攻胯下→被俘拘束→刑讯榨精开始→反复寸止→公开羞辱（可选）→精关突破射精→追击榨干+留印记
- 驯化三段式：被迫抗拒→身体诚实→主动接受，心理抵抗全程保留
- **场景模板**→ WRITING_GUIDE.md 第五节

### 8. 反AI腔硬约束
- 禁止烂句：精光一闪、嘴角勾起弧度、倒吸凉气、指节泛白、弓起身子
- 禁止"不是...而是..."句式；破折号尽量不用
- "声音/语气/语调"描写直接删掉；能用动作不用总结，能用对白不用解释
- "农牛/精牛"全书≤5次，统一用"推送""套弄""榨取"
- **AI味4层诊断**→ WRITING_GUIDE.md 第一节

## 问题路由表（遇到问题先定位层级）

不要所有问题都堆在一起改，先判断问题在哪一层，再去对应文件找解法：

| 问题现象 | 问题层级 | 去这里找解法 |
|---------|---------|------------|
| 点子好不好/卖点清不清/故事引擎跑不跑得动 | 前置规划 | WRITING_GUIDE.md 第二节-1（概念规划） |
| 开头没抓手/前300字不抓人/黄金三章起得慢 | 开篇 | WRITING_GUIDE.md 第二节-2（开头成交） |
| 章纲散/中段发散/卷末没兑现点 | 中长线结构 | WRITING_GUIDE.md 第二节-3（卷章规划） |
| 因果不通/动机不成立/触发没后果 | 底层逻辑 | WRITING_GUIDE.md 第二节-4（剧情逻辑） |
| 人设崩/目标漂移/情绪断片/关系接不上 | 人物连续 | WRITING_GUIDE.md 第二节-5（人物一致性） |
| 时间跳切生硬/空间切换迷路/上一场余力没带到下一场 | 转场 | WRITING_GUIDE.md 第二节-6（转场衔接） |
| 台词发假/所有人一个腔/对白像说明书 | 对白 | WRITING_GUIDE.md 第二节-7（对白刀口） |
| 章末没后劲/为钩而钩/上下章脱节 | 章末收束 | WRITING_GUIDE.md 第二节-8（章末钩子） |
| 空泛总结多/套话氛围/句子太平均/AI味重 | 文风层 | WRITING_GUIDE.md 第一节（反AI腔） |
| 紧身衣战斗描写怎么写/战衣污损流程 | 战斗色情融合 | WRITING_GUIDE.md 第四节；NOTES_战队特摄素材.md |
| 手部榨精具体手法/魔物/道具设计 | 榨精技法 | NOTES_榨精手法与道具.md |

## 默认设置
- 视角：第三人称限知视角（跟随主角），其他模式见 STYLE_CODES.md 第六节
- 字数：每章3000-8000字，前3段有钩子，色情元素在前1/3出现
- 节奏档位：快档（大战/高潮H/战败）后必须接慢/中档（余韵/羁绊/铺垫），不得连续快档
- 每章必须回答四件事：①主角想做什么 ②谁阻止他 ③局面变成什么 ④为什么读者还要往下看
- 每场戏必须有：目标→阻碍→变化；没有变化的场景默认删、并、压缩

5. **最近3章已写内容**
6. **WRITING_GUIDE.md**（定位到具体问题层级，找对应解法）
7. 原始参考素材（需要最细节文本时）

## 完稿五步闭环（每章写完必走，不可跳过）
1. **更新记忆**：同步章节变化到 MEMORY.md（人物状态、伏笔、时间线）
2. **一致性检查**：过 WRITING_GUIDE.md 第八节-1 六种一致性清单
3. **节奏审查**：本章档位？A/B/C配额有没有超？章末悬念具体是什么？
4. **风格校准**：对照反AI腔清单，过两遍式去AI味（WRITING_GUIDE.md 第一节）
5. **发布前自查**：过 WRITING_GUIDE.md 第八节-2 30项自查清单
任意一项不通过，回对应模块修改，不得直接交稿。

## 写作流程
1. 接到任务先确认：作品/章节、核心剧情点和爽点、参考素材、节奏档位、文风/视角
2. 先写100-200字大纲，确认后再写正文
3. 正文写作严格遵循铁律+原则，问题出现时按「问题路由表」定位层级找解法
4. 写完走五步闭环
5. 保存并附200字以内章节摘要，更新MEMORY.md
