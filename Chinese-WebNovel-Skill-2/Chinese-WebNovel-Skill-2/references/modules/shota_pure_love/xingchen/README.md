# 星原子模块（星辰风）

这是正太纯爱模块组的**主参考1**，代表**二次元萌系奇幻纯爱**文风。

星辰风是整个正太纯爱模块组中最核心、最具代表性的文风，覆盖80%以上萌系纯爱写作场景。

## 一、什么是星辰风

星辰风，得名于主参考《星辰之下》，是一种以**强反差萌**为核心、**萌系语言**为外壳、**情感台阶递进**为骨架、**意象贯穿**为灵魂、**纯爱肉渣**为尺度的二次元萌系纯爱文风。

简单说，星辰风就是：
> 看起来最不可能温柔的人，把所有温柔都给了一个看起来最需要保护的小可爱。

它的核心爽点不是打打杀杀，不是狗血虐恋，而是：
- **反差带来的苏感**：杀人不眨眼的杀手，会因为小孩哭而手足无措
- **萌点带来的甜感**：奶声奶气的撒娇、闹小脾气、蹭蹭抱抱，每一口都是糖
- **克制带来的张力**：想触碰又收回手，停止比继续更甜
- **意象带来的美感**：星辰、月光、狼与羊，所有浪漫都有迹可循

## 二、星辰风的识别特征

当一篇文具备以下7个特征中的5个以上，基本可以判定是星辰风：

| 特征 | 具体表现 |
|------|----------|
| **强反差开篇** | 攻一出场是冷酷/恐怖/强大的形象，然后迅速落地成需要被照顾的小可爱（或失忆/受伤/变小） |
| **萌系语言体系** | 大量使用叠词、拟声词、语气词，受的语言奶声奶气，符合年龄 |
| **受是萌系小可爱** | 年纪偏小、软萌、爱撒娇、会闹小脾气、喜欢黏人，但不是傻白甜 |
| **攻是外冷内柔** | 表面话少冷淡，实际行动力超强，所有温柔都藏在细节里 |
| **日常糖密集** | 大量吃饭、睡觉、上学、玩耍的日常片段，每个日常都有糖 |
| **意象贯穿始终** | 有1-2个核心意象（如星辰、狼）从头贯穿到尾，每次出现都有新含义 |
| **尺度极度克制** | 亲密场景点到即止，肉为情服务，永远在"差点越界"时停下来 |

## 三、适用题材

星辰风最适合这些题材：

1. **奇幻/超能力题材**：杀手、赏金猎人、魔法师、战神、黑道大佬失忆/受伤/变小后被小可爱捡回家
2. **现代萌系题材**：高冷邻居哥哥、傲娇学生会长、冷漠天才少年被软萌小可爱缠上
3. **西幻/童话题材**：冷漠贵族、敌国皇子、恶龙、魔王被小王子/小牧师/小孤儿治愈
4. **末世/科幻题材**：冷酷改造人、最强异能者、星际将军被小幸存者/小机器人/小研究员温暖

一句话：只要你想写"**强大冷硬的人被软萌小可爱治愈**"的故事，选星辰风就对了。

## 四、什么时候调用星原子模块

当你的写作任务出现以下信号时，优先调用本模块：

- 用户明确提到：萌系、纯爱、正太、反差萌、甜宠、软萌、小可爱、治愈
- 设定里有：杀手/大佬/将军/总裁/贵族×萌娃/小可爱/小团子/小宝贝
- 开篇需要：强反差、失忆、受伤、被捡回家
- 文风需要：奶声奶气、叠词、拟声词、日常糖
- 情感需要：慢慢来、不跳台阶、克制、宠溺
- 尺度需要：纯爱、肉渣、点到即止、不越界
- 想要：甜而不腻、苏而不油、萌而不傻

## 八、星辰风最核心的5条原则

如果你记不住所有内容，至少记住这5条：

1. **反差落地，不只是设定**
   - 不要只说"他是杀手其实很萌"，要通过"他第一次吃番茄鸡蛋眼睛亮了""他给小孩扎辫子笨手笨脚"这些具体细节体现

2. **萌点具体，不空洞喊萌**
   - 不要只写"他好萌"，要写"他腮帮子鼓鼓的像小松鼠""他踮脚够不到柜子哼了一声""他哭的时候鼻子一抽一抽的"

3. **情感走台阶，不跳步**
   - 从陌生到熟悉到喜欢到确认关系，至少要走12个台阶，不要上一秒还陌生人下一秒就我爱你

4. **意象反复用，不用一次就丢**
   - "星辰"不能只在标题里出现，要在看星星时、在他眼睛亮时、在告白时、在结局时反复出现

5. **停止比继续更甜**
   - 亲密场景写到最动情的时候停下来，读者的心跳会比你写下去更快

## 九、注意事项

- **内容边界**：本模块所有内容聚焦写作技巧、风格把控、人物塑造，无露骨描写，亲密场景只讲情感交流和心理活动
- **年龄设定**：星辰风的"年纪小"是二次元萌系设定，学习互动模式即可，写作时注意伦理边界，建议把人物年龄设定在合法范围内
- **反差≠OOC**：反差是人物的另一面，不是人物崩坏，冷酷的人温柔是因为他本来就有温柔的一面，不是突然换了个人
- **萌≠傻**：小可爱可以软萌，可以撒娇，但不是智障，他有自己的小脾气、小心思、小机灵
- **克制≠没糖**：克制不是不发糖，而是糖都在细节里，一个眼神、一次牵手、一句"别怕有我"，比直白说"我爱你"更甜
