# 英雄风模块

这是一个专门处理**黑暗英雄调教陷落**题材的独立模块。

英雄风的核心定义：**英雄/正义角色被俘获后，经历身体与精神的双重调教，信念逐步动摇最终陷落**。它不是单纯的虐文，也不是单纯的色情文，核心张力来自三层冲突：正义vs堕落、坚强vs脆弱、公众身份vs被支配身份。

## 模块目标

这个模块主要解决：

- 英雄角色写得太脸谱化，没有脆弱面
- 陷落过程太快太突兀，缺乏说服力
- 调教场景只有色情没有张力
- 羞耻感营造层次单一，只会写脱衣服
- H场景粗暴无层次，只有活塞运动
- 人物陷落之后弧光断裂，变成纯玩具
- 为黑暗而黑暗，虐而无爱
- 反派塑造扁平化，只有坏没有魅力
- 团队陷落时人物没有差异化
- 结局处理草率，缺乏人物转变或救赎感

## 模块适用题材

英雄风模块适用于以下题材场景：

- 超级英雄/义警被俘调教
- 魔法少年/战队成员陷落
- 正派侠客/修士被邪道俘获
- 警察/军人/特工落入敌手
- 圣骑士/神父被黑暗诱惑
- 任何"原本立场坚定的正义方角色逐步堕落"的剧情

不适用于：
- 纯甜文/纯恋爱文（没有陷落过程）
- 纯虐文（只有痛苦没有快感和心理转变）
- 反派为主角的文（没有"从正义到堕落"的反差）

## 三大子类

本模块将英雄风分为三大子类，各有不同写法侧重：

1. **moecloud少年英雄系（递进陷落模式）**
   - 单人英雄完整五阶段陷落
   - 核心是身份反差：公众英雄vs私密玩物
   - 宠物化（猫/狗）、身体改造、直播/拍卖展示是常见桥段
   - 参考范本：《少年英雄》四部曲

2. **正义战队系（团队/群体调教）**
   - 多名英雄逐个陷落，互相影响
   - 核心是信任崩塌：队友背叛、为救队友主动屈服
   - 内部腐化、连锁打击、差异化陷落是关键
   - 参考范本：《正义战队》

3. **神奇少年系（超能力觉醒与掌控）**
   - 兄弟/搭档双英雄陷落
   - 核心是羁绊被利用：用陷落的一方控制另一方
   - 机械调教、魔法控制、轮奸示众是常见桥段
   - 参考范本：《神奇少年》

## 现在这套模块怎么读

推荐按这个顺序用：

1. 先读 [source_index.md](source_index.md)，知道6篇原文的分类定位和重点章节
2. 再读 [tutorial.md](tutorial.md)，建立英雄风的核心方法论框架
3. 写作或改稿时，按 [runtime.md](runtime.md) 的顺序判断问题类型、制定动笔计划
4. 去 [good_examples.md](good_examples.md) 找 3-5 个近似正例，学习技法结构
5. 去 [bad_examples.md](bad_examples.md) 找 2-3 个近似反例，避免踩坑
6. 先写陷落节奏计划，再写正文

## 核心原则

英雄风写作的三条铁律：

1. **黑暗为张力服务，不为黑暗而黑暗**
   痛苦和羞辱是手段，不是目的。目的是展现"坚强的人如何在重压下逐步改变"。

2. **即使陷落，也要保持人物弧光**
   英雄变成奴，不是变成没有灵魂的玩具。他的坚持、他的挣扎、他的骄傲，即使在最屈辱的时刻也要有痕迹。

3. **身体快感和精神动摇必须双线并行**
   不能只有身体沦陷而精神毫无波动，也不能只有精神说教而没有身体反应。两者必须交织，互相促进。

## 说明

- 当前模块的"主参考"是 `sources/raw/` 里的6篇原文，所有技法都从原文提取，有文本依据。
- 正例强调可迁移的写法结构，不强调具体句子皮相。
- 反例不是为了批判某篇作品，而是为了帮助识别英雄风写作中的常见失误。
- 本模块涉及成人内容描写，所有技法分析仅用于写作学习，请遵守相关法律法规。
