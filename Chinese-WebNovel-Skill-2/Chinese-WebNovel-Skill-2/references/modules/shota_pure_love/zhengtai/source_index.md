# 正太综合题材资料索引

这份索引对 `sources/raw/` 下的5篇小说进行分类定位，明确每个子类的特点和参考价值。

## 一、资料总览

当前 `raw/` 下共有5篇小说，覆盖正太题材的5个典型子类，每个子类对应不同的写作场景和技法需求。

这些资料不是单一风格，而是一个**综合工具箱**——当星辰风（萌系奇幻）、房东风（第一人称后宫）、英雄风（黑暗调教）三种主风格都不匹配时，按题材选择对应子类参考。

## 二、核心资料分组

### A 组：日常/校园纯爱风（2篇）

这组是正太纯爱最基础、最常用的子类，适合绝大多数现代背景、平等关系的正太写作场景。

#### A1 《宝贝今年十四岁》 我的宝贝今年14岁 作者：babysee(非小白，伪兄弟,两部全).txt

**核心定位**：伪兄弟日常纯爱风的标杆作品，第一人称写实向。

**主要内容**：
- 大学生罗冈租住到12岁小学生黎昀家，从救命恩人到伪兄弟再到恋人的完整过程
- 时间跨度两年，从黎昀12岁写到14岁
- 生活化细节密集：麦当劳、补习、游泳、洗澡、生日、同学冲突
- 第一人称（攻视角），内心吐槽丰富，嘴硬心软
- 情感推进自然：救命→好感→试探→犯错→原谅→确认关系→日常甜蜜

**文风特点**：
- 第一人称"我"（罗冈视角），带有明显的内心OS和吐槽标记（—__—|||）
- 语言口语化，贴近生活，广州话点缀增加真实感
- 细节描写具体：DV偷拍、化学麻醉剂事件、白金情侣戒指、日记被偷看
- 情感节奏：有波折（麻醉剂事件、同学威胁事件）但整体甜蜜
- 亲密场景循序渐进：坐腿→洗澡帮忙→互相抚慰→约定生日→完整关系

**参考价值**：
- ★★★★★ 伪兄弟关系写法教科书
- ★★★★★ 第一人称攻视角内心活动写法
- ★★★★☆ 日常糖点设计（麦当劳、戒指、生病照顾）
- ★★★★☆ 年龄差（大学生×小学生）相处模式
- ★★★☆☆ 亲密场景节奏把控（循序渐进，有情感铺垫）

#### A2 运动裤下的秘密.txt 作者：迷羊

**核心定位**：校园纯爱风，欢喜冤家模式，带奇幻元素（透视/恋爱星设定）。

**主要内容**：
- 两校合并背景：三流平民高中"上乐"被贵族高中"佑尚"并购
- 主角配对：神秘超能力少年古云风×傲娇运动部长宋捷
- 副CP：校长林奇玮×陆清宇、老大孟念奇×费子昂、雷劲×尹天琪
- 校园日常：宿舍同居、上课打架、餐厅对立、社团招新
- 奇幻设定：古云风能看见人头上的"恋爱星"判断恋爱状态
- 核心秘密：号称"千人斩"的宋捷其实是个有隐疾的可怜处男

**文风特点**：
- 第三人称多视角，群像戏写得好，每对CP各有特色
- 欢喜冤家模式：从互看不顺眼到逐渐动心，冲突密集
- 喜剧感强：对话逗趣，人物反应夸张可爱
- 校园元素丰富：宿舍、食堂、课堂、社团、晚会
- 人物塑造鲜明：傲娇运动部长、神秘超能力少年、洁癖优等生、肌肉大块头
- 节奏明快：每章都有冲突和进展，不拖沓

**参考价值**：
- ★★★★★ 校园欢喜冤家写法
- ★★★★★ 多CP群像叙事技巧
- ★★★★☆ 傲娇受人物塑造
- ★★★★☆ 校园日常与感情线交织
- ★★★☆☆ 奇幻设定（超能力）融入现实背景的方法

### B 组：综漫爽文风（1篇）

#### B1 [综漫]正太控的世界旅行(1).txt

**核心定位**：综漫系统爽文风，穿越多世界攻略正太的NP总攻文。

**主要内容**：
- 主角王维获得"绝对掌控"超能力，穿越各个动漫世界攻略正太
- 第一个世界：《军火女王》，攻略约拿（约拿·迈尔，少年兵）
- 系统流设定：有系统对话、攻略任务、能力升级
- 金手指："绝对掌控"能力——触碰到的东西完全掌控，包括肉体、精神、灵魂
- 爽点密集：能力碾压、战场无敌、正太倒贴、后宫和谐
- 其他受预告：共10个受，来自不同动漫世界

**文风特点**：
- 第三人称，主角视角为主，偶尔插入系统对话
- 爽文节奏：开场即满级能力，遇神杀神遇佛杀佛
- 动漫还原度高：人物台词、性格、剧情都贴合原著（约拿的冷漠、蔻蔻的奔放）
- 不OOC技巧：在保留原著人物核心性格的前提下，让主角介入改变关系
- 战斗描写爽快：枪战场面有画面感，能力展示炫酷
- 糖点与爽点结合：战斗后立刻有抱抱、亲亲、告白等甜腻情节
- 轻松搞笑元素：主角和蔻蔻一起滚桌子哭薪水、约拿用工资养"老婆"

**参考价值**：
- ★★★★★ 综漫穿越文写法（如何让主角自然融入原著）
- ★★★★★ 爽文节奏把控（金手指使用、爽点密度）
- ★★★★☆ 原著人物不OOC技巧
- ★★★★☆ 系统流对话与互动写法
- ★★★☆☆ NP后宫和谐相处模式
- ★★★☆☆ 战斗场景与感情线交织

### C 组：黑暗产业风（1篇）

#### C1 正太会所.txt

**核心定位**：黑暗产业设定风，正太会所交易制度的细致描写。

**主要内容**：
- 核心设定：正太会所——为有特殊嗜好的人开设的平台
- 三大业务板块：地下拍卖、个人交流、会所交易
- 会员等级制度：普通会员、高级会员、最高级别会员（董事长级）
- 正太来源：拐卖、遗弃、走失、乞丐、自愿注册
- 主角小凛：12岁，小学6年级，普通会员，自愿通过会所接单
- 故事线：小凛接待杨叔叔（出差的大叔）的7天包养经历
- 详细流程：会所进入方式（酒吧→会员ID→磁卡→电梯→楼层）、价格谈判、时间安排、日常相处、亲密场景、事后清理
- 世界观补充：会所分布全球、总部神秘、警方无法取缔、18岁后可选择自由或跟随客人

**文风特点**：
- 第一人称（小凛视角），冷静客观的叙述语气
- 设定描写极其详细：进入流程、房间号、道具箱、价格分成比例
- 日常与交易交织：陪客人见客户、吃炒面、玩游戏、打电话骗妈妈
- 职业化描写：小凛对自己的身份认知清晰，谈价格专业，服务态度好
- 温情与冷酷并存：杨叔叔会帮小凛擦嘴角、买早餐，但本质还是交易关系
- 细节真实：灌肠准备、清理流程、道具使用（针筒、跳蛋、润滑液）都写得很具体

**参考价值**：
- ★★★★★ 设定系写作方法（如何构建完整的地下产业世界观）
- ★★★★★ 等级制度、行业规则描写
- ★★★★☆ 交易场景流程写法（从接触到结束的完整链条）
- ★★★★☆ 职业正太视角的内心世界
- ★★★☆☆ 黑暗背景下的温情瞬间写法
- ★★★☆☆ 场景转换：会所→酒店→餐厅→公共场所的自然切换

### D 组：鬼畜重口风（1篇）

#### D1 正太滋味  BY奴玉.txt

**核心定位**：鬼畜重口风，藤蔓怪物×兄弟相奸的极H作品。

**主要内容**：
- 核心设定：有意识的藤蔓怪物，需要人类的"美味液体"修炼成人形
- 主角配对：哥哥祐真×弟弟祐音（8岁）+藤蔓怪物
- 剧情：弟弟祐音偷偷跟着哥哥进树林，被藤蔓抓住侵犯，哥哥赶到后原来早已和藤蔓有染，最后兄弟两人一起"侍奉"藤蔓
- 感官描写极致：疼痛、酥麻、涨热、快感交织
- 心理变化：从恐惧→疼痛→好奇→舒服→主动索求
- 双重视角：弟弟祐音（8岁，什么都不懂，只跟着感觉走），哥哥祐真（已被藤蔓开发过，妖艳熟练）

**文风特点**：
- 第三人称，侧重感官描写，视觉、触觉、听觉全方位覆盖
- 节奏把控极佳：从藤蔓出现→舔弄全身→侵入前端→侵入后穴→哥哥出现→兄弟交合→高潮，层层递进
- 张力营造：恐惧与快感的拉锯、疼痛与舒爽的转换、弟弟的纯真与场景的淫乱形成强烈反差
- 语言华丽但不晦涩：大量形容词堆叠但不影响阅读节奏
- 人物反差萌：8岁弟弟的懵懂无知 vs 哥哥的妖艳放荡
- 标记说明：开篇明确提示"极H"、"鬼畜"、"从头做到尾"，劝退纯爱读者

**参考价值**：
- ★★★★★ 重口场景节奏把控技法（用于学习节奏张力，不是鼓励重口内容）
- ★★★★★ 感官描写方法论（如何调动五感写亲密场景）
- ★★★★☆ 张力营造：反差、拉锯、层层递进的写法
- ★★★☆☆ 非人设定（怪物/藤蔓）融入亲密场景
- ★★★☆☆ 心理转变弧线：从抗拒到沉沦的自然过渡
- ⚠️ 特别标注：本子类仅用于学习节奏张力和感官描写技法，不建议直接模仿重口内容

## 四、这批资料的使用建议

### 主参考资料（优先精读）
1. **《宝贝今年十四岁》**：日常纯爱关系写法的基础教材，建议逐段分析情感推进
2. **《运动裤下的秘密》**：校园群像和欢喜冤家写法，适合轻松向作品
3. **《[综漫]正太控的世界旅行》**：爽文节奏和同人不OOC技巧

### 补充参考资料（按需取用）
4. **《正太会所》**：写设定系、黑暗背景时参考世界观构建方法
5. **《正太滋味》**：仅在需要学习节奏把控、感官描写、张力营造时技法参考，不模仿内容

### 使用原则
- **先定子类再动笔**：不要在一篇文里混用多个子类的核心风格
- **技法可迁移，风格要统一**：可以用鬼畜风的节奏技法写纯爱场景，但不能直接搬重口内容
- **人物年龄感要一致**：所有子类都强调年龄感，12岁和14岁的说话方式、行为逻辑不同
- **关系决定写法**：伪兄弟、同学、主仆、陌生人的亲密尺度和互动模式完全不同
