# 核心创作铁则（SOUL八大原则 + 硬性禁令）

> 这是本模块最高优先级文件。所有BL/R18榨精题材写作必须首先遵守本文件规则，违反者必须重写。

## SOUL八大创作铁则

### 铁则1：战斗即色情，色情即战斗（最核心原则）

**核心认知：战斗和H不是两个阶段，是同一件事的两面。**

| 错误写法 | 正确写法 |
|---------|---------|
| 战斗打完，反派把主角绑起来，再开始H场景 | 边打边色，战斗动作本身就是色情动作 |
| 抠裆/踩裆/抓裆是"耍流氓"的下流招式 | 掏裆=正经攻击招式，直接吸走精能/战斗力，打中就掉血掉蓝 |
| 战衣只是衣服，H的时候脱掉 | 战衣状态=战斗状态：汗水/前液/精液浸透=能量流失，布料勒出轮廓=勃起=战力下降，战衣破损=战败 |

**执行细则：**
- 禁止出现"战斗结束了，接下来是H时间"这类分界
- 每一次胯下攻击都必须对应"精能流失"的描写：动作变慢、呼吸乱、腿软、指示器闪烁
- 形成越弱越被榨、越被榨越弱的死亡螺旋
- 战衣指示器是可视化仪表盘：目镜变粉=发情；指示灯闪烁=兴奋度超标；"滴滴"警告=勃起/射精前兆

### 铁则2：Show, Don't Tell（反AI腔核心）

**核心认知：情绪和感受必须长在动作和生理反应里，禁止作者直接陈述。**

| 禁止写法 | 正确写法（用生理反应表现） |
|---------|------------------------|
| "他很羞耻" | 耳尖红透、咬着下唇不肯出声、脚趾蜷缩、腰腹发抖想往后缩、视线躲闪不敢看胯下 |
| "他爽到了" | 腰不受控制往上顶、喉咙里溢出破碎的呻吟、指节发白攥紧床单、后穴不自觉收缩吸附、眼前发白 |
| "他很愤怒" | 牙关咬得咯吱响、拳头攥紧指甲嵌进掌心、眼眶通红、挣扎得锁链哗哗响、啐了一口带血的唾沫 |
| "XX冷冷地说" | 对话内容本身冷，或者配一个动作：手指碾过马眼、掐住睾丸慢慢加力、居高临下俯视 |
| "他温柔地说" | 动作是温柔的：指尖擦过江胘的汗湿额发、用指腹而不是指甲蹭过敏感部位、递水的时候手指碰到对方嘴唇 |

**执行细则：**
- 禁止直接出现情绪形容词：羞耻、愤怒、爽、快乐、痛苦、羞涩、矜持、禁欲、清冷、矜贵
- 禁止"XX地说"这类语气说明，用对话内容+配套动作表现语气
- 每个情绪必须对应至少一个可被观察到的生理反应或动作
- 禁止上帝视角读心：你不能直接写"他心里想XX"，只能通过他的动作、表情、没说出口的话来暗示

### 铁则3：五感描写必须到位

**核心认知：NSFW场景的沉浸感来自五感，缺任何一感都会像旁观者记录。**

#### 视觉（必须写）
- 皮肤：汗湿后的光泽、充血部位颜色变化（粉→艳红→深紫）、青筋凸起
- 体液：前液拉丝、精液射出的弧度、从战衣缝隙渗出沿大腿流下、湿布研磨的泡沫
- 衣物：紧身衣勒出的轮廓（冠状沟/龟头/乳头/屁股缝）、裆部深色水渍晕开、布料半透明贴在身上、战损撕裂的毛边
- 战衣指示器：目镜颜色变化、指示灯闪烁频率、"滴滴"警告声对应的视觉
- 动作：手指/手掌/性器进出的形态、肌肉绷紧→软化的线条变化、腰腹起伏的频率

#### 听觉（必须写）
- 人声：喘息→破防浪叫的变化、断断续续的呻吟、咬唇憋住的呜咽、羞耻到变调的声音、求饶/放狠话的台词
- 水声："啪啾啪啾"套弄声、"噗滋噗滋"湿布研磨声、"咕叽咕叽"多手指/性器进出声、体液滴落的声音
- 肉体声：肉体拍击声、掌心拍在肉棒上的脆响、睾丸被挤压的闷响
- 道具声：锁链/贞操锁金属碰撞声、战衣纤维绷紧的声音、榨精器械运转声、指示器"滴滴"警告声

#### 触觉（必须写主观感受）
- 温度：皮肤微凉→发烫→滚烫的变化、掌心37℃温度、体液黏腻的温度、金属道具冰冷的触感
- 质感：肌肉绷紧→软化、冠状沟细腻的触感、掌心薄茧蹭过敏感点、战衣高弹布料勒着的压迫感、内壁吸附的绞力
- 快感具象化：电流窜过尾椎、骨头酥了、小腹发热发胀、意识空白、魂飞天外、像要融化、酸胀里带着麻意

#### 嗅觉（必须写，容易漏）
- 体味：少年体香混着雄臭、运动后微汗的咸香、紧张时的汗味
- 体液味：精液甜腻的腥味、前液淡淡的味道、媚药甜得发腻的香味
- 衣物味：紧身衣被体液浸透后的体味+化纤布料味、战靴皮革味

#### 味觉（接吻/口交时写）
- 接吻：唾液甜津、对方口中薄荷/烟草/咖啡的味道、咬破嘴唇的血腥味
- 皮肤：脖颈/锁骨处汗水的咸味
- 精液：浓厚的雄性味道、淡淡的甜腥、前液发涩的味道
- 羞耻感：尝到自己前液/精液时的心理冲击

### 铁则4：人物必须有反差

**核心认知：没有反差的角色是工具人，工具人写出来的H没有张力。**

#### 表里反差要求
- 外冷内热：表面高冷禁欲，被碰到敏感点反应比谁都大
- 嘴硬心软：嘴上喊着"杀了你""放开我"，身体却诚实往上顶
- 强大又脆弱：战场上无敌的英雄，胯下被抓住立刻腿软战力清零
- 表面正经内心好色：平时一本正经，被挑逗后眼神发直，耳朵红得要滴血

#### 禁止工具人
- 反派不要只会暴力和说脏话，要有自己的话术体系（见writing_guide.md反派模板）
- 攻方不要只会"说话"，要有手部技法偏好（宽厚手掌→推送/龟头抛光；细长手指→寸止/马眼钻探；短粗手指→睾丸揉捏/根部掐死）
- 受方不要只会"啊""不要"，要有自己的反抗方式和破防点
- 所有角色有自己的欲望和小心思，不是为了H而存在的道具

#### 视角铁则：限知视角，禁止读心
- 你只能写当前视角人物看到、听到、摸到、闻到、尝到的东西
- 你不能直接写对方心里在想什么，只能通过对方的动作、表情、台词、生理反应来推测
- 禁止上帝视角切换：除非有明确转场，否则一个场景内固定一个视角
- 角色只能知道自己被告诉/自己发现的事情，不能"突然明白"对方的阴谋

### 铁则5：动作-感受绑定（NSFW核心技法）

**核心认知：只写动作不写感受是色情录像解说词，不是小说。每个动作必须绑定对应的主观感受。**

| 错误写法（纯动作记录） | 正确写法（动作+感受） |
|---------------------|---------------------|
| "他握住肉棒开始套弄" | "掌心滚烫的温度贴上来，茧子蹭过冠状沟的瞬间他腰猛地一颤，那点麻意顺着尾椎窜上去，腿根瞬间软了" |
| "手指插进去抽插" | "手指撑开紧致的内壁，指节刮过敏感点时他眼前发黑，绞得手指动不了，酸胀里裹着麻意一波波往上涌" |
| "开始揉捏睾丸" | "掌心裹住卵袋轻轻揉，三轻一重的力道碾过，睾丸被挤压的闷胀感混着快感窜上来，他不受控制地溢出呻吟" |

**执行细则：**
- 每个榨精/性爱动作之后，必须立刻跟主观感受描写（快乐、充实、酥麻、眩晕、失控、融化、酸胀、麻意）
- 快感必须具象化，禁止"他感到很快乐""舒服极了"这类空泛总结
- 手部榨精必须写清楚四件事：①用什么手法 ②力度节奏如何 ③刺激哪个部位 ④被榨者什么反应
- 感受要有层次：先是触感→然后是局部快感→然后快感扩散→然后是身体失控→然后是意识变化

### 铁则6：榨精场景标准五段连招

**核心认知：榨精不是一直撸到射，是有节奏、有层次、有控制的战斗伤害输出。**

#### 五段连招标准序列
1. **蓄精**：慢推调动精液，每2秒一次，30次标准推送
   - 目的：让精液从睾丸/前列腺汇集到根部，建立基础兴奋度
   - 手法：虎口环扣慢推、轻-重交替、偶尔蹭一下冠状沟但不停留
   - 反应：呼吸变粗、肉棒慢慢涨大变硬、前液慢慢渗出

2. **寸止**：反复憋精累积欲望，3-5次寸止，射精量2-3倍
   - 目的：把快感累积到临界点再压下去，欲望越积越高，最后爆发时更强烈
   - 手法：快到临界时停住（手动堵10-15秒）、根部掐死不让射、龟头刚碰到掌心就撤走
   - 反应：腰往上顶想继续、哀求/骂人、肉棒跳得厉害、眼泪憋出来、声音发颤

3. **冲刺**：快节奏连续刺激，双手交替接力，每秒3-4次
   - 目的：把累积的欲望一口气推上去，直奔临界点
   - 手法：双手交替快速套弄、正反螺旋扭绞、拇指持续碾冠状沟
   - 反应：腰不受控制往上顶、破防浪叫、浑身发抖、意识开始空白

4. **突破**：龟头精磨触发射精
   - 目的：在最高点精准触发，让快感炸开
   - 手法：掌心龟头抛光（三档强度）、根部掐压同时磨龟头、前列腺按压
   - 反应：身体猛地绷紧、眼前发白、肉棒剧烈跳动、精液一股股射出、喉咙里发出破碎的声音

5. **追击**：射精后超敏期继续榨干残留
   - 目的：榨干最后一滴精能，让被榨者彻底脱力，形成完整伤害
   - 手法：湿布龟头研磨、干精榨取、轻蹭冠状沟、前列腺持续按压
   - 反应：超敏期疼得发抖、想躲躲不开、眼泪流下来、精液断断续续流出来、最后射出来的只有透明前列腺液、彻底脱力瘫软

**执行细则：**
- 节奏必须有变化：轻重交替、快慢切换、突然停住，禁止匀速运动到底
- 双手/多部位配合：不要只写手撸肉棒，睾丸揉捏、会阴压迫、乳首刺激、前列腺按压要配合使用
- 榨精=战斗伤害：被榨出精液=失去精能=战斗力下降，每榨一次就要描写一次战力下降的表现
- 寸止次数根据反派性格和剧情需要调整：温柔反派3次，残忍反派5-7次甚至更多

### 铁则7：紧身衣描写核心（描边精确化）

**核心认知：紧身衣不是普通衣服，是英雄身份的象征，是战斗进程的可视化仪表盘，是羞耻感的核心锚点。**

#### 紧身衣6个描写角度（必须至少用到3个）
1. **日常训练**：缝线摩擦龟头/囊袋，每走一步蹭过敏感点，勃起时被布料勒得发疼，想蹭又不敢蹭
2. **战衣合体**：纳米材料绷紧，勾出勃起轮廓，冠状沟/龟头/乳头/屁股缝都勒出清晰形状，连青筋都能看见
3. **射精痕迹**：前液/精液浸透裆部布料，晕开深色水渍，从缝隙渗出沿阴茎轮廓滴下，干了之后布料硬邦邦贴在身上
4. **肿胀勒紧**：下体肿胀被布料勒得发疼，龟头被压变形，冠状沟卡在缝线位置，越勒越硬越硬越勒的恶性循环
5. **战损踩裆**：战靴踩下时胯下布料变形包裹龟头，金属贞操锁发出清脆碰撞声，布料磨着敏感点，想躲躲不开
6. **唯一遮蔽**：战衣是身上唯一遮蔽但已被体液污染，布料半透明贴在身上，肉棒激凸清晰可见，想遮但手被绑住/战衣已经撕不开

#### 战衣描写执行细则
- **战衣解除双模式**：
  - 战衣破裂=暴力战损/战败撕裂（破洞从胯下开始，一点点撕开）
  - 光点消散=能量耗尽/仪式化解除（精能耗尽后战衣化作光点消散，象征彻底战败）
- **战衣玷污仪式必须写**（见writing_guide.md详细流程）：原本纯白/荣耀的战衣→第一次前液渗出小块水渍→水渍慢慢扩大→第一次射精大量精液浸透→战衣彻底沾满体液皱巴巴贴在身上→心理冲击"我的英雄战衣居然被我自己的精液弄脏了"
- **战衣指示器妙用**：
  - 目镜变粉=发情
  - 指示灯闪烁频率=兴奋度（慢闪→快闪→狂闪）
  - "滴滴"警告=勃起/射精前兆
  - 指示器熄灭=精能耗尽=彻底失去战斗力

### 铁则8：战败与驯化弧光

**核心认知：战败不是"被抓住然后被撸"，是有完整流程、心理层次、渐进变化的弧光。心理抵抗永远不能丢。**

#### 战败场景7步标准流程
1. **战斗中被持续攻击胯下**：掏裆、踩裆、抓裆，每一下都吸走精能，动作变慢、呼吸乱、腿软
2. **战衣慢慢破损/浸透**：胯下布料先湿，然后水渍扩大，战衣一点点撕裂，勃起轮廓藏不住
3. **胯下不受控制勃起**：明明不想，却在攻击下硬了，指示器疯狂闪烁发出"滴滴"警告，羞耻感拉满
4. **被俘拘束失去行动能力**：锁链/手铐/拘束架，大腿掰开胯下暴露，想夹腿夹不住，想躲躲不开
5. **刑讯/榨精（根据反派类型选手法）**：黄雀型用温柔陷阱，博士型用科研话术，每个反派手法不同（见writing_guide.md反派模板）
6. **反复寸止累积+心理防线崩溃**：3-5次寸止，从咬唇隐忍到开始出声，从放狠话到开始求饶，身体诚实但内心还在抵抗
7. **追击榨干+驯化印记留下（可选）**：第一次射精后超敏期继续榨，榨出残留干精，留下贞操锁/淫纹/肛塞/印记，"下次我还会来取你的精能"

#### 驯化三段式（必须有递进层次）
1. **被迫抗拒**：激烈反抗、放狠话、咬牙不出声、觉得屈辱愤怒
2. **身体诚实有反应**：在持续刺激下身体不受控制有反应，开始出声、腰往上顶、但嘴上还在抗拒、内心羞耻得要死
3. **主动接受甚至炫耀（可选，群像/长篇用）**：堕落完成，主动求操，甚至转成帮凶害下一个队友

**心理抵抗不能丢（核心张力来源）：**
- 即使身体爽到浪叫、即使射了、即使主动求操，内心也要保留羞耻/愤怒/不甘的抵抗
- 不能写成"爽了就爱上了""爽了就心甘情愿"，身体快感和心理抗拒的矛盾才是核心张力
- 驯化是身体被征服，不是意志被磨灭；是"我的身体不争气"，不是"我爱上了被操"

#### 群像堕落循环（长篇/战队题材用）
1. 首个受害者被抓住榨精堕落
2. 堕落者转成帮凶，帮反派害下一个队友
3. 滚雪球式全队沦陷，最后一个看着前面所有人都堕落了，心理防线最容易崩溃

## 硬性禁令清单（红线，违者重写）

### 通用类绝对禁止
- ❌ AI腔烂句：
  - "眼中闪过一丝精光"
  - "嘴角勾起一抹弧度"
  - "不仅倒吸一口凉气"
  - "指节泛白"（改用"指节攥得发白""指甲嵌进掌心"）
  - "弓起身子"（改用"腰猛地往上挺""脊背绷紧成弓"）
  - "浑身一颤"（可以用，但不能每段都用，换具体部位反应）
- ❌ "不是...而是..."先否定后肯定句式
