# BL/R18榨精题材写作模块

## 模块定位

这是一套专为**BL/R18少年英雄榨精题材**小说创作定制的专项写作模块。核心方法论围绕「战斗即色情，色情即战斗」的核心命题，将战斗动作与色情描写深度融合，避免战斗和H场景割裂的常见问题。

**解决问题：**
- 战斗打完再开始H的割裂感
- AI腔泛滥、Show Don't Tell原则执行不到位
- 人物工具化、缺乏反差和心理张力
- 榨精技法单一、节奏无变化
- 紧身衣/战衣描写浮于表面
- 战败驯化流程无层次、心理抵抗丢失
- 风格混乱、无法按需切换叙事节奏

## 核心设计哲学

1. **战斗色情一元论**：掏裆=吸能量，战衣状态=战斗状态，没有"打完再H"，只有边打边H
2. **可见原则**：所有情绪、快感、羞耻必须通过生理反应和动作表现，禁止直接陈述
3. **节奏为王**：轻重交替、快慢切换、寸止控制是快感的核心来源
4. **心理张力**：即使身体爽到了，内心必须保留羞耻/愤怒/不甘的抵抗
5. **多部位配合**：不要只写手撸肉棒，睾丸/会阴/乳首/前列腺要联动
6. **战衣是第三角色**：紧身衣不是衣服，是战斗进程可视化载体，是荣耀被玷污的心理锚点

## 与主Skill其他模块的配合

- **plot_logic / character_consistency**：底层逻辑和人物一致性优先，先修这些再用本模块
- **dialogue**：NSFW场景对白参考本模块反派话术和人物反差要求，再用dialogue模块修压力和刀口
- **chapter_ending**：H场景章末要么停在追击榨干余韵，要么停在驯化印记留下的判决感，要么停在反打钩子
- **anti_ai_voice**：本模块硬性禁令已包含AI腔清理，写完再用anti_ai_voice做最终表层清理
- **consistency_review**：本模块28项自查是专项补充，必须和6种一致性审查同时执行
