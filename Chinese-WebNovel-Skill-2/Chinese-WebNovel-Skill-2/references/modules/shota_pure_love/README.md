# 正太纯爱文风模块

这是一个专门处理中文网文正太纯爱题材写作风格的独立模块。

这次模块的组织方式，按三大文风体系组织，不再把正太纯爱仅仅当成"一种题材"，而是拆成体系化的写作方法：

- 星辰风：二次元萌系纯爱（第三人称，强反差开篇，萌系语言，纯爱肉渣）
- 房东风：第一人称写实后宫甜宠（第一人称内心吐槽，生活化对话，日常→亲密自然过渡，多人物和谐后宫）
- 其他风格群：5种补充风格（黑暗英雄调教、日常纯爱、综漫爽文、黑暗产业、鬼畜极H）按场景调用

## 模块目标

这个模块主要解决：

- 正太纯爱题材人物形象单薄
- 语言不够萌/不够生活化
- 情感递进太快或太慢
- 日常和亲密场景过渡突兀
- 多人物关系处理不好
- 风格不统一（要么太萌要么太写实）

## 现在这套模块怎么读

推荐按这个顺序用：

1. 先读 [source_index.md](source_index.md)，知道这批资料的来源和重点。
2. 再读 [tutorial.md](tutorial.md)，建立统一的正太纯爱文风方法框架。
3. 写作或改稿时，按 [runtime.md](runtime.md) 的顺序判断问题类型。
4. 去 [good_examples.md](good_examples.md) 找 2-4 个近似正例。
5. 去 [bad_examples.md](bad_examples.md) 找 1-2 个近似反例。
6. 先定调，再写正文。

## 说明

- 当前模块的"主参考"是星辰风和房东风，覆盖80%以上正太纯爱写作场景。
- 其他风格群仅作特定场景补充，不建议作为主风格使用。
- 正例强调可迁移结构，不强调句子皮相。
- 反例不是为了嘲笑拙劣写法，而是为了帮助模型识别人物扁平、语言生硬、节奏失控、过渡突兀、关系混乱、风格割裂这些常见失误。
- 所有内容聚焦于写作技巧、风格把控、叙事结构、人物塑造，不涉及露骨描写。
