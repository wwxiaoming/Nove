# 星辰风模块

这是一个专门写二次元萌系奇幻纯爱（冷酷杀手×软萌正太）的独立写作模块。

这次模块的组织方式，已经按 `sources/raw/星辰之下.txt` 的核心技法重排过，不再把萌系纯爱仅仅当成"可爱词汇堆砌"，而是拆成六层：

- 新手能立刻上手的强反差开篇法
- 萌系词汇库（叠词/拟声词/语气词/萌系词汇）
- 人物反差萌分层塑造（基诺三层反差/辰辰五维萌点）
- 情感节奏12阶段细化（从相遇到H到确认关系）
- 意象系统（星辰6层含义/狼羊隐喻贯穿）
- 纯爱肉渣H场景专属写法（重感受轻动作、点到即止）
- 适配网文写作时的运行顺序（诊断→计划→动笔）

## 模块目标

这个模块主要解决：

- 开篇不够抓眼，没有强烈反差
- 萌点写得假，像大人装可爱，不是真小孩
- 叠词/萌词滥用，看得人尴尬
- 攻受反差只写在设定里，行动上看不出来
- 情感推进太快或太慢，没有节奏
- 意象用一次就丢，没有贯穿
- H场景要么太突兀破坏萌感，要么太露骨像色情文
- 傲娇心理写得像精分，不是真实的口是心非
- 日常写得无聊，没有萌点
- 人物容易OOC，攻突然变受或者受突然成熟

## 星辰风适用场景

**最适合写：**
- 冷酷/强大/黑暗身份攻 × 软萌/天真/阳光受
- 正太纯爱（年龄差3-5岁最佳）
- 奇幻/都市背景下的身份反差（杀手/超能力者/妖怪 × 普通小孩）
- 日常甜文为主，带少量肉渣
- 治愈系、温暖系、家庭温馨感
- 有核心意象贯穿的故事（如星辰、狼羊、某种信物）

**不适合写：**
- 纯肉文、无感情直接上
- 强取豪夺、虐身虐心
- 成年人间的爱情故事
- 现实主义严肃题材
- 快节奏爽文（需要慢热铺垫）

## 现在这套模块怎么读

推荐按这个顺序用：

1. 先读 [source_index.md](source_index.md)，知道《星辰之下》的章节重点和技法分布。
2. 再读 [tutorial.md](tutorial.md)，建立统一的星辰风写作框架。
3. 写作或改稿时，按 [runtime.md](runtime.md) 的顺序诊断问题、制定计划。
4. 去 [good_examples.md](good_examples.md) 找2-5个近似正例，模仿写法。
5. 去 [bad_examples.md](bad_examples.md) 找1-3个近似反例，避开雷区。
6. 先搭情感节奏框架，再写具体场景。

## 核心原则

1. **反差要落地**：攻的冷酷不能只靠旁白说"他很冷酷"，要通过他的行动（杀人不眨眼、说话简洁、对旁人冷漠）来体现；他的温柔也不能只靠旁白说"他其实很温柔"，要通过他对受的特殊待遇（嘴上嫌弃身体诚实、偷偷守护、只对受心软）来体现。
2. **萌要真实**：受的萌不是说叠词、装傻，是8岁小孩真实的逻辑（想吃、想玩、想撒娇、会吃醋、会假哭、会得意忘形）。
3. **H要纯爱**：肉渣的核心是"爱"不是"欲"，重点写心理感受（害羞、舒服、喜欢、安心），动作点到为止，不插入正面描写，事后一定要有情感对话。
4. **意象要贯穿**：核心意象（如星辰、五角星挂坠、狼羊）不能只出现一次，要在关键节点反复出现，承载情感变化。
5. **节奏要慢**：情感是一步步推进的，从陌生→好奇→在意→喜欢→爱→身体接触→确认关系，每一步都要有具体事件支撑。

## 说明

- 当前模块的"主参考"是 `sources/raw/星辰之下.txt`。
- 正例强调可迁移结构，不强调句子皮相，学的是"怎么写出这种感觉"而不是"抄这句话"。
- 反例不是为了嘲笑拙劣写法，而是为了帮助识别叠词滥用、反差写在设定里、H场景破坏萌感这些星辰风特有失误。
- 记住：星辰风的核心不是"写得可爱"，是"用最冷酷的外壳包裹最柔软的爱"，是"狼爱上羊后心甘情愿收起利爪"的反差感。
