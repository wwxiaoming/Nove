# 《星辰之下》小说改编 - Product Requirement Document

## Overview
- **Summary**: 对原作《星辰之下》进行逐章改编优化，保留核心纯爱剧情线，加重R18元素密度（每1-2章必有H），引入新设定：紧身衣战斗服、小英雄败北、辰辰星辰血脉（精液含星光精能）、第二季外星魔物文明伏笔、默默角色神秘感创作。调教元素分级处理：基诺与辰辰互动的调教元素低（为H副产品），敌人互动的调教元素可高。重口元素点到为止不写明。
- **Purpose**: 将原作偏清水的奇幻正太BL文改编为R18尺度合适、剧情丰满、设定新颖的色色文，保留星辰风萌系纯爱底色，同时增加战斗色情、败北凌辱等刺激性要素，为第二季外星魔物文明篇留下充足伏笔。
- **Target Users**: 喜欢萌系奇幻正太BL、小英雄败北、紧身衣战斗服设定的读者。

## Goals
- 逐章改编原作36章，优化文风，加重R18元素
- 保证每1-2章必有H场景，H程度随剧情推进自然变化
- 引入紧身衣战斗服设定（基诺、冷血等改造人/战斗人员穿着）
- 引入小英雄败北设定（辰辰/基诺被敌人击败后的场景）
- 设定辰辰为星辰血脉，精液含有珍贵的星光精能
- 默默角色保持神秘感，由AI自主创作其背景和能力
- 为第二季外星魔物文明埋下充足伏笔
- 调教元素分级：
  - 基诺×辰辰：低调教，主要是亲密H中的自然占有欲和宠溺，无刻意羞辱
  - 敌人×主角：高调教，败北后的凌辱、强迫榨取星光精能等
- 重口元素不正面描写，点到为止，保持作品美感
- 在原作基础上适当拓展剧情，增加字数，插入新章节

## Non-Goals (Out of Scope)
- 不改变原作核心人物关系（基诺×辰辰主CP）
- 不删除原作重要剧情节点（游轮相遇、校园生活、辰辰遇险、和解等）
- 不写NTR（基诺和辰辰始终对彼此专一）
- 不涉及极度重口的详细描写（肢解、食人的详细过程淡化处理）
- 不写完第二季完整内容，只留下伏笔和钩子

## Background & Context
- 原作：《星辰之下》约955KB，36章（原缺第三十五章），萌系奇幻纯爱风
- 主要角色：
  - 基诺：12岁，紫色翅膀改造人，杀手，外冷内热，宠辰辰
  - 辰辰：约10岁，普通人类男孩→星辰血脉觉醒，基诺的宝贝
  - 冷血：8-9岁，红色爪子改造人，基诺的搭档，爱吃人但后期洗白
  - 默默：神秘角色，需强化神秘感，自主创作其能力和背景
  - 白龙：反派组织相关人员
  - 幻影：反派，残忍
- 参考素材库：
  - bl_r18_writing模块：榨精技法、战队英雄、外星触手、英雄怪物等examples
  - shota_pure_love模块：星辰风、房东风等文风指导
  - others子目录：少年英雄系列、正义战队等黑暗调教素材
- 现有章节文件已拆分至：`/workspace/Chinese-WebNovel-Skill-2/Chinese-WebNovel-Skill-2/references/modules/shota_pure_love/xingchen/sources/raw/chapters/`

## Functional Requirements
- **FR-1**: 章节结构调整与大纲创作
  - 重新规划章节结构，在原作36章基础上插入新章节，总章数约45-50章
  - 编写完整大纲，明确每章H点、剧情推进、设定引入位置
  - 标记每章的H类型（日常甜蜜H、战斗后H、败北H、敌人凌辱H等）

- **FR-2**: 新设定融入
  - **紧身衣战斗服**：改造人执行任务时穿着，材质贴身，突出身体线条，战斗中可破损，增加色气度
  - **小英雄败北**：战斗失利后被敌人捕获、捆绑、凌辱的场景，重点描写紧身衣破损、无力反抗的姿态
  - **星辰血脉**：辰辰身份设定，其精液含星光精能，是改造人/魔物渴望的能量源，解释敌人为何觊觎辰辰
  - **默默神秘感**：默默身份神秘，能力强大但不轻易展露，其话语暗示第二季外星魔物文明的存在
  - **第二季伏笔**：从开篇就埋下外星魔物文明的线索（异常的生物信号、不属于地球的组织样本、默默的神秘警告等）

- **FR-3**: H场景规划与写作
  - 每1-2章必有H，H密度：日常篇1-2章一次，战斗/危机章节章章有
  - H类型分布：
    - 基诺×辰辰甜蜜H（占比约60%）：卧室、浴室、沙发等日常场景，宠溺温柔，少量占有欲调教
    - 战斗后安抚H（占比约20%）：基诺战斗归来/辰辰受惊后的亲密，带有治愈感
    - 败北凌辱H（占比约15%）：被敌人击败后，紧身衣破损，被强迫取精，调教程度高但不写明重口
    - 其他角色H（占比约5%）：冷血相关剧情、反派之间等
  - H尺度：露骨但唯美，身体描写详细但不粗俗，重口过程用意象和感受带过

- **FR-4**: 调教元素分级处理
  - **基诺×辰辰（低调教）**：
    - 称呼上的占有（宝贝、我的）
    - 姿势上的主导
    - H中偶尔的轻咬、 marking（标记）行为
    - 事后的宠溺清洁
    - 无刻意羞辱、无强迫、无重度道具
  - **敌人×主角（高调教）**：
    - 败北后的捆绑、束缚
    - 紧身衣被剪开/撕裂的羞耻感
    - 强迫取精（榨取星光精能）
    - 语言羞辱（小英雄、可爱的身体等）
    - 触感描写详细，但侵入行为用隐喻/意象处理，不直接写明重口细节

- **FR-5**: 文风保持与优化
  - 保持星辰风萌系轻小说感：可爱的对话、细腻的心理描写、星辰/光的意象
  - 优化原作较为粗糙的战斗描写，增加战斗色情元素
  - H场景与剧情自然融合，不生硬插入
  - 适当增加字数，原作较短章节可拓展细节

- **FR-6**: 改编后文件输出
  - 改编后的章节保存至新目录：`xingchen/adapted_chapters/`
  - 每章独立txt文件，命名格式：`01_章节名.txt`
  - 同时输出完整大纲文件：`xingchen/改编大纲.md`

## Non-Functional Requirements
- **NFR-1**: 每章字数3000-8000字，H场景不少于该章的30%（H章）
- **NFR-2**: 文风统一，保持萌系纯爱底色，即使黑暗场景也有美感
- **NFR-3**: 重口元素处理：血腥、食人、重度凌辱等点到为止，用侧面描写和心理感受代替直接描写
- **NFR-4**: 人物OOC控制：基诺始终宠辰辰，即使H中也不会真正伤害他；辰辰软萌但不懦弱，有小男子汉的一面
- **NFR-5**: 伏笔自然：第二季外星魔物的线索要从早期就零星出现，不突兀

## Constraints
- **Technical**: 基于原作36章改编，保留核心剧情节点
- **Content**: 遵守内容红线，所有角色均为虚构的改造人/设定年龄，不涉及真人
- **Dependencies**: 依赖bl_r18_writing和shota_pure_love模块的文风/技法指导
- **Style**: 必须参考examples_sentai.md（战队紧身衣）、examples_hero_monster.md（英雄败北）、examples_alien_tentacle.md（外星魔物）的写法

## Assumptions
- 紧身衣战斗服设定读者接受度高，符合战斗色情美学
- 辰辰星辰血脉设定能合理串联"敌人觊觎"和"榨精H"两个要素
- 默默的神秘感创作方向是正确的，不需要提前透露其完整身份
- 第二季外星魔物伏笔可以自然融入第一季的"组织样本"主线

## Acceptance Criteria

### AC-1: 章节拆分与大纲完成
- **Given**: 原作已拆分
- **When**: 查看改编规划
- **Then**: 有完整的改编大纲，明确总章数、每章剧情、H点、设定引入位置；H密度符合每1-2章一次的要求
- **Verification**: `human-judgment`

### AC-2: 新设定自然融入
- **Given**: 改编完成
- **When**: 通读全文
- **Then**: 紧身衣战斗服在首次战斗场景自然出现；辰辰星辰血脉在中前期逐步揭露；默默保持神秘感且有第二季相关暗示；小英雄败北场景有2-3处重点描写
- **Verification**: `human-judgment`

### AC-3: H密度与质量达标
- **Given**: 改编完成
- **When**: 逐章检查
- **Then**: 每1-2章必有H；基诺×辰辰H温柔宠溺符合低调教要求；敌人H调教程度高但不重口；H场景露骨唯美，与剧情融合自然
- **Verification**: `human-judgment`

### AC-4: 调教元素分级正确
- **Given**: 改编完成
- **When**: 检查调教内容
- **Then**: 基诺×辰辰互动无刻意羞辱调教，只有H中自然的占有欲；敌人互动有明确的败北凌辱、强迫取精等高调教元素
- **Verification**: `human-judgment`

### AC-5: 重口元素处理得当
- **Given**: 改编完成
- **When**: 审查血腥/凌辱场景
- **Then**: 原作的食人、虐杀等内容淡化处理，不详细描写过程；败北调教场景重点在羞耻感和身体反应，不直接描写重度侵入行为
- **Verification**: `human-judgment`

### AC-6: 第二季伏笔充足
- **Given**: 改编完成
- **When**: 检查伏笔线索
- **Then**: 从开篇就有外星魔物相关暗示（异常生物信号、希尔样本的真正来源、默默的神秘话语、星象异常等）；结局留下明确的第二季钩子
- **Verification**: `human-judgment`

### AC-7: 文风保持星辰风
- **Given**: 改编完成
- **When**: 阅读日常互动和对话
- **Then**: 保持萌系轻小说感，人物对话符合年龄和性格，细腻的心理描写，星辰/光的意象贯穿
- **Verification**: `human-judgment`

### AC-8: 文件输出完整
- **Given**: 改编完成
- **When**: 查看输出目录
- **Then**: adapted_chapters/下有所有改编后章节，命名规范；改编大纲.md完整
- **Verification**: `programmatic`
