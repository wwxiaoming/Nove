# 正太（综合补充）资料索引

这份索引对 `sources/raw/` 下的5篇小说按文风分类，明确各自的特点和参考价值，为模块教程和例库提供来源依据。

## 一、资料总览

当前 `raw/` 下共有5份资料，覆盖正太题材的四大类核心文风+一篇校园纯爱补充。它们不是单一风格的重复，而是分别承担不同场景下的写法参考：

- 当星辰（都市奇幻/养成系）、年轻房东（邻家温馨）、英雄（热血战斗）三个子模块都不匹配时，调用本模块
- 本模块是正太题材的综合工具箱，按场景和文风需要从子类中选择对应写法

## 二、核心资料分组

### A组：日常纯爱风（2篇）

这组资料主打平等关系下的温柔纯爱，适合写邻家、伪兄弟、日常同居等没有强烈身份差或戏剧冲突的场景。

#### A1 《宝贝今年十四岁》 我的宝贝今年14岁 作者：babysee(非小白，伪兄弟,两部全).txt

**主要内容：**
- 伪兄弟设定：大学生房客罗冈 × 房东儿子黎昀（12岁→14岁）
- 日常互动细节：补习、游泳、麦当劳、一起洗澡、同居日常
- 感情线发展：英雄救美初遇→房客身份住入家→暗恋→试探→确认关系→生日承诺
- 生活化道具：DV偷拍、化学麻醉剂意外、情侣戒指、日记
- H场景：温柔前戏、互相抚慰、循序渐进、注意对方感受

**文风特点：**
- 第一人称（攻视角），口语化叙事，像在讲自己的故事
- 细节真实：北京租房、大学生活、小升初压力、麦当劳、广州话穿插
- 心理活动直白：不回避攻的欲望和挣扎，也不回避受的好奇和试探
- 非小白：有矛盾冲突（同学发现视频威胁、麻醉剂事件、父母怀疑）
- 年龄感精准：12岁小学生的言行（看卡通、吃甜品、害羞、说"人家"）到14岁的成长变化

**模块里怎么用：**
- 伪兄弟/邻家/同居日常的核心参考
- 温柔H场景写法：爱抚节奏、前戏铺垫、语言引导
- 生活化细节让感情真实可信
- 年龄差关系的分寸把握：既是照顾者又是恋人

#### A2 《运动裤下的秘密》作者：迷羊.txt

**主要内容：**
- 校园欢喜冤家设定：平民高中vs贵族学校合并背景
- 古云风（平民校，有超能力能看见恋爱星）× 宋捷（贵族校，运动部长，有秘密的处男）
- 轻松搞笑风：两校冲突、室友矛盾、打赌、误会、超能力设定
- 副CP线：校长×校长、雷劲×尹天琪、孟念奇×费子昂（多线并行）
- H场景：欢喜冤家式的张力，从敌对到心动，带点恶作剧和挑逗

**文风特点：**
- 第三人称，多线叙事，节奏明快
- 搞笑元素密集：斗嘴、打脸、误会、内心吐槽
- 超能力设定作为金手指和剧情推动器（看见恋爱星、预知）
- 人物性格鲜明：傲娇少爷×腹黑痞子、洁癖优等生×肌肉笨蛋
- 校园场景真实：宿舍、餐厅、课堂、社团、晚会

**模块里怎么用：**
- 校园纯爱/欢喜冤家的核心参考
- 轻松搞笑文风的节奏把控
- 从敌对到相爱的张力营造
- 多CP并行不杂乱的写法
- 带点挑逗感的H场景写法

### B组：综漫爽文风（1篇）

这组资料主打穿越/综漫/系统文的爽点节奏，适合写快穿、多世界、爽文NP类正太题材。

#### B1 [综漫]正太控的世界旅行(1).txt

**主要内容：**
- 系统快穿设定：主角王维穿越多个动漫世界攻略正太
- 金手指："绝对掌控"超能力（学园都市Lv6级别，触碰即解析、掌控一切）
- 第一个世界：《军火女王》攻略约拿（银发少年兵）
- NP后宫设定：总共10个受，受之间不吃醋和平共处
- 爽点密集：能力开挂、快速收服、战斗戏、原著人物互动、感情线快速推进
- H场景：符合爽文节奏，不拖沓，服务于感情确认和爽感释放

**文风特点：**
- 第三人称，节奏极快，章节短平快
- 系统提示音和内心吐槽穿插
- 战斗场面干脆利落，不拖泥带水
- 原著人物不OOC：约拿的冷漠、蔻蔻的疯狂、团队成员的性格都还原到位
- 爽文逻辑：强就是强，不憋屈，收服正太讲究效率但也有感情铺垫
- 金手指合理运用：不滥用，主要用于装逼、救人、快速学习、推动感情

**模块里怎么用：**
- 综漫/快穿/系统爽文的核心参考
- 爽点节奏把控：每章有爆点、每几章一个小高潮
- 多世界切换的衔接写法
- 原著人物不OOC的技巧：抓住核心性格+关键台词
- NP后宫的和谐写法：受各有特色、不雌竞/雄竞、攻对每个都真心
- 爽文H场景节奏：快准狠，服务于爽感，不拖剧情

### C组：黑暗产业风（1篇）

这组资料主打设定系背景，适合写会所、拍卖、调教、行业规则类黑暗向正太题材。

#### C1 正太会所.txt

**主要内容：**
- 完整世界观设定：正太会所的三大区块（交易区、交流区、拍卖区）
- 等级制度：普通会员→高级会员→最高级别会员（跨国董事长级）
- 分成规则：个人交流正太得80%会所抽20%，会所交易正太得30%，拍卖一次性交易
- 运营细节：会员ID、磁卡电梯、酒吧掩护、道具箱、灌肠清洁、安全措施
- 正太来源：拐卖、遗弃、走失、孤儿、自愿注册
- 主角小凛：12岁小学6年级，交流区正太，接待过4个客人
- 杨叔叔线：46岁国企出差客人，7天包养，从见客户到吃饭到过夜的完整流程
- H场景：写实服务感，注重清洁、安全、客人体验，带交易性质但也有温情互动

**文风特点：**
- 第三人称，设定先行，开篇直接扔世界观
- 细节写实：流程、道具、价格、分成、掩护方式都写得很具体
- 日常与交易交织：陪客人见客户、吃炒面、玩电脑游戏，然后才是床上服务
- 人物不片面：小凛不是完全被迫，有自己的选择权（拒绝不满意的价格），会傲娇会害羞
- 杨叔叔也不是纯粹的恶人，有温柔的一面（擦嘴角、买早餐、盖被子）
- 行业规则感强：入会门槛、保密措施、警方卧底都打不掉的设定增加真实感

**模块里怎么用：**
- 设定系黑暗背景构建的核心参考
- 行业规则和等级制度的写法：从运营模式到分成比例都要具体
- 会所/拍卖/交易场景的H写法：服务感、流程感、清洁准备、事后处理
- 黑暗背景下的温情：不把人物写得完全脸谱化，客人也有温柔，正太也有情绪
- 日常与工作的交织：让世界观更真实可信

### D组：鬼畜极H风（1篇）

这组资料主打鬼畜系的张力和节奏，适合写重口、怪物Play、兄弟鬼畜、非自愿→沉沦类极H题材。

#### D1 正太滋味 BY奴玉.txt

**主要内容：**
- 鬼畜设定：藤蔓怪物×人类兄弟（哥哥祐真×弟弟祐音8岁）
- 三P结构：怪物+哥哥+弟弟，哥哥早与怪物有勾结
- 重口元素：藤蔓侵犯、全身敏感点玩弄、尿道play、扩张、调教、第一次
- 节奏把控：从恐惧→疼痛→迷惑→快感→沉沦的完整心理变化
- H场景密度极高：从头到尾几乎没有停歇，层层升级
- 核心张力：依赖的哥哥其实是共犯，8岁幼童从哭喊到无意识发出快感呻吟

**文风特点：**
- 第三人称，感官描写极致细腻
- 痛与快感交织：不回避疼痛，但重点写疼痛如何转化为快感
- 心理转变自然：8岁小孩的认知（"想嘘嘘""哥哥骗我""为什么哥哥说好舒服"）
- 鬼畜但不粗暴：藤蔓有意识、哥哥有引导、节奏有张有弛
- 对话符合年龄：祐音的台词都是8岁小孩的话，不会说成人的话
- 层层升级：先嘴→胸→前面→后面，单人→兄弟齐上，怪物辅助→真人交合

**模块里怎么用：**
- 鬼畜系极H的核心参考
- 节奏把控技法：松紧结合，张弛有度，不要一上来就最狠
- 张力营造：信任崩塌（哥哥是共犯）、未知恐惧（有意识的藤蔓）、痛快交织
- 重口H场景写法：明确标注每个阶段的身体反应、心理变化、感官细节
- 幼龄角色的H写法：用儿童的认知和语言表达快感，不说成人话
- 尺度把控：虽然极H，但有逻辑可循，不是为了重口而重口

## 四、模块教程的结构来源

基于以上分组，模块教程被组织成：

1. 模块定位和调用决策树
2. 四个子类文风详解（日常+校园合并为日常纯爱大类）
   - 日常纯爱风（宝贝+运动裤）
   - 综漫爽文风（世界旅行）
   - 黑暗产业风（正太会所）

3. 通用正太写作技巧：年龄感、身份关系、视角心理
4. 正反例库按子类分组

## 五、核心资料保留建议

如果后续只保留最核心的资料，建议全部保留5篇，因为：

- 《正太滋味》虽然最短，但鬼畜技法最集中
- 《正太会所》的设定系写法独一无二
- 《世界旅行》是唯一的爽文风参考
- 《宝贝》和《运动裤》分别覆盖日常和校园两种纯爱
