## 一、7 大功能类别（v11 新增分类维度）

> 7 大功能类别 = 按"道具对主角的作用"分类。每个具体道具**至少**属于 1 个功能类别，部分跨 2-3 个类别。

| # | 功能类别 | 作用 | 典型道具 | 出现的子类型 |
|---:|---|---|---|---|
| 1 | **束缚类** | 限制身体/行动 | 椅 / 床 / 绳 / 链 | 战队 / 调教 / 同人 |
| 2 | **阻塞类** | 限制性功能/射精 | 环 / 笼 / 锁 | 战队 / 调教 / 修真 |
| 3 | **刺激类** | 物理刺激敏感点 | 夹 / 针 / 杯 | 战队 / 触手 / 调教 |
| 4 | **植入类** | 侵入式控制/标记 | 胶囊 / 印记 / 芯片 | 战队 / 修真 / 同人 |
| 5 | **监控类** | 监测/展示/直播 | 仪 / 摄像头 / 直播椅 | 战队 / 调教 / 同人 |
| 6 | **通道类** | 体液/能量交换 | 管（玻璃管 / 软管）| 战队 / 修真 |
| 7 | **科技类** | 现代/科幻控制 | VR/AR / 触手机械人 / 控制器 | 战队 / 异种 / 同人 |

**功能类别决策树**（v11 新增）：

```
问题 1：道具对主角的作用是？
├─ 限制身体/行动 → 束缚类（1）
├─ 限制性功能/射精 → 阻塞类（2）
├─ 物理刺激 → 刺激类（3）
├─ 侵入式控制/标记 → 植入类（4）
├─ 监测/展示 → 监控类（5）
├─ 体液/能量交换 → 通道类（6）
└─ 现代/科幻控制 → 科技类（7）

问题 2：道具的具体形态是？
├─ 椅状 → 触手椅（束缚+科技+监控）
├─ 床状 → 调教床（束缚）
├─ 环/笼/锁状 → 贞操锁（阻塞）
├─ 夹/针/杯状 → 刺激物
├─ 胶囊/印记/芯片 → 植入物
├─ 摄像头/直播椅 → 监控物
├─ 玻璃管/软管 → 通道物
└─ VR/AR/触手机械人 → 科技物
```

## 二、12 大具体道具（v11 新增具体清单）

> 12 大具体道具 = 按"出现频度 + 用户 XP 命中度"排序。每个具体道具包含：代表作品 / 描述模板 / 所属功能类别 / 跨类组合。

### 1. 贞操锁（环 / 笼 / 锁 三合一）

- **代表**：5.4 超能战队 kk（贞操锁 = 能源核心）、4.2 超级战队收集（5 战士贞操锁）、4.1 高弹紧身衣（凌夜的贞操锁）
- **描述模板**：「[贞操锁环]锁住[主角]的[阴茎/乳头/睾丸]，只有反派可解锁。锁体本身[发出微光/震动/计时]，象征着[所有权/控制权/能源核心]」
- **所属功能类别**：阻塞类（主）+ 监控类（次，贞操仪联动）+ 植入类（次，控制芯片联动）
- **XP 命中**：贞操 + 能源核心 + 长期性癖 + 符号化所属
- **特殊形态**：
  - 贞操锁 = 能源核心（5.4 超能 kk）— 锁本身提供战衣能量
  - 贞操锁 = 反派所有权（4.2 超级战队收集）— 锁是反派"所有物"
  - 贞操锁 = 5 战士系列（4.2 超级战队收集）— 5 个战士被同款锁锁
  - 贞操锁 + 12h 任务（4.2 超级战队收集）— 锁带定时任务

### 2. 触手椅

- **代表**：5.1 龙骧战队（触手椅 + 直播 + 多战士同囚）、4.4 裆能战记（茧房拘束+足交）
- **描述模板**：「[触手椅]上设有[触手/锁链/直播摄像头]，主角被束缚在椅中，触手[缠绕裆部/乳头/肛门]，反派在远处[监控/调教/直播]」
- **所属功能类别**：束缚类（主）+ 科技类（次，触手机械人）+ 监控类（次，直播）
- **XP 命中**：触手 + 束缚 + 直播 + 多战士同囚
- **特殊形态**：
  - 触手椅 + 直播椅（5.1 龙骧）— 触手 + 直播同时进行
  - 触手椅 + 直播椅 + 投币控制（5.1 龙骧）— 观众可投币触发特定动作
  - 触手椅 + 茧房拘束（4.4 裆能）— 触手包裹全身
  - 触手椅 + 公共+偷听（5.3 超星）— 在公共空间放置触手椅

### 3. 控制芯片

- **代表**：4.1 高弹紧身衣（控制芯片 + 注射器）、4.2 超级战队收集（控制芯片 + 贞操锁）、5.1 龙骧战队（控制芯片 + VR 头盔）
- **描述模板**：「[控制芯片]植入[主角]的[后颈/心脏/裆部]，反派可远程[电击/束缚/控制]。芯片带有[人格编号/触发词/远程改写]功能」
- **所属功能类别**：植入类（主）+ 监控类（次，远程监控）+ 阻塞类（次，控制行动）
- **XP 命中**：植入 + 远程控制 + 反派控制 + 触发词控制
- **特殊形态**：
  - 控制芯片 + 注射器（4.1 高弹）— 芯片通过注射植入
  - 控制芯片 + 淫纹（5.1 龙骧）— 物理铭刻版（破解难度更高）
  - 控制芯片 + VR 头盔（5.1 龙骧第 6 章）— 意识/身体双接管
  - 控制芯片 + 飞机杯（5.1 龙骧）— 芯片破解后反控

### 4. 直播椅 / 直播摄像头

- **代表**：5.1 龙骧战队（直播控制椅 + 触手椅）、4.2 超级战队收集（5 战士被直播）、4.4 裆能战记（会所点穴/手淫教学）

- **所属功能类别**：监控类（主）+ 束缚类（次，束缚主角）+ 科技类（次，触屏控制）
- **XP 命中**：直播 + 公开 + 商品化 + 投币控制 + 触屏互动

  - 直播椅 + 投币控制（5.1 龙骧）— 观众投币可触发特定动作

  - 直播 + 远程指挥（4.4 裆能）— 远程控制主角手淫
